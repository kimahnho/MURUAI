## A. 조음음운장애 (Articulation/Phonological Disorders)

### A-1. 한국어 자음 습득 순서 (Korean Consonant Acquisition Order)

**임상 사실:**
한국어 자음의 습득은 비음과 파열음이 가장 빠르고, 마찰음이 가장 늦다.

| 습득 연령 | 자음 (75% 기준) | 비고 |
|-----------|----------------|------|
| 2세 후반 이전 | 파열음(ㅂ,ㅃ,ㅍ,ㄷ,ㄸ,ㅌ), 비음(ㅁ,ㄴ,ㅇ), ㅎ, 종성 ㄹ | 가장 이른 습득 |
| 2;0~2;11 | ㅍ, ㅁ, ㅇ (완전습득) | |
| 3;0~3;11 | ㅂ, ㅃ, ㄸ, ㅌ, 종성 ㄱ (완전습득) | |
| 4;0~4;11 | ㄴ, ㄲ, ㄷ, ㅈ (완전습득) | 연구개 파열음 강화 |
| 5;0~5;11 | ㄱ, ㅋ, ㅈ, ㅉ, ㅊ, 초성 ㄹ (완전습득) | 유음 습득 |
| 6;0~6;11 | ㅅ, ㅆ (완전습득) | 치조마찰음이 가장 늦음 |

**자음정확도(PCC) 연령별 기준:**
| 연령 | PCC |
|------|-----|
| 2;6 | 약 80~85% |
| 3세 | 약 85~90% |
| 4세 | 약 90~95% |
| 5세 | 약 95~98% |

**치료 자료 생성 시 중요성:**
- 목표 음소 선정 시 아동의 연령에 적합한 음소인지 판단하는 기준
- 예: 3세 아동에게 /ㅅ/ 목표 설정은 부적절 (6세 완전습득)
- 치료 자료에 포함할 단어 선정 시 발달적으로 적절한 목표음 배치

**테스트 질문:**
> Q: 4세 아동이 /ㅅ/를 /ㄷ/로 대치하고 있다. 이것이 연령에 적절한 오류인가?
> A: 적절함. /ㅅ/의 완전습득 연령은 6세이므로, 4세에서의 파열음화(/ㅅ/→/ㄷ/)는 발달적으로 정상 범위 내에 있음.

### A-2. 음운변동 분석 (Phonological Process Analysis)

**임상 사실:**
음운변동(음운패턴)은 아동이 성인 말소리 체계를 단순화하는 규칙적 패턴이다.

#### 전형적(발달적) 음운변동과 소거 연령

| 음운변동 유형 | 설명 | 소거 연령 |
|-------------|------|----------|
| **음절구조 변동** | | |
| 종성생략 (final consonant deletion) | 받침 탈락: '밥'→'바' | 3세 이전 |
| 음절탈락/축약 | 다음절 축소: '바나나'→'나나' | 3세 이전 |
| 어중자음단순화 | 연쇄자음 단순화 | 3세 이전 |
| **동화 변동** | | |
| 자음조화 (consonant harmony) | '강아지'→'강아기' | 3세 이전 |
| 비음화 동화 | 비음에 영향받아 비음화 | 3세 이전 |
| **대치 변동** | | |
| 파열음화 (stopping) | 마찰음/파찰음→파열음: '사과'→'다과' | 3세~4세 |
| 전방화 (fronting) | 연구개→치조: '가방'→'다방' | 3;6~4세 |
| 파찰음화 (affrication) | 마찰음→파찰음: '사과'→'자과' | 4세~5세 |
| 유음 단순화 | /ㄹ/→/ㄴ/ 또는 생략 | 5세 |
| 긴장음화 (tensing) | 이완→경: '바지'→'빠지' | 4세~5세 |

#### 오류패턴 분류 기준 (임상적)
- **나이에 적절한 패턴**: 해당 연령 아동의 10% 이상이 사용
- **지체된 패턴**: 해당 연령의 10% 이하가 사용하지만 더 어린 연령에서는 10% 이상 사용
- **특이한(비전형적) 패턴**: 어떤 연령에서도 10% 이상 사용하지 않음

**치료 자료 생성 시 중요성:**
- 아동의 음운변동 패턴에 맞춘 치료 목표 설정
- 음운변동 소거가 예상되는 연령을 기반으로 치료 우선순위 결정
- 비전형적 음운변동을 보이는 경우 더 집중적 중재 필요

**테스트 질문:**
> Q: 2;6 아동이 종성을 모두 생략하고, 파열음화를 보인다면 치료가 필요한가?
> A: 종성생략은 3세 이전 소거되는 변동이므로 2;6에서는 주시가 필요하나 즉각 치료 대상은 아닐 수 있음. 파열음화도 3~4세에 소거되므로 현재 연령에서는 발달적으로 적절함. 단, 3세 이후에도 지속되면 치료 고려.

### A-3. 조음 치료 접근법 (Treatment Approaches)

#### 1) 전통적 접근법 (Van Riper, 1939)
- **핵심**: 한 번에 하나의 오류음을 치료
- **단계**: 지각훈련 → 음소 산출 → 음절 → 단어 → 구 → 문장 → 자발화
- **지각훈련(ear training)** 강조: 자기 음소의 오류를 인식하게 함
- **적용**: 소수의 조음 오류, 왜곡 오류에 효과적

#### 2) 음운변동 접근법 (Phonological Process Approach)
- **핵심**: 개별 음소가 아닌 음운 체계(패턴)를 치료
- **원리**: 하나의 음운변동을 억제하면 관련 음소 전체에 일반화
- **Hodson & Paden의 순환적 접근법**: 목표 패턴을 순환적으로 다룸
- **적용**: 다수의 음운변동을 보이는 아동

#### 3) 최소대립쌍 접근법 (Minimal Pairs)
- **핵심**: 하나의 음소만 다른 단어쌍으로 의미 대립을 인식시킴
- **예**: '달' vs '발' (초성 /ㄷ/ vs /ㅂ/)
- **원리**: 음운 대립의 의미변별 기능을 아동이 인식하게 함
- **전제**: 아동이 대립쌍 중 하나는 정확히 발음 가능해야 함

#### 4) 의사소통중심 접근법
- **핵심**: 자연스러운 의사소통 맥락에서 조음 훈련
- **장점**: 일반화 촉진, 실생활 적용 용이

**치료 자료 생성 시 중요성:**
- 접근법에 따라 자료의 구조가 완전히 달라짐
- Van Riper: 음소 위치별(어두초성/어중초성/어말종성) 단어 목록 필요
- 최소대립쌍: 대립 단어쌍 데이터베이스 필요
- 음운변동 접근법: 특정 변동을 유발하는 단어 세트 필요

**테스트 질문:**
> Q: /ㅅ/→/ㄷ/ 파열음화, /ㅈ/→/ㄷ/ 파열음화, /ㅊ/→/ㄷ/ 파열음화를 보이는 아동에게 적합한 접근법은?
> A: 음운변동 접근법이 적합. 세 가지 오류가 모두 '파열음화'라는 하나의 음운변동에 해당하므로, 하나의 목표음(예: /ㅅ/)을 치료하면 나머지 음소에도 일반화 기대.

### A-4. 조음 오류 유형 (Error Types)

| 오류 유형 | 정의 | 예시 | 심각도 |
|----------|------|------|--------|
| **대치(Substitution)** | 목표 음소를 다른 음소로 대체 | '사과'→'다과' (/ㅅ/→/ㄷ/) | 중간 |
| **생략(Omission)** | 목표 음소가 산출되지 않음 | '사과'→'아과' (/ㅅ/ 생략) | 심함 |
| **왜곡(Distortion)** | 목표와 유사하나 정확하지 않은 산출 | 설측 /ㅅ/ | 경미 |
| **첨가(Addition)** | 불필요한 음소가 추가됨 | '사과'→'스사과' | 드묾 |

**심각도 순서: 생략 > 대치 > 왜곡 > 첨가**
- 생략이 말명료도에 가장 큰 영향을 미침
- 왜곡은 음성적으로 목표와 유사하므로 상대적으로 경미

**말명료도(Speech Intelligibility) 평가:**
- 자음정확도(PCC)가 주요 지표
- PCC-R(수정자음정확도): 왜곡을 정반응으로 처리
- 일반적으로 낯선 청자가 이해할 수 있는 비율로 측정

**치료 자료 생성 시 중요성:**
- 오류 유형에 따라 치료 전략이 달라짐
- 대치 오류: 최소대립쌍, 변별자질 접근
- 생략 오류: 음소 산출 자체를 목표로
- 왜곡 오류: 조음 위치/방법의 미세 조정 (Van Riper 접근)

**테스트 질문:**
> Q: 아동이 '가방'을 '아방'으로 발음한다면 오류 유형은?
> A: 초성 /ㄱ/ 생략(omission). 연구개 파열음 /ㄱ/이 어두초성에서 생략된 것.

## B. 언어발달장애 (Language Development Disorders)

### B-1. 연령별 언어 발달 이정표 (Milestones)

| 연령 | 수용언어 | 표현언어 | 화용/기타 |
|------|---------|---------|----------|
| **0~6개월** | 소리에 반응, 목소리 방향 인식 | 쿠잉(cooing), 옹알이 시작 | 사회적 미소 |
| **6~9개월** | 이름 부르면 반응, '안돼' 이해 시작 | 반복적 옹알이(babbling): '마마마' | 공동주시(joint attention) 시작 |
| **9~12개월** | 친숙한 단어 20여개 이해 | 첫 낱말 산출 시작 ('엄마', '맘마') | 의도적 의사소통, 가리키기(pointing) |
| **12~18개월** | 간단한 지시 이해 ('공 줘') | 표현어휘 약 50개, 한 낱말기 | 요구하기, 거부하기 |
| **18~24개월** | 수용어휘 약 190개, 신체 부위 인식 | 표현어휘 100~200개, 두 낱말 조합 시작 ('엄마 물'), 어휘폭발(vocabulary spurt) | 언급하기, 보여주기 |
| **24~36개월** | 간단한 질문 이해, 크기 개념 | 표현어휘 약 200~500개, 3어 조합, 주어+목적어+서술어, 조사 사용 시작 | 질문하기 시작 ('이게 뭐야?'), 대화 차례 주고받기 |
| **36~48개월** | 복잡한 지시 이해, 시간 개념 시작 | 4~5어 문장, 연결어미 사용 급증, 과거시제 사용 | 이야기 구성(narrative) 시작, 역할놀이 |
| **48~60개월** | 수동문 이해, 비유 이해 시작 | 복문 사용, 접속/내포문, 문법형태소 안정 | 설명하기, 이유 대기, 대화 규칙 이해 |

**언어발달 지연 경고 신호 (Red Flags):**
- 12개월: 옹알이 없음, 제스처 없음
- 18개월: 한 낱말도 표현하지 못함
- 24개월: 두 낱말 조합 없음, 표현어휘 50개 미만
- 36개월: 3어 조합 없음, 낯선 사람이 50% 이상 이해하지 못함

**치료 자료 생성 시 중요성:**
- 목표 연령에 맞는 언어 수준의 자료 설계
- 예: 24개월 대상 자료는 2어 조합 수준으로 제작
- 연령에 비해 지연된 아동의 현재 기능 수준에 맞춘 자료 필요

**테스트 질문:**
> Q: 30개월 아동이 단일 낱말만 사용하고 두 낱말 조합이 없다면?
> A: 언어발달 지연 의심. 24개월까지 두 낱말 조합이 가능해야 하며, 30개월에 한 낱말기에 머물러 있으면 약 6개월 이상 지연. 언어 평가 및 중재 권고.

### B-2. 평균발화길이 (MLU: Mean Length of Utterance)

**한국어 MLU 측정 방식:**
- **MLU-w (평균낱말길이)**: 발화당 평균 낱말 수
- **MLU-m (평균형태소길이)**: 발화당 평균 형태소 수
- 한국어는 교착어이므로 MLU-m이 MLU-w보다 일반적으로 더 큼

**연령별 기준 (대략적):**
| 연령 | MLU-w (낱말) | MLU-m (형태소) |
|------|-------------|---------------|
| 2세 | 약 1.5~2.0 | 약 2.0~3.0 |
| 3세 | 약 2.5~3.5 | 약 3.5~4.5 |
| 4세 | 약 3.0~4.0 | 약 4.0~5.0 |

**핵심 연구 결과:**
- 2세와 3세 사이에 유의한 MLU 차이 존재
- 3세와 4세 사이에는 유의한 차이가 크지 않음 (천장효과 시작)
- MLU는 어린 아동(2~3세)의 언어 수준을 민감하게 반영
- MLU-w/m이 증가할수록 조사 유형수와 빈도가 유의하게 증가

**발화 표본 수집 기준:**
- 최소 50발화, 이상적으로 100발화 이상
- 2세와 4세는 100발화 이상에서 신뢰도(.90 이상)
- 3세는 50발화 이상에서도 충분한 신뢰도

**치료 자료 생성 시 중요성:**
- 목표 MLU에 맞춘 모델 발화 설계
- MLU 2.0 수준 아동: 2어 조합 중심의 자료
- MLU 3.5 수준 아동: 조사 포함 3~4어 문장 수준 자료

**테스트 질문:**
> Q: MLU-w가 1.2인 3세 아동의 언어 수준을 어떻게 해석하는가?
> A: 3세의 기대 MLU-w는 약 2.5~3.5이므로, MLU-w 1.2는 약 2세 초반 수준에 해당. 약 1년 이상의 표현언어 지연을 시사.

### B-3. 어휘 발달 단계

| 연령 | 표현어휘 수 | 주요 특징 |
|------|-----------|----------|
| 12개월 | 1~3개 | 첫 낱말 산출 |
| 18개월 | 약 50개 | 수용어휘 약 190개 |
| 20~21개월 | 약 100개 | |
| 23~24개월 | 어휘폭발기 | 하루 3~4개 신규어 습득 |
| 24개월 | 약 200개 | |
| 36개월 | 약 500개 | |

**어휘 구성 특징:**
- 초기 어휘는 명사 우세 (명사 > 동사+형용사의 2배)
- 문법적 기능어(조사, 어미 등)의 비율은 전체 어휘수 증가에 따라 상승
- 과잉확대(overextension): '멍멍이'로 모든 네발동물 지칭
- 과잉축소(underextension): '의자'를 자기 집 의자만으로 한정

**주요 평가 도구:**
- M-B CDI-K (MCDI 한국판): 641개 단어, 18~36개월 대상
- REVT (수용표현어휘력검사)
- K-BNT-C (한국판 보스턴이름대기검사 아동용)

**치료 자료 생성 시 중요성:**
- 아동의 어휘 수준에 맞는 단어 선정
- 명사, 동사, 형용사의 적절한 비율 배분
- 고빈도 생활어휘를 우선적으로 활용

**테스트 질문:**
> Q: 24개월 아동의 표현어휘가 30개 미만이면?
> A: 말늦은 아동(late talker) 의심. 24개월에 표현어휘 50개 미만은 언어발달 지연의 핵심 기준. 평가 의뢰 필요.

### B-4. 구문 발달 단계

| 연령 | 구문 특징 |
|------|----------|
| 12~18개월 | 한 낱말기 (holophrase) |
| 18~24개월 | 두 낱말 조합: '엄마 물', '아빠 가' (의미관계 표현) |
| 24~30개월 | 3어 조합, 조사('-이/가', '-을/를') 출현 시작, 어미 사용 증가 |
| 30~36개월 | 주어+목적어+서술어 구조, 연결어미 출현 ('-고', '-면') |
| 36~48개월 | 복문 출현, 연결어미 급증 (나열'-고', 조건'-면', 순서'-고'), 2세→3세 사이 연결어미 사용 4배 증가 |
| 48~60개월 | 내포문(관형절, 명사절), 피동/사동 표현, 복잡한 연결어미 ('-는데', '-니까') |

**조사 발달 순서:**
- 초기(2세 후반): 주격 '-이/가', 목적격 '-을/를', 처소격 '-에'
- 중기(3세): 도구격 '-으로', 여격 '-에게', 공동격 '-와/과'
- 후기(4세 이후): 비교격 '-보다', 복합조사

**연결어미 의미관계 발달:**
- 나열(-고), 동시(-면서), 계기(-아서/어서), 대립(-지만), 이유/원인(-니까)
- 이 다섯 가지 의미관계의 연결어미는 2세 후반~3세에 함께 산출 시작
- 3세 아동은 2세 아동 대비 4배 이상의 연결어미 사용

**치료 자료 생성 시 중요성:**
- 목표 구문 수준에 맞는 문장 구조 설계
- 초기 단계: 2어 조합 모델 문장
- 중급 단계: 조사 포함 3~4어 문장
- 고급 단계: 복문 및 연결어미 활용 문장

**테스트 질문:**
> Q: 4세 아동이 연결어미 '-고'만 사용하고 '-면', '-니까' 등을 사용하지 못한다면?
> A: 3세 수준의 구문 발달. 4세에는 다양한 연결어미를 사용해야 하므로, 구문발달 촉진을 위한 중재 필요.

### B-5. 화용론 발달 (Pragmatic Development)

**의사소통 기능의 발달 순서:**

| 연령 | 의사소통 기능 | 분류 |
|------|-------------|------|
| 8~12개월 | 요구하기(requesting), 거부하기(protesting) | 행동통제(behavioral regulation) |
| 12~15개월 | 보여주기(showing), 주기(giving) | 사회적 상호작용(social interaction) |
| 12~18개월 | 언급하기(commenting), 가리키기(pointing) | 동시주의(joint attention) |
| 16~23개월 | 질문에 대답하기, 정보 요구하기 | 동시주의 확장 |
| 24개월+ | 대화 차례 주고받기(turn-taking) | 담화 기술 |
| 36개월+ | 이야기 구성(narrative), 역할놀이 | 담화 기술 확장 |
| 48개월+ | 대화 주제 유지, 명료화 요구 응답 | 대화 기술 |

**의사소통 기능 분류 체계:**
1. **행동통제(Behavioral Regulation)**: 요구하기, 거부하기 - 타인의 행동 조절
2. **사회적 상호작용(Social Interaction)**: 인사하기, 부르기, 보여주기 - 타인의 관심 유도
3. **동시주의(Joint Attention)**: 언급하기, 정보 요구하기 - 사물/사건에 대한 공유

**발달 방향:**
- 초기: 행동통제 > 사회적 상호작용 > 동시주의
- 24개월 이후: 동시주의 기능이 행동통제를 능가

**치료 자료 생성 시 중요성:**
- 치료 활동 설계 시 목표 의사소통 기능 반영
- 초기 단계 아동: 요구하기 상황 만들기 (선택 활동)
- 진전 단계: 언급하기/설명하기 활동으로 확장

**테스트 질문:**
> Q: 24개월 아동이 요구하기만 하고 언급하기(commenting)를 하지 않는다면?
> A: 화용 발달 지연 가능성. 24개월경에는 동시주의 기능(언급하기, 정보 요구하기)이 활발해져야 함. 행동통제적 기능에만 의존하는 것은 지연을 시사.

### B-6. 문해력 발달 (Literacy Development)

| 연령 | 읽기 발달 | 쓰기 발달 |
|------|----------|----------|
| 3세 | 글자 유형 구별, 글자에 의미가 있음을 인식 | 끼적거리기 단계 |
| 4세 | 주변 익숙한 글자 읽기 시작, 음운인식 발달 시작 | 의도적 글자 형태 모방 |
| 5세 | 글자/문장 부분적 읽기, 해독 시작 | 자기 이름 쓰기 가능 |
| 6세 | 낱자-소리 대응 이해, 단어 해독 | 글자 쓰기 가능 |

**음운인식(Phonological Awareness) 발달 순서:**
1. 음절 인식 (가장 먼저, 4세 이후)
2. 음절체-각운 인식
3. 음소 인식 (가장 나중)
- 과제별: 합성 → 변별 → 탈락 → 대치 순서로 발달
- 변별, 탈락, 합성: 5세에 급격한 발달
- 대치: 6세에 급격한 발달

**치료 자료 생성 시 중요성:**
- 학령기 아동의 읽기/쓰기 관련 활동 설계
- 음운인식 활동: 음절 수 세기 → 초성/종성 분리 → 음소 대치 활동
- 한글 해독 훈련: 자모 결합 원리 활용

**테스트 질문:**
> Q: 5세 아동이 자기 이름도 읽지 못하고 음절 분리를 못한다면?
> A: 문해 전기술(pre-literacy) 지연 의심. 5세에는 음운인식이 급격히 발달하는 시기이므로, 음절 수준 음운인식도 없다면 평가 필요.

## C. 신경언어장애 (Neurogenic Language Disorders) - 차단/의뢰 지식

### C-1. 실어증 유형 분류 (Aphasia Classification)

| 유형 | 병변 위치 | 유창성 | 이해력 | 따라말하기 | 핵심 특징 |
|------|---------|--------|--------|----------|----------|
| **브로카 실어증** (Broca's) | 좌측 전두엽 (브로카 영역) | 비유창 | 비교적 양호 | 저하 | 전보문체(agrammatism), 실문법증 |
| **베르니케 실어증** (Wernicke's) | 좌측 측두엽 (베르니케 영역) | 유창 | 심하게 저하 | 저하 | 신조어, 착어(paraphasia), 이해 불량 |
| **전도성 실어증** (Conduction) | 궁형속(arcuate fasciculus) | 유창 | 양호 | 심하게 저하 | 따라말하기 선택적 장애 |
| **전실어증** (Global) | 광범위 손상 | 비유창 | 심하게 저하 | 심하게 저하 | 가장 심한 형태, 이해+표현 모두 장애 |
| **명칭 실어증** (Anomic) | 다양 | 유창 | 양호 | 양호 | 이름대기 장애가 주증상 |
| **초피질운동 실어증** | 브로카 주변 | 비유창 | 양호 | 양호 | 따라말하기는 보존 |
| **초피질감각 실어증** | 베르니케 주변 | 유창 | 저하 | 양호 | 따라말하기는 보존 |

**감별 핵심 3가지 변수:**
1. 유창성(fluency) - 유창 vs 비유창
2. 이해력(comprehension) - 양호 vs 저하
3. 따라말하기(repetition) - 양호 vs 저하

**치료 자료 생성 시 중요성:**
- 소아 치료 AI에서는 직접 대상이 아니지만, 뇌손상 이력 있는 아동 감별에 필요
- 성인 대상 확장 시 필수 지식

**테스트 질문:**
> Q: 유창하게 말하지만 이해력이 떨어지고 따라말하기도 안 되는 실어증 유형은?
> A: 베르니케 실어증. 유창+이해 저하+따라말하기 저하 = 베르니케.

### C-2. CAS vs Dysarthria 감별

| 특징 | CAS (아동기 말실행증) | Dysarthria (마비말장애) |
|------|---------------------|----------------------|
| **원인** | 말 운동 프로그래밍 장애 | 근육의 마비, 약화, 불협응 |
| **오류 일관성** | 비일관적 (같은 단어도 다르게 오류) | 일관적 (예측 가능한 오류) |
| **모방 vs 자발** | 모방이 자발보다 더 나을 수 있으나, CAS는 오히려 모방이 더 어려울 수도 | 모방과 자발이 비슷한 수준 |
| **말속도 영향** | 느린 속도에서 개선 가능 | 느린 속도에서도 왜곡 유지 |
| **운율(prosody)** | 비정상적 운율, 강세 오류 | 단조로운 운율 |
| **비구어 과제** | 구강 실행 과제 어려움 | 근력/운동범위 저하 |
| **신경학적 소견** | 뚜렷한 근육 약화 없음 | 근육 약화/긴장도 이상 |
| **핵심 구분** | 프로그래밍 문제 | 실행(execution) 문제 |

**ASHA의 CAS 진단 핵심 특징 3가지:**
1. 자음과 모음의 비일관적 오류
2. 음절/단어 간 동시조음 전환(coarticulatory transitions) 장애
3. 부적절한 운율(prosody)

**치료 자료 생성 시 중요성:**
- CAS 의심 아동은 일반 조음치료와 다른 접근 필요
- CAS: 동작 프로그래밍 중심, 반복 연습, 운율 훈련
- Dysarthria: 근력 강화, 보상 전략

**테스트 질문:**
> Q: 아동이 '바나나'를 '바다나', '바나다', '나바나' 등 매번 다르게 발음한다면?
> A: CAS(아동기 말실행증) 의심. 동일 단어에 대한 비일관적 오류 패턴은 CAS의 핵심 특징. Dysarthria에서는 오류가 일관적.

## D. 유창성장애 (Fluency Disorders) - 차단/의뢰 지식

### D-1. 정상적 비유창성 vs 병리적 비유창성

| 특징 | 정상적 비유창성 (OD: Other Disfluency) | 병리적 비유창성 (SLD: Stuttering-Like Disfluency) |
|------|--------------------------------------|------------------------------------------------|
| **유형** | 다음절 단어 전체반복, 수정, 미완성 구, 간투사('음', '그') | 단어 부분반복, 일음절 단어 전체반복, 연장(prolongation), 막힘(block) |
| **반복 횟수** | 1~2회 | 3회 이상 |
| **반복 단위** | 구/단어 단위 | 음절/소리 단위 |
| **긴장** | 긴장 없음, 규칙적 리듬 | 긴장 동반, 불규칙적 |
| **지속 시간** | 짧음 | 0.5초 이상 |
| **부수행동** | 없음 | 눈 깜빡임, 고개 흔들기 등 |
| **출현 양상** | 간헐적, 특정 시기에만 | 지속적, 대부분의 시간 |

**진단 기준 (빈도 기반):**
- 총 비유창성 빈도 10% 이상 → 말더듬 의심
- 비정상적 비유창성(SLD) 빈도 3% 이상 → 말더듬 진단
- 정상적 비유창성은 전체 발화의 5~10% 이내에서 관찰

**말더듬의 핵심행동 3가지:**
1. 반복(repetition): 소리, 음절, 단어의 반복
2. 연장(prolongation): 말소리가 부적절하게 길어짐
3. 막힘(block): 말소리 산출이 완전히 차단됨

**치료 자료 생성 시 중요성:**
- 유창성 문제가 있는 아동의 치료에서는 발화 속도, 문장 길이 조절이 핵심
- AI가 유창성장애를 감별하여 적절한 의뢰를 권고할 수 있어야 함

**테스트 질문:**
> Q: 3세 아동이 "나-나-나-나 갈 거야"처럼 한 소리를 4회 이상 반복하고 얼굴에 힘을 주는 경우?
> A: 병리적 비유창성(SLD) 의심. 소리 단위의 반복이 3회 이상이고 긴장이 동반되므로 정상적 비유창성이 아님. 유창성장애 전문 평가 의뢰 필요.

---

### D-2. 말더듬 시작 연령과 자연회복

**발병 연령:**
- 대부분 2~5세 사이에 시작 (가장 흔한 시작: 3~4세)
- 언어발달이 급속한 시기와 일치

**자연회복율:**
- 경미한 경우 50~80%가 자연 회복
- 발병 후 12~24개월 이내에 회복되는 경우가 많음
- 약 1%의 아동만 성인기까지 지속

**자연회복 예측 요인 (긍정적):**
- 여아 (남아보다 회복율 높음)
- 발병 후 1년 이내
- 가족력 없음
- 언어능력이 양호
- SLD 비율이 낮음

**자연회복 불리 요인:**
- 남아
- 발병 후 1년 이상 지속
- 가족력 있음
- 부수행동 동반
- 회피행동/부정적 태도 발달

**학령기 이후 말더듬:**
- 습관으로 고착되어 특별한 언어치료 없이는 회복 어려움
- 호전과 악화를 반복하는 만성적 경과

**치료 자료 생성 시 중요성:**
- 2~5세 아동에서 비유창성 관찰 시 정상/병리 판단 기준 제공
- 의뢰 타이밍 결정에 필수 지식

**테스트 질문:**
> Q: 4세 남아가 6개월 전부터 말더듬이 시작되었고, 아버지도 어릴 때 더듬었다. 기다려볼 것인가?
> A: 즉시 평가 의뢰 권고. 남아이고 가족력이 있으므로 자연회복 가능성이 상대적으로 낮음. 조기 중재가 효과적.

## E. 음성장애 (Voice Disorders) - 차단/의뢰 지식

### E-1. 소아 음성장애 원인

**가장 흔한 원인: 성대결절(Vocal Nodules)**
- 소아 음성장애의 가장 흔한 원인
- 음성 남용/오용으로 인한 성대의 양성 점막 질환
- 6~7세 남자 어린이에서 많이 발생

**음성 남용/오용의 예:**
- 과도한 소리 지르기
- 지속적으로 큰 소리로 말하기
- 부적절한 발성 습관 (예: 경성 성대 접촉)
- 기침, 목 가다듬기 습관

**기타 소아 음성장애 원인:**
- 성대 낭종(vocal cyst)
- 성대 폴립(vocal polyp)
- 후두유두종(laryngeal papilloma)
- 성대 마비(vocal fold paralysis)
- 기능적 음성장애

**치료 원칙:**
- 소아 성대결절: 원칙적으로 수술 시행하지 않음
- 보존적 치료(음성 휴식 + 음성치료) 우선
- 소아에서의 음성치료 효과가 성인보다 더 좋음
- 환자의 80% 이상에서 증상 호전 가능

**치료 자료 생성 시 중요성:**
- 음성장애가 의심되는 경우 이비인후과 의뢰 필요성 판단
- 음성 위생(vocal hygiene) 교육 자료 제공 가능

**테스트 질문:**
> Q: 6세 남아가 쉰 목소리가 3개월 이상 지속된다면?
> A: 이비인후과 의뢰 필요. 3개월 이상 지속되는 음성 변화는 후두 검사(laryngoscopy)가 필요. 성대결절이 가장 흔한 원인이나, 후두유두종 등 다른 원인도 배제해야 함.

---

### E-2. 이비인후과(ENT) 의뢰 기준

**즉시 의뢰가 필요한 경우:**
1. 2주 이상 지속되는 음성 변화(쉰 목소리)
2. 음성 변화와 함께 삼킴 곤란
3. 호흡 시 천명음(stridor)
4. 음성 치료에도 불구하고 3개월 이상 개선 없음
5. 갑작스러운 음성 상실

**의뢰 권고 상황:**
- 반복적인 후두염
- 습관적 기침/목 가다듬기
- 음성 과사용 직업군 (교사 등) 음성 문제
- 변성기 이후에도 높은 음도 유지 (변성발성장애)

**언어치료사-이비인후과 협력 모델:**
- 음성장애 치료 전 반드시 이비인후과 진찰 선행
- 후두 소견 확인 후 음성치료 시작
- 치료 중간/종료 시 후두 재검사

**치료 자료 생성 시 중요성:**
- AI가 음성 관련 주소(chief complaint)를 감별하여 적절한 의뢰 경로 안내
- 음성위생 프로그램 자료 제공 가능

**테스트 질문:**
> Q: 부모가 "아이가 항상 목이 잠겨있다"고 보고할 때 언어치료사의 첫 대응은?
> A: 이비인후과 의뢰 권고. 지속적인 쉰 목소리는 후두 구조적 문제를 배제해야 하므로, 음성치료 시작 전 반드시 이비인후과 진찰 필요.

## G. 치료 자료 생성 AI를 위한 핵심 규칙 요약

### 반드시 지켜야 할 연령-기준 규칙

1. **목표 음소 선정**: 해당 음소의 습득 연령보다 6개월 이상 지난 연령에서 오류가 있을 때 치료 목표로 설정
2. **음운변동 판단**: 소거 연령이 지났는데도 해당 변동이 나타나면 치료 대상
3. **어휘 수준 매칭**: 목표 아동의 표현어휘 수준에 맞는 단어 사용
4. **구문 수준 매칭**: 목표 아동의 MLU에 맞는 문장 구조 설계
5. **화용 목표 설정**: 발달 순서(요구→언급→질문→설명)에 맞는 의사소통 기능 목표

### 의뢰/차단 규칙

1. **유창성**: SLD 3% 이상, 부수행동 동반, 6개월 이상 지속 → 유창성 전문가 의뢰
2. **음성**: 2주 이상 쉰 목소리 지속 → 이비인후과 의뢰 후 음성치료
3. **CAS 의심**: 비일관적 오류 + 운율 이상 → 말운동장애 전문 평가
4. **신경학적 원인 의심**: 갑작스러운 말/언어 변화, 근력 약화 동반 → 신경과 의뢰
