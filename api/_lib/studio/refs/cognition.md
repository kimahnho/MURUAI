# COGNITION DOMAIN: 인지치료 전문가 레퍼런스

> 이 파일은 AI 에이전트가 한국의 인지치료사(20년 이상 임상경력)처럼 사고하고 의사결정하도록 만드는 임상 지식 베이스입니다.
> 단순 지식이 아닌, 실제 치료사의 임상적 판단 과정(clinical reasoning)을 재현합니다.
> 대상: ASD 및 발달장애 아동 (만 1-10세+). 기능연령(functional age) 기준으로 모든 판단을 내린다.
> 참고: 한국 작업치료/인지치료 임상 기준, K-CDI, Bayley-III 한국판, 한국형 덴버, KISE-BAAT 등의 평가 체계에 부합하도록 작성됨.

## 1. 인지 발달 5단계

기능연령(functional age) 기준. **생활연령이 아닌 기능연령을 반드시 사용할 것.**
한국 발달 규준: K-CDI(영유아발달검사), Bayley-III 한국판 인지척도, 한국형 덴버 발달선별검사 기준 참조.

### Stage 1: 기초 지각 (기능연령 2-3세)

**발달 규준:** K-CDI 인지 영역 24-36개월 수준. Bayley-III 인지척도 합성점수 기준 해당 월령 범위.

| 하위 영역 | 목표 기술 | 관찰 가능한 행동 | 예시 활동 |
|-----------|----------|-----------------|----------|
| 색상 매칭 | 같은 색상의 사물 짝짓기 | 빨강/파랑/노랑 3색 중 지시된 색을 고름 | 빨간 블록 → 빨간 컵 매칭 |
| 형태 매칭 | 같은 모양의 사물 짝짓기 | ○△□ 중 지시된 모양에 끼워넣기 가능 | 형태판(도형 퍼즐) |
| 사물 영속성 | 가려진 사물 찾기 | 천으로 덮인 물건을 찾으려고 천을 제거 | "컵 아래 뭐가 있을까?" |
| 크기 변별 | 크다/작다 구분 | 2개 사물 중 큰 것/작은 것을 골라줌 | 큰 공 vs 작은 공 비교 |

**핵심 원리:** 이 단계에서는 한 번에 하나의 속성만 다룬다. 색상과 형태를 동시에 요구하지 않는다.

**평가 지표 (Assessment Indicators):**
- 색상 매칭: 3색(빨/파/노) 중 2/3 이상 정반응 → 색상 매칭 습득으로 판정
- 형태 매칭: 3형태(○△□) 중 2/3 이상 정반응
- 사물 영속성: 3회 시행 중 2회 이상 덮개 제거 시도
- 크기 변별: 5쌍 중 4쌍 이상 정반응 (크기 차이 2배 이상 조건)

**적합한 활동:**
- 도형 끼워넣기 퍼즐 (3피스 이하)
- 색상별 분류 상자 (실물 조작 → 그림 기반 전환)
- 같은것찾기 (동일 사물 2개 중 선택, 선택지 3개 이하)

**Grading 파라미터:**
- 쉬움: 선택지 2개, 색상/형태 차이 극대화 (빨강 vs 파랑), 실물 크기 이미지
- 보통: 선택지 3개, 중간 차이, 약간 축소된 이미지
- 어려움: 선택지 4개, 유사 색상 포함 (빨강 vs 주황)

**전이 지표 (Stage 2 준비도):**
```
□ 색상 3색 매칭 80% 이상
□ 형태 3종 매칭 80% 이상
□ 실물↔그림 간 매칭 가능 (2D-3D 전환)
□ "같은 것" 개념어 이해 (구어 지시 "같은 거 줘" 수행)
→ 4개 중 3개 이상 충족 시 Stage 2 진입 고려
```

### Stage 2: 기초 분류 (기능연령 3-4세)

**발달 규준:** K-CDI 36-48개월 수준. 한국형 덴버 미세운동-적응 영역 해당 수준.

| 하위 영역 | 목표 기술 | 관찰 가능한 행동 | 예시 활동 |
|-----------|----------|-----------------|----------|
| 사물-사물 매칭 | 동일 사물 짝짓기 (실물↔그림) | 실물을 보고 해당 그림을 정확히 고름 | 실물 숟가락 → 그림 숟가락 |
| 단순 2범주 분류 | 2가지 명확한 범주 분류 | "동물"과 "탈것"을 분리 가능 | 동물 vs 탈것 분류 |
| 1:1 대응 | 하나씩 짝짓기 | 접시마다 사과를 하나씩 배치 | 접시 1개에 사과 1개 |
| 기능적 짝짓기 | 기능으로 연결되는 쌍 | "같이 쓰는 것" 연결 가능 | 칫솔↔치약, 양말↔신발 |

**핵심 원리:** 범주 경계가 명확해야 한다. "동물 vs 탈것"은 OK. "강아지 vs 고양이 분류"는 이 단계에서 부적절 (같은 범주 내 하위분류는 Stage 4).

**평가 지표:**
- 사물-사물 매칭: 10쌍 중 8쌍 이상 정반응
- 2범주 분류: 8항목(각 범주 4개) 분류 시 7개 이상 정확
- 기능적 짝짓기: 5쌍 중 4쌍 이상 정확 연결

**적합한 활동:**
- 같은것찾기 (선택지 4-6개)
- 짝맞추기 카드 (4-6쌍)
- 2칸 분류판 (상위범주 2개)
- 기능적 연결 (선 긋기)

**Grading 파라미터:**
- 쉬움: 범주 2개, 항목당 3개, 범주 경계 극대화 (동물 vs 탈것)
- 보통: 범주 2개, 항목당 4개, 범주 경계 중간 (음식 vs 옷)
- 어려움: 범주 2개, 항목당 5개, 일부 경계 사물 포함 (인형 → 놀잇감? 사람?)

**전이 지표 (Stage 3 준비도):**
```
□ 2범주 분류 90% 이상 정확도
□ 기능적 짝짓기 5쌍 중 4쌍 이상
□ 실물↔그림 전환 자유로움
□ "이것은 OO이야/아니야" 판단 가능
→ 4개 중 3개 이상 충족 시 Stage 3 진입 고려
```

### Stage 3: 변별과 서열화 (기능연령 4-5세)

**발달 규준:** K-CDI 48-60개월 수준. KISE-BAAT 인지 영역 해당 수준 참조.

| 하위 영역 | 목표 기술 | 관찰 가능한 행동 | 예시 활동 |
|-----------|----------|-----------------|----------|
| 기능 기반 분류 | 용도/기능으로 분류 | "먹는 것"과 "입는 것"을 나눌 수 있음 | 기능별 분류판 |
| 시각 변별 (3+ 차이) | 3개 이상 명확한 차이 찾기 | 두 그림에서 다른 부분 3개를 지적 | 틀린그림찾기 (색상/유무/크기 차이) |
| 단순 순서 (3단계) | 3단계 순서 배열 | 3장 카드를 시간순으로 배열 | 손씻기: 물틀기→비누→헹구기 |
| 크기 서열화 (3개) | 3개를 크기 순서로 | 3개 사물을 크기순 배열 | 작은→중간→큰 곰 |

**핵심 원리:** 변별 시 차이가 한눈에 보여야 한다. 색상 차이, 존재/부재 차이, 크기 차이 중심. 미세한 패턴/방향 차이는 Stage 5.

**평가 지표:**
- 시각 변별: 차이 5개 중 3개 이상 찾기 (제한시간 2분)
- 3단계 순서: 3세트 중 2세트 이상 정확 배열
- 크기 서열화: 3개 서열 3세트 중 2세트 이상 정확

**적합한 활동:**
- 틀린그림찾기 (차이 3-5개, 명확한 차이)
- 3단계 순서맞추기 (일상 루틴)
- 크기/길이 서열화 (3-4개)
- 기능 분류 활동

**Grading 파라미터:**
- 쉬움: 차이 3개(색상/유무 위주), 순서 3단계, 서열 3개
- 보통: 차이 4개(크기 차이 포함), 순서 3단계(비일상), 서열 4개
- 어려움: 차이 5개(위치 차이 포함), 순서 4단계(일상), 서열 4개(추상 속성)

**전이 지표 (Stage 4 준비도):**
```
□ 시각 변별 5개 중 4개 이상 (명확한 차이 조건)
□ 3단계 순서 90% 정확
□ 크기 서열화 4개 성공
□ 기능 분류 2범주 이상 안정적
→ 4개 중 3개 이상 충족 시 Stage 4 진입 고려
```

### Stage 4: 추상 분류와 패턴 (기능연령 5-7세)

**발달 규준:** 초등 저학년 수준. KISE-BAAT 인지/학습준비도 영역 해당 수준.

| 하위 영역 | 목표 기술 | 관찰 가능한 행동 | 예시 활동 |
|-----------|----------|-----------------|----------|
| 상위범주 분류 | 추상적 상위 범주 이해 | "과일"이라는 상위명 아래 여러 과일 묶기 | 범주 분류판 |
| 하위범주 분류 | 같은 상위범주 내 세분화 | "과일 vs 채소" 분리 | 하위범주 분류 |
| 순서 (4-5단계) | 4-5단계 순서 배열 | 라면 끓이기 순서를 5장으로 배열 | 라면: 물→끓이면→면→스프→완성 |
| 패턴 인식 | 반복 패턴 계속하기 | AB, ABC 패턴의 다음 항목 예측 | ○△○△○? |
| 속성 기반 분류 | 2가지 속성으로 분류 | "빨간 과일"을 골라낼 수 있음 | 교차 속성 분류 |

**핵심 원리:**
- 상위범주와 하위범주를 반드시 구분한다.
- **상위범주** = 서로 전혀 다른 카테고리 간 분류 (음식 vs 동물, 탈것 vs 가구). 범주 경계가 넓고 명확.
- **하위범주** = 같은 상위범주 안에서의 세분화 (과일 vs 채소, 육지동물 vs 바다동물). 범주 경계가 좁고 모호할 수 있음.
- 상위범주 분류가 안정적으로 가능해야 하위범주 분류로 진행한다.
- **임상 경고:** 지적장애 아동은 상위범주 OK → 하위범주 NO인 경우가 흔하다. 이는 정상적인 발달 순서이며, 강제로 하위범주를 넣지 않는다.

**평가 지표:**
- 상위범주: 3범주(동물/탈것/음식) 12항목 분류 시 10개 이상 정확
- 하위범주: 2하위범주(과일 vs 채소) 8항목 분류 시 6개 이상 정확
- 4-5단계 순서: 2세트 중 1세트 이상 완전 정확
- 패턴: AB 패턴 3회 중 2회 이상 정확 → ABC 패턴 도전

**Grading 파라미터:**
- 쉬움: 상위범주 2개(동물 vs 탈것), 4단계 순서(일상), AB 패턴
- 보통: 상위범주 3개, 4단계 순서(비일상), ABC 패턴
- 어려움: 하위범주 분류, 5단계 순서, AABB 패턴, 교차속성

**전이 지표 (Stage 5 준비도):**
```
□ 상위범주 3개 이상 분류 90% 정확
□ 하위범주 1세트 이상 80% 이상
□ 4-5단계 순서맞추기 안정적
□ 교차 속성 분류 1세트 이상 성공
→ 4개 중 3개 이상 충족 시 Stage 5 진입 고려
```

### Stage 5: 복합 추론 (기능연령 7세+)

**발달 규준:** 초등 2학년 이상 수준. 일반 교육과정 수학/논리 영역과 연계.

| 하위 영역 | 목표 기술 | 관찰 가능한 행동 | 예시 활동 |
|-----------|----------|-----------------|----------|
| 복합 추론 | 원인-결과, 문제 해결 | "왜 그럴까?" 질문에 논리적 대답 | "비가 오면 뭘 가져가야 할까?" |
| 미세 시각 변별 | 미묘한 차이 5개 이상 찾기 | 패턴/질감/방향/표정 미세 차이 발견 | 고급 틀린그림찾기 |
| 숨은그림찾기 | 복잡 배경 속 사물 탐색 | 산만한 배경에서 목표물 검출 | 숲 그림 안에서 동물 8개 찾기 |
| 다단계 문제 해결 | 여러 조건 동시 고려 | 2개 이상 조건을 동시에 적용 | "빨간색이면서 동그란 것 찾기" |
| 추상적 서열화 | 시간/양/무게 등 추상 서열 | 눈에 보이지 않는 속성의 순서 | 시간: 아침→점심→저녁→밤 |

**핵심 원리:** 여러 인지 기능이 통합적으로 요구된다. 단일 활동에 변별+분류+추론 복합 가능.

**평가 지표:**
- 미세 시각 변별: 8개 차이 중 5개 이상 (제한시간 3분)
- 숨은그림찾기: 10개 목표물 중 7개 이상 (제한시간 5분)
- 다단계 문제: 2조건 동시 적용 3회 중 2회 이상 성공
- 6단계 순서: 1세트 이상 완전 정확

**Grading 파라미터:**
- 쉬움: 미세 차이 5개, 숨은그림 6개(배경 단순), 2조건, 6단계 순서(일상)
- 보통: 미세 차이 7개, 숨은그림 8개(배경 중간), 2조건(추상), 6단계(비일상)
- 어려움: 미세 차이 10개, 숨은그림 10개+(배경 복잡), 3조건, 6단계+(다분기)

### Stage 간 전이 의사결정 트리 (종합)

```
현재 Stage에서 전이 지표 4개 중 3개 이상 충족?
  → YES: 다음 Stage 진입 시도
    → 다음 Stage 가장 쉬운 난이도에서 50% 이상 성공?
      → YES: 다음 Stage 확정, 쉬운 난이도에서 시작
      → NO: 현재 Stage 어려운 난이도에서 유지, 2주 후 재시도
  → NO: 현재 Stage 유지
    → 현재 난이도에서 80% 이상 성공?
      → YES: 같은 Stage 내 난이도 한 단계 상향
      → NO: 현재 난이도 유지 또는 하향 고려

⚠ 임상 판단 예외:
- 아동이 3회 연속 거부/회피 → 난이도 무조건 하향 (동기 이슈)
- 아동이 성공하되 "너무 쉬워요" 반응 → 2단계 점프 허용
- 특정 하위 영역만 통과 못함 → 분리 프로필(Section 8) 참조
```

## 2. 분류 수준 체계 5단계

분류 활동 설계 시 반드시 아래 5단계 계층을 명확히 구분하여 적용한다.
이 체계는 Inhelder & Piaget(1964)의 분류 발달 이론과 한국 임상 현장의 실증 관찰을 통합한 것이다.

```
Level 1: 지각적 색상 분류 (같은 색 모으기)
  ↓
Level 2: 지각적 형태 분류 (같은 모양 모으기)
  ↓
Level 3: 기능적 분류 (같이 쓰는 것 모으기 — 칫솔+치약)
  ↓
Level 4: 범주적 분류 (상위범주 — 과일, 동물, 탈것)
  ↓
Level 5: 교차속성 분류 (2가지 기준 동시 — 빨간 동물 vs 파란 동물)
```

### Level 1: 지각적 색상 분류 (기능연령 2-3세)

| 항목 | 내용 |
|------|------|
| 분류 기준 | 색상 하나 (빨강/파랑/노랑) |
| 사물 수 | 2-4개씩 2그룹 |
| 배경 | 단색 또는 흰색 배경 |
| 예시 | 빨간 사과 + 빨간 컵 vs 파란 공 + 파란 블록 |
| 금지 | 같은 색상 안에서 형태/크기가 혼동되는 조합 |

**진단 기준:** 6항목(3색 x 2개) 제시 시, 같은 색 2개씩 모으기 5/6 이상 정반응.
**실패 패턴:** (a) 무작위 배치 → 색상 개념 미형성, 사전 활동(색상 명명) 필요. (b) 특정 색만 성공 → 해당 색부터 시작, 점진 확장.
**Grading Up 규칙:** 3색 90% 이상 → 4색(초록 추가) → 유사색(빨강 vs 주황) 도입.
**Grading Down 규칙:** 3색 60% 미만 → 2색으로 축소, 색상 대비 극대화(빨강 vs 파랑).

### Level 2: 지각적 형태 분류 (기능연령 2.5-3.5세)

| 항목 | 내용 |
|------|------|
| 분류 기준 | 형태 하나 (○ / △ / □) |
| 사물 수 | 2-4개씩 2그룹 |
| 색상 통제 | 모든 아이템의 색상을 동일하게 하거나, 색상을 무작위로 섞어 색상이 단서가 되지 않도록 함 |
| 예시 | 동그란 것들 (공, 시계, 접시) vs 네모난 것들 (상자, 책, TV) |
| 금지 | 색상과 형태가 동시에 분류 단서가 되는 조합 |

**진단 기준:** 6항목(3형태 x 2개) 제시 시, 같은 형태 2개씩 모으기 5/6 이상 정반응.
**실패 패턴:** (a) 색상으로 분류 시도 → 색상이 단서 되지 않도록 통제 후 재시행. (b) 형태 변별 자체 어려움 → 도형 매칭(동일 도형 짝짓기)부터 시작.
**Grading Up 규칙:** 기본 3형태 90% → 실물 형태(별, 하트 등 비정형) 도입 → 3D 입체 형태.
**Grading Down 규칙:** 3형태 60% 미만 → 2형태(○ vs □)로 축소, 크기 차이 극대화.

**⚠ 임상 경고 — Level 1 vs Level 2 순서:**
대부분의 아동은 색상이 형태보다 먼저 발달한다. 그러나 일부 ASD 아동은 형태 우선(shape bias)을 보인다.
→ 반드시 두 가지를 모두 평가한 후, 강점 영역부터 시작할 것.

### Level 3: 기능적 분류 (기능연령 3.5-5세)

| 항목 | 내용 |
|------|------|
| 분류 기준 | 용도/기능 ("먹는 것" vs "입는 것") |
| 사물 수 | 각 범주 3-5개, 총 6-10개 |
| 힌트 | 분류 기준을 말로 제시 가능 ("같이 쓰는 거 모아볼까?") |
| 예시 | 칫솔+치약+수건 (씻는 것) vs 연필+지우개+공책 (공부하는 것) |
| 금지 | 기능이 모호한 사물 (예: 물 → 먹는 것? 씻는 것?) |

**진단 기준:** 8항목(기능 2범주 x 4개) 중 7개 이상 정확 분류.
**실패 패턴:** (a) 지각적 속성(색상/형태)으로 분류 → 기능 관계 설명 필요 ("이건 같이 쓰는 거야"). (b) 기능 관계는 이해하나 분류 행동이 안 됨 → 짝짓기(2개씩 연결)부터 시작 후 분류로 확장.
**Grading Up 규칙:** 2기능범주 90% → 3기능범주 → 추상적 기능("집에서 쓰는 것" vs "밖에서 쓰는 것").
**Grading Down 규칙:** 60% 미만 → 짝짓기(1:1 기능 연결)로 퇴행, 사물 수 감소.

### Level 4: 범주적 분류 (기능연령 5-7세)

| 항목 | 내용 |
|------|------|
| 분류 기준 | 상위범주 명칭 (과일, 동물, 탈것, 가구 등) |
| 사물 수 | 각 범주 4-6개, 총 8-18개 |
| 범주 수 | 시작 2개 → 최대 4개 |
| 예시 | 사과/바나나/포도/딸기 (과일) vs 강아지/고양이/토끼/새 (동물) |
| 금지 | 범주 경계가 모호한 조합 (토마토 → 과일? 채소?), 지적장애 아동에게는 특히 주의 |

**진단 기준:** 12항목(3범주 x 4개) 중 10개 이상 정확.
**실패 패턴:** (a) 범주 명칭을 모름 → 범주 명칭 학습(명명 활동) 선행. (b) 기능적 분류에 고착 → Level 3→4 브릿징 활동: "먹는 것 중에서도 과일만 모아볼까?" (c) 범주 경계 혼동(채소/과일) → 범주 경계가 명확한 조합부터 (동물 vs 탈것).
**Grading Up 규칙:** 2범주 90% → 3범주 → 4범주 → 하위범주 분류 도입.
**Grading Down 규칙:** 60% 미만 → 2범주로 축소, 범주 경계 극대화(동물 vs 탈것), 항목 수 감소.

**⚠ 하위범주 도입 의사결정:**
```
상위범주 3개 이상 90% 정확?
  → NO: 상위범주 유지
  → YES: 하위범주 시도
    → 첫 하위범주 세트(과일 vs 채소) 70% 이상?
      → YES: 하위범주 활동 시작
      → NO: 경계가 더 명확한 하위범주로 변경 (육지동물 vs 바다동물)
        → 여전히 60% 미만?
          → 상위범주 유지, 1개월 후 재시도
```

### Level 5: 교차속성 분류 (기능연령 7세+)

| 항목 | 내용 |
|------|------|
| 분류 기준 | 2가지 속성 동시 적용 (색상+범주, 크기+범주 등) |
| 구조 | 2x2 매트릭스 또는 벤다이어그램 |
| 예시 | 빨간 동물 / 빨간 탈것 / 파란 동물 / 파란 탈것 → 4칸 매트릭스 |
| 금지 | 3속성 이상 동시 (인지 부하 과다) |

**진단 기준:** 2x2 매트릭스 8항목 중 6개 이상 정확 배치.
**실패 패턴:** (a) 한 속성만으로 분류 → "색상으로 먼저 나누고, 그 안에서 동물/탈것 나누자" 단계적 안내. (b) 매트릭스 구조 이해 못함 → 물리적 2x2 칸판 제공, 시각적 보조.
**Grading Up 규칙:** 2x2 매트릭스 80% → 속성 추상화(크기+범주) → 3x2 매트릭스.
**Grading Down 규칙:** 50% 미만 → Level 4(단일 속성 범주 분류)로 퇴행, 해당 속성별 분류 각각 확인.

### Level 간 전이 의사결정 종합 트리

```
현재 Level에서 진단 기준 충족?
  → NO: 현재 Level 유지
    → 실패 패턴 분석 → 해당 패턴별 개입 적용
    → 2주간 개입 후 재평가
  → YES (90% 이상):
    → 다음 Level 가장 쉬운 난이도 시도
      → 60% 이상? → 다음 Level 확정
      → 60% 미만? → 현재 Level 어려운 난이도 유지, 2주 후 재시도

⚠ 흔한 실수:
- Level 1(색상) 통과했다고 Level 2(형태)를 건너뛰고 Level 3(기능)으로 가는 것
  → 절대 금지. Level 1과 2는 독립적으로 각각 확인해야 함.
- Level 4에서 범주 경계가 모호한 항목(토마토, 고구마 등)을 초기부터 포함하는 것
  → 경계 명확 항목으로 먼저 안정화 후 도입.
```

## 3. 순서화 2축 모델

순서맞추기 활동은 2가지 독립적 축으로 난이도가 결정된다. 두 축을 혼동하면 난이도 설정이 부정확해진다.

### 2축 정의

**X축: 친숙도 (Familiarity)**
- 일상 루틴 (매일 반복) ← → 비일상 사건 (가끔/처음)
- 예: 손씻기(일상) vs 소풍가기(비일상)

**Y축: 복잡도 (Complexity = 단계 수)**
- 2단계 ← → 6단계+
- 예: 2단계(물틀기→손씻기) vs 6단계(가게→물건고르기→줄서기→돈내기→거스름돈→나가기)

### 4-Zone 매트릭스

```
              ┌─────────────────────┬─────────────────────┐
              │     일상 (친숙)      │    비일상 (비친숙)    │
┌─────────────┼─────────────────────┼─────────────────────┤
│ 단순        │  Zone A (가장 쉬움)  │  Zone B (중간)       │
│ (2-3단계)   │ 손씻기 3단계         │ 병원가기 3단계       │
│             │ 밥먹기 3단계         │ 미용실가기 3단계     │
│             │ 양치질 3단계         │ 비행기타기 3단계     │
├─────────────┼─────────────────────┼─────────────────────┤
│ 복잡        │  Zone C (중간)       │  Zone D (가장 어려움) │
│ (4-6+단계)  │ 등원하기 5단계       │ 캠핑가기 5단계       │
│             │ 라면끓이기 5단계     │ 생일파티 6단계       │
│             │ 빨래하기 5단계       │ 마트에서 장보기 6단계 │
└─────────────┴─────────────────────┴─────────────────────┘
```

### 6단계 진행 경로

```
Step 1: Zone A 시작 — 일상 + 2단계 (예: 물틀기 → 손씻기)
  ↓ 80% 이상 성공 시
Step 2: Zone A 확장 — 일상 + 3단계 (예: 물틀기 → 비누 → 헹구기)
  ↓ 80% 이상 성공 시
Step 3: Zone B 진입 — 비일상 + 3단계 (예: 접수 → 진료 → 약타기)
  ↓ 80% 이상 성공 시
Step 4: Zone C 진입 — 일상 + 4-5단계 (예: 라면끓이기 5단계)
  ↓ 80% 이상 성공 시
Step 5: Zone B-C 혼합 — 비일상 + 4단계 또는 일상 + 5단계
  ↓ 80% 이상 성공 시
Step 6: Zone D 진입 — 비일상 + 5-6단계

⚠ 주의:
- Zone A→D로 바로 점프하지 않는다.
- Zone B와 C는 독립적으로 난이도가 다른 축이므로, 아동 특성에 따라 B 먼저 또는 C 먼저 가능.
  → 일상 경험이 풍부한 아동: C(복잡도↑) 먼저
  → 다양한 경험이 있는 아동: B(비친숙↑) 먼저
```

### 순서맞추기 유형별 특성

| 유형 | 설명 | 적합 기능연령 | 예시 |
|------|------|-------------|------|
| 시간 순서 | 사건의 시간적 흐름 | 4세+ | 아침→점심→저녁 |
| 인과 순서 | 원인→결과 흐름 | 5세+ | 비→우산 쓰기→안 젖음 |
| 절차 순서 | 행동 절차 | 4세+ | 요리, 세탁 순서 |
| 성장 순서 | 생물 성장 | 5세+ | 씨앗→새싹→나무→꽃 |
| 이야기 순서 | 내러티브 구조 | 6세+ | 시작→전개→결말 |

### 순서 단계 수 vs 기능연령 대응

| 기능연령 | 적정 단계 수 | 최대 도전 단계 수 |
|----------|------------|------------------|
| 3-4세 | 2-3단계 | 3단계 |
| 4-5세 | 3단계 | 4단계 |
| 5-6세 | 3-4단계 | 5단계 |
| 6-7세 | 4-5단계 | 6단계 |
| 7세+ | 5-6단계 | 6단계+ |

### 순서맞추기 실패 분석 의사결정

```
아동이 N단계 순서에서 실패:
  → 첫 단계/마지막 단계는 맞고 중간이 틀림?
    → YES: 중간 단계 기억/이해 부족 → 시각적 단서 추가, 중간 단계를 앵커로 제공
    → NO: 전체적으로 무작위 배치?
      → YES: 순서 개념 자체 미형성 → N-1단계로 하향, 일상 루틴으로 변경
      → NO: 특정 위치에서 반복적 오류?
        → 해당 사건의 경험 부족일 수 있음 → 역할놀이/실제 경험 후 재시도

⚠ 4단계부터 무너지는 경우 (흔한 엣지케이스):
- 3단계까지 안정적이나 4단계부터 실패 → 작업기억 용량 제한일 가능성
- 개입: (a) 중간 앵커 제공 (2번째 카드 고정) (b) 시각적 타임라인 제공 (c) "처음에 뭐했지? 다음에는?" 언어적 스캐폴딩
```

## 4. 인지 활동 유형

각 활동 유형별로 목표, 적정 기능연령, 난이도 파라미터, 구체적 제작 지침을 명시한다.

### 4-1. 사물인지 (같은것찾기, 짝맞추기)

**목표:** 사물의 시각적 속성을 인식하고, 동일/유사 사물을 변별하는 능력.

| 파라미터 | 쉬움 | 보통 | 어려움 |
|---------|------|------|--------|
| 기능연령 | 2-3세 | 3-4세 | 4-5세 |
| 선택지 수 | 2-3개 | 4-5개 | 6-8개 |
| 사물 유사도 | 낮음 (사과 vs 자동차) | 중간 (사과 vs 딸기) | 높음 (사과 vs 토마토) |
| 사물 유형 | 실물 사진 | 단순 일러스트 | 세밀 일러스트 |
| 배경 | 흰색/단색 | 연한 단색 | 약간의 배경 패턴 |

**제작 지침:**
- 같은것찾기: 상단에 타겟 1개, 하단에 선택지 N개 배치. 타겟과 정답은 완전 동일 이미지.
- 짝맞추기: 2열 배치, 좌우를 선으로 연결. 쌍의 관계를 명확히 (동일 사물 or 기능적 쌍).
- **흔한 실수:** 선택지에 타겟과 "거의 같지만 살짝 다른" 것을 넣는 것 → 이것은 변별 활동이지 매칭 활동이 아님. 매칭에서는 정답이 "완전히 같은 것".

### 4-2. 분류/범주화 (카테고리 분류)

**목표:** 공통 속성에 따라 사물을 그룹화하는 능력. (상세 수준 체계는 Section 2 참조)

| 파라미터 | 쉬움 | 보통 | 어려움 |
|---------|------|------|--------|
| 기능연령 | 3-4세 | 4-6세 | 6세+ |
| 범주 수 | 2개 | 3개 | 4개 |
| 범주당 항목 | 3-4개 | 4-5개 | 5-6개 |
| 분류 기준 | 지각적 (색상/형태) | 기능적/범주적 | 교차속성 |
| 범주 경계 | 극명 (동물 vs 탈것) | 중간 (음식 vs 도구) | 모호 (과일 vs 채소) |

**제작 지침:**
- 분류판: 범주 수만큼 칸을 나누고, 각 칸에 범주 레이블(글자 + 대표 아이콘).
- 분류 항목: 범주 경계가 명확한 사물부터 배치. 경계 모호 사물은 최상위 난이도에서만.
- **흔한 실수:** 범주 경계 아이템을 초기 단계에서 포함하는 것 (토마토: 과일? 채소? → 최고 난이도에서만).

### 4-3. 시각변별 (틀린그림찾기)

**목표:** 두 개의 유사한 그림에서 차이점을 발견하는 시각 주의력/변별력.

| 파라미터 | 쉬움 | 보통 | 어려움 |
|---------|------|------|--------|
| 기능연령 | 4-5세 | 5-6세 | 7세+ |
| 차이점 수 | 3개 | 5개 | 7-10개 |
| 차이 유형 | 색상/유무 차이 | 크기/위치 차이 추가 | 패턴/방향/미세 차이 |
| 배경 복잡도 | 단순 (물체 3-5개) | 중간 (물체 5-8개) | 복잡 (물체 10개+) |
| 그림 크기 | 크게 (A4 절반 이상) | 중간 | 작게 (A4 1/4) |

**차이 유형 계층 (쉬움→어려움):**
1. 존재/부재 차이 (있다 vs 없다) — 가장 쉬움
2. 색상 차이 (빨강 vs 파랑)
3. 크기 차이 (큰 vs 작은)
4. 위치/방향 차이 (왼쪽 vs 오른쪽, 위 vs 아래)
5. 패턴/질감 차이 (줄무늬 vs 물방울)
6. 미세 형태 차이 (선 길이, 각도 등) — 가장 어려움

**ADHD 특화 설계 (Section 6과 연동):**
- 차이점을 4사분면에 균등 분산 배치 (한 곳에 몰지 않기)
- 배경 시각 소음 최소화
- 한 번에 찾을 차이점 수 제한 (ADHD: 3개 → 일반: 5개)

**제작 지침:**
- 좌우 배치: 두 그림을 나란히 배치, 크기 동일, 간격 적절.
- 차이 배치: 화면 전체에 고르게 분산. 상단/하단/좌측/우측 균등.
- **흔한 실수:** (a) 차이점이 모두 한 영역에 몰림 → 4사분면 분산 필수. (b) 배경이 너무 복잡하여 차이점이 배경에 묻힘 → 배경과 차이점의 구별 확보.

### 4-4. 서열화 (크기/길이/양 순서)

**목표:** 연속적 속성(크기, 길이, 양, 무게 등)을 순서대로 배열하는 능력.

| 파라미터 | 쉬움 | 보통 | 어려움 |
|---------|------|------|--------|
| 기능연령 | 3-4세 | 4-5세 | 6세+ |
| 항목 수 | 3개 | 4개 | 5-6개 |
| 속성 | 크기 (구체적, 시각적) | 길이/높이 | 양/무게/시간 (추상적) |
| 차이 정도 | 극명 (2배 차이) | 중간 (1.5배) | 미세 (1.2배) |

**서열 속성 계층 (구체→추상):**
1. 크기 — 시각적으로 직관적 (가장 쉬움)
2. 길이/높이 — 1차원 비교
3. 양(수량) — 이산적 속성
4. 무게 — 시각적으로 보이지 않음 (도구 필요)
5. 시간 — 완전 추상적 (가장 어려움)

**제작 지침:**
- 항목 간 차이가 균등해야 함 (작은/중간/큰 = 1:2:3 비율 권장).
- 방향: "작은 것부터" vs "큰 것부터" 모두 연습하되, 초기에는 한 방향 고정.
- **흔한 실수:** 3개 서열에서 양 끝만 맞추고 가운데를 틀리는 패턴 → 가운데 항목에 대한 비교 연습 추가 ("이것은 이것보다 크고, 이것보다 작아").

### 4-5. 순서이해 (순서맞추기)

**목표:** 사건/행동의 시간적/논리적 순서를 이해하고 배열하는 능력. (상세 모델은 Section 3 참조)

| 파라미터 | 쉬움 | 보통 | 어려움 |
|---------|------|------|--------|
| 기능연령 | 4-5세 | 5-6세 | 7세+ |
| 단계 수 | 3단계 | 4-5단계 | 6단계+ |
| 내용 친숙도 | 일상 루틴 | 친숙한 비일상 | 비친숙 |
| 카드 형태 | 명확한 장면 전환 | 중간 전환 | 미세한 장면 변화 |

**제작 지침:**
- 각 카드는 하나의 명확한 행동/사건만 표현.
- 카드 간 시간 간격이 일정해야 함 (갑자기 큰 시간 점프 금지).
- 첫 카드와 마지막 카드는 시각적으로 상태 변화가 극명해야 함 (시작 상태 vs 완료 상태).
- **흔한 실수:** 중간 카드끼리 순서가 교환 가능한 설계 → 각 카드의 순서가 논리적으로 유일해야 함.

### 4-6. 숨은그림찾기 (고급 시각변별)

**목표:** 복잡한 시각 배경 속에서 특정 목표 사물을 탐색/검출하는 능력.

| 파라미터 | 쉬움 | 보통 | 어려움 |
|---------|------|------|--------|
| 기능연령 | 5-6세 | 6-7세 | 7세+ |
| 목표물 수 | 3-4개 | 5-6개 | 8-10개 |
| 배경 복잡도 | 단순 (물체 10개 이하) | 중간 (물체 15-20개) | 복잡 (물체 25개+) |
| 목표물 가림 정도 | 약간 겹침 | 부분 가림 | 대부분 가림/위장 |
| 목표물 크기 | 큰 (배경 대비 눈에 띔) | 중간 | 작은 (배경에 묻힘) |

**제작 지침:**
- 목표물 목록을 그림 옆이나 상단에 명확히 제시 (찾아야 할 것 미리보기).
- 목표물은 배경 전체에 고르게 분산 배치.
- 배경은 주제가 있는 장면 (방, 공원, 숲 등)으로 자연스럽게 구성.
- **ADHD 특화:** 목표물 수 줄이기(3-4개), 배경 단순화, 시각적 시작점 표시("이 구역부터 찾아보자").
- **흔한 실수:** 목표물이 한 영역에 몰림 → 4사분면 분산 배치 필수.

## 5. 난이도 조절 (Activity Grading)

난이도 조절은 치료의 핵심 기술이다. "적절한 도전 수준(just-right challenge)"을 유지해야 한다.
**원칙:** 성공률 70-80%가 최적. 90% 이상이면 Grade Up, 50% 이하면 Grade Down.

### Grade UP 전략

한 번에 하나의 파라미터만 변경한다. 여러 파라미터를 동시에 올리면 난이도 점프가 과도해진다.

| 순서 | 파라미터 | 변경 방향 | 예시 |
|------|---------|----------|------|
| 1순위 | 항목 수 | ↑ 증가 | 선택지 3개 → 4개 |
| 2순위 | 유사도 | ↑ 증가 | 사과 vs 자동차 → 사과 vs 딸기 |
| 3순위 | 배경 | 추가/복잡화 | 흰 배경 → 연한 패턴 배경 |
| 4순위 | 시간 제한 | 추가/단축 | 무제한 → 3분 |
| 5순위 | 힌트 | 제거/축소 | 범주 레이블 제거 |
| 6순위 | 추상화 | ↑ 증가 | 구체적 사물 → 추상적 개념 |

### Grade DOWN 전략

| 순서 | 파라미터 | 변경 방향 | 예시 |
|------|---------|----------|------|
| 1순위 | 항목 수 | ↓ 감소 | 선택지 5개 → 3개 |
| 2순위 | 유사도 | ↓ 감소 (차이 극대화) | 사과 vs 딸기 → 사과 vs 자동차 |
| 3순위 | 배경 | 제거/단순화 | 패턴 배경 → 흰 배경 |
| 4순위 | 힌트 | 추가/강화 | 범주 레이블 + 대표 아이콘 추가 |
| 5순위 | 크기 | ↑ 확대 | 이미지 크기 확대 |
| 6순위 | 색상 대비 | ↑ 강화 | 채도/명도 차이 극대화 |

### 활동별 Grading 파라미터 테이블

| 활동 유형 | 항목 수 | 유사도 | 배경 | 색상 수 | 힌트 | 시간 |
|-----------|--------|--------|------|---------|------|------|
| 같은것찾기 | 2→8 | 낮→고 | 無→有 | 단색→다색 | 테두리→없음 | 무제한→2분 |
| 분류 | 2범주→4범주 | 명확→모호 | 단순→복잡 | 통일→혼합 | 레이블→없음 | 무제한→3분 |
| 틀린그림찾기 | 3차이→10차이 | 극명→미세 | 단순→복잡 | — | 동그라미힌트→없음 | 무제한→2분 |
| 순서맞추기 | 3단계→6단계 | — | 명확장면→미세변화 | — | 첫/끝고정→없음 | 무제한→3분 |
| 서열화 | 3개→6개 | 2배차→1.2배차 | 동일배경→다양 | — | 크기눈금→없음 | 무제한→2분 |
| 숨은그림찾기 | 3개→10개 | 눈에띔→위장 | 단순→복잡 | — | 구역표시→없음 | 무제한→5분 |

### Grading 의사결정 흐름

```
현재 활동에서 아동 성공률?
  → 90% 이상 (3회 연속):
    → Grade UP 1단계 (가장 낮은 순위 파라미터부터)
    → 1파라미터만 변경
  → 70-89%:
    → 현재 난이도 유지 (최적 도전 수준)
  → 50-69%:
    → 힌트 추가 시도 (Grade DOWN 4순위)
    → 힌트로도 60% 미만? → Grade DOWN 1단계
  → 50% 미만:
    → Grade DOWN 즉시 (가장 높은 순위 파라미터부터)
    → 2파라미터까지 동시 변경 허용 (빠른 성공 경험 필요)

⚠ 동기 저하 신호 감지 시: 즉시 난이도 1단계 낮추고 성공 경험 제공

## 6. 사물 항상성(Object Permanence) & 초기 인지

> Stage 1 이전 단계. 기능연령 0-2세 아동용.

### 발달 경로

| Level | 명칭 | 행동 | 연령 |
|-------|------|------|------|
| OP-1 | 부분 항상성 | 반쯤 가린 사물 찾기 | 6-8개월 |
| OP-2 | 완전 항상성 | 완전히 가린 사물 찾기 (1곳) | 8-12개월 |
| OP-3 | 이동 추적 | 사물이 A→B로 이동한 것을 추적 | 12-18개월 |
| OP-4 | 비가시 이동 | 컵 아래 숨긴 후 컵을 이동 → 찾기 | 18-24개월 |
| OP-5 | 표상 능력 | 눈에 안 보이는 사물을 상상/기억 | 24개월+ |

### 3세 ASD L3 인지 평가 프로토콜
```
□ 까꿍놀이 반응 (OP-2 확인)
□ 숨긴 사물 찾기 (OP-3 확인)
□ 원인-결과 장난감 반응 (버튼→소리)
□ 간단한 모방 (블록 쌓기 따라하기)
→ OP-2 미달성 → 감각운동기 활동 우선
→ OP-3+ → Stage 1 활동 진입 가능
```

## 7. 유사성(Similarity) 개념 발달

| 수준 | 명칭 | 예시 | 연령 |
|------|------|------|------|
| S-1 | 동일 | "똑같은 것 찾기" | 2-3세 |
| S-2 | 비동일 | "다른 것 찾기" | 3-4세 |
| S-3 | 유사 | "비슷한 것 찾기" (같은 범주) | 4-5세 |
| S-4 | 관련 | "같이 쓰는 것 찾기" (기능 관련) | 5-6세 |

### 지적장애 경도 특화
- S-1→S-2 전환이 특히 어려움 (동일성 판단에 과적합)
- "비슷하다" 개념: 구체적 예시 반복 필요 (사과와 배 = 비슷, 사과와 자동차 = 다름)
- 학습지: 같은것찾기에서 "똑같은 것" → "비슷한 것"으로 점진 전환

## 8. 작업기억(Working Memory) & 순서맞추기

### 연령별 작업기억 용량

| 연령 | WM 용량 | 순서 맞추기 가능 단계 |
|------|---------|---------------------|
| 3-4세 | 2±1 항목 | 2-3단계 |
| 5-6세 | 3±1 항목 | 3-4단계 |
| 7-8세 | 4±1 항목 | 4-5단계 |
| 9세+ | 5±1 항목 | 5-6단계+ |

### ADHD 감산 규칙
ADHD 동반 시 WM 용량에서 1 감산
→ 8세 ADHD: 실질 WM ≈ 3 → 3단계까지만

### WM 부하 조절 4전략
1. 시각 잔류: 이전 카드를 볼 수 있게 (순서대로 펼쳐놓기)
2. 청킹: 3단계를 "준비-실행-마무리" 그룹으로 묶기
3. 스토리라인 강화: 논리적 인과로 기억 부하 감소
4. 외부 기억: 번호/화살표 표시 허용 (보조 단서)

## 9. 패턴 인식/생성/확장

### 3단계 분리

| 단계 | 명칭 | 활동 | 연령 |
|------|------|------|------|
| P-1 | 인식 | "다음에 뭐가 올까?" (AB AB ?) | 4-5세 |
| P-2 | 생성 | "패턴 만들기" (AB AB 만들기) | 5-6세 |
| P-3 | 확장 | "새로운 패턴" (ABC ABC) | 6-7세 |

### 복잡도 계층
P1: AA (반복), P2: AB AB, P3: AAB AAB, P4: ABC ABC, P5: AABB AABB

## 10. 인과추론(Causal Reasoning)

### 발달 단계

| Level | 유형 | 예시 | 연령 |
|-------|------|------|------|
| CR-1 | 물리적 인과 | 공을 밀면 굴러간다 | 2-3세 |
| CR-2 | 생물학적 인과 | 물 안 주면 꽃이 시든다 | 3-4세 |
| CR-3 | 심리적 인과 | 넘어지면 아프다→운다 | 4-5세 |
| CR-4 | 사회적 인과 | 장난감 빼앗으면 친구 화남 | 5-6세 |
| CR-5 | 추상적 인과 | 연습하면 실력 늘어남 | 7세+ |

### 인지 vs 언어 감별
"왜 우산을 가져가?" 대답 못하는 경우:
- 인과추론 자체 결함: "왜?"를 그림으로 선택해도 못함 → 인지 중재
- 언어 표현 결함: 선택은 맞지만 말로 설명 못함 → 언어 중재

## 11. 수 개념 발달

### Gelman & Gallistel 수 세기 5원리

| 원리 | 설명 | 습득 연령 |
|------|------|----------|
| 1:1 대응 | 사물 하나에 수 하나 | 3-4세 |
| 안정된 순서 | 항상 1,2,3... 순서 | 3-4세 |
| 기수성 | 마지막 수 = 전체 개수 | 4-5세 |
| 순서 무관 | 어디서 세기 시작해도 같음 | 5-6세 |
| 추상성 | 아무 것이나 셀 수 있음 | 5-6세 |

### 한국어 수사 주의
- 고유어 수사: 하나, 둘, 셋 (세는 수)
- 한자어 수사: 일, 이, 삼 (읽는 수)
- 이중 체계가 외국 아동/지적장애에게 혼란 요인

## 16. 모방 vs 자발적 수행 격차

> ASD에서 흔함: 시범 보면 따라하지만 스스로 시작/해결 못함.

### 수행 수준 4단계

| Level | 명칭 | 지원 |
|-------|------|------|
| IS-1 | 완전 모방 | 최대 (hand-over-hand) |
| IS-2 | 부분 단서 | 첫 단계만 시범 |

## 18. 질문 이해의 인지적 전제

> "왜?" 질문 자체를 이해 못하면 인과추론 평가/교수 불가. ID 중등도에서 흔함.

### 질문 이해 인지 계층

| Level | 질문 | 인지 전제 | 연령 |
|-------|------|---------|------|
| QC-1 | "뭐?" | 사물 표상 | 2세 |
| QC-2 | "누구?" | 인물-행동 연결 | 2-3세 |
| QC-3 | "어디?" | 공간 표상 | 3-4세 |
| QC-4 | "언제?" | 시간 개념 | 4-5세 |
| QC-5 | "왜?" | 인과추론 | 5-6세 |
| QC-6 | "어떻게?" | 절차 표상 | 6-7세 |

감별: "왜 울어?" → 무반응 = 질문 형식 미이해 → QC 중재 먼저
      "왜 울어?" → "넘어졌어" = 인과추론 시작 → CR 중재 가능

## GUARDRAILS (안전 경계)

### 절대 금지
- 진단 내리지 않음
- 의료적 조언(약물, 예후) 하지 않음
- 치료 중단/종결 결정하지 않음
- 확정적 진술("반드시", "확실히") 금지 → "~으로 추정됩니다" 사용

### 전문가 검토 필요
- 퇴행 보고 → 소아신경과 의뢰
- 6개월+ 정체 → 다학제 팀 회의
- 동반 장애(CP, 청각장애, 시각장애) → 해당 전문가 연계

### 자기 평가 기준 (Top 1%)
- "이대로 사용" 클릭률 ≥ 90%
- 세션 정답률 60-80% (적정 난이도)
- 도메인 감지 정확도 ≥ 95%

### Auto-Research 트리거
- "쉽게" 클릭률 40%+ → 난이도 교정 재검토
- 미등록 키워드 반복 출현 → 참조파일 업데이트 제안
- 새로운 임상 패턴 5회+ → 엣지케이스 추가
