# SOCIAL DOMAIN: 사회성치료 전문가 레퍼런스

> 이 파일은 AI 에이전트가 한국의 사회성치료 전문가처럼 사고하고 의사결정하도록 만드는 임상 지식 베이스입니다.
> 대상: ASD 및 발달장애 아동 (만 2-12세).
> 핵심 원칙: 사회성은 "가르쳐야 할 규칙"이 아니라 "단계적으로 경험해야 할 기술"입니다.
> 학습지는 인지적 이해(cognitive understanding)를 지원하며, 실행(execution)은 직접 치료/중재 영역입니다.

## 1. 사회 발달 6단계 (Social Development Stages)

기능연령(functional age) 기준. 생활연령이 아닌 기능연령을 반드시 사용할 것.

### Stage 1: 사회적 인식 (기능연령 2-3세)

| 하위 영역 | 목표 기술 | 예시 활동 |
|-----------|----------|----------|
| 타인 존재 인식 | 다른 사람이 있음을 알아차림 | 사진에서 사람 찾기 |
| 사회적 참조 | 성인의 반응을 보고 상황 판단 | "엄마 표정 보기" 활동 |
| 공동 주의 기초 | 같은 대상 함께 보기 | 가리키기 → 따라보기 |
| 모방 기초 | 간단한 동작 따라하기 | 손뼉치기, 만세 |

**핵심 원리:** 이 단계의 아동은 타인에 대한 기본 인식 자체가 목표. 상호작용 요구는 부적절.

**학습지 적합성 판단:**
```
이 단계의 아동에게 종이 학습지는 제한적 효과.
→ 사회적 동기(social motivation) 자체가 부족한 경우:
  - 학습지보다 직접 중재(환경 구성, 선호 활동 내 사회적 요소 삽입)가 우선
  - 학습지 사용 시: 아동의 특별 관심사(special interest)를 매개로 한 활동만 가능
  - 예: 공룡을 좋아하는 아동 → "공룡 친구들" 이미지로 사회적 상황 노출
→ 공동 주의가 있으나 또래 무관심인 경우:
  - 학습지로 "또래 개념" 시각적 노출은 가능
  - 단, 실행은 직접 중재로만 가능
```

#### 1-1A. 사회적 동기 부재 프로토콜 강화 (Enhanced Social Motivation Absence Protocol)

```
ASD Level 3 + 또래에 전혀 관심 없음 → 3단계 판단 프레임워크:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
GATE 1: "학습지 이전에 직접 중재가 우선인가?" — 판단 기준
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

다음 4개 중 2개 이상 YES → 학습지 단독 사용 부적절. 직접 중재 우선.

  □ 공동 주의(joint attention) 불가: 성인이 가리키는 곳을 따라보지 않음
  □ 사물 관심 > 사람 관심: 방에 사람이 들어와도 반응 없음, 사물에만 몰두
  □ 사회적 참조(social referencing) 부재: 새로운 상황에서 보호자 얼굴을 보지 않음
  □ 모방 기초 없음: 간단한 동작 모방(손뼉, 만세)을 성인과도 하지 않음

→ 직접 중재 우선 사항 (학습지 범위 밖, 치료사 안내용):
  1. 환경 구성: 아동의 강화물(reinforcer)을 사회적 상호작용이 필요한 형태로 배치
     예: 좋아하는 장난감을 투명 용기에 → "열어주세요" 요구 유도
  2. 감각-사회 연계: 아동이 좋아하는 감각 경험(그네, 트램폴린)을 성인과의
     상호작용으로만 제공 → "더?" 요구 시작점
  3. 관심사 삽입: 특별 관심사 활동 중 자연스러운 사회적 루틴 삽입
     예: 자동차 정렬 좋아하면 → 성인이 차 하나씩 건네주기 → 기다림+요구 발생
  4. 반응성 상호작용(responsive interaction): 아동 주도 활동에 성인이 끼어들기
     예: 아동이 블록 쌓으면 → 성인이 하나 올려놓기 → 아동 반응 관찰

→ 치료사 안내문:
  "이 아동은 현재 사회적 동기 자체가 형성 초기 단계입니다.
   학습지 활동은 시기상조이며, 직접 중재를 통한 사회적 동기 촉진이 우선입니다.
   아래 학습지 보조 도구 활용은 직접 중재와 병행 시에만 의미가 있습니다."

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
GATE 2: "학습지가 보조 도구로 쓰일 수 있는 경우" — 6가지 조건
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

다음 중 1개 이상 YES → 직접 중재 + 학습지 보조 가능:

  □ 공동 주의 부분 가능: 관심 있는 대상에 대해서만 따라보기 가능
  □ 특별 관심사가 명확함: 특정 캐릭터/사물에 강한 선호 (매개체 활용 가능)
  □ 그림/사진에 주의 유지 가능: 시각적 자료를 10초 이상 주시
  □ 성인과의 최소 상호작용 존재: 원하는 것 있을 때 성인에게 접근
  □ 모방 기초 부분 존재: 좋아하는 활동 맥락에서 간단한 모방 가능
  □ 루틴 기반 예측 가능: 반복 루틴에서 다음 단계를 기대하는 모습

→ 학습지 보조 도구 활용법:
  A. 관심사 매개 사회적 노출 (Social Exposure via Special Interest):
     - 아동의 특별 관심사 캐릭터가 사회적 상황에 있는 장면
     - 목적: 사회적 맥락에 대한 시각적 친숙화 (행동 변화 기대 X)
     - 예: 기차 좋아하면 → "기차 친구들이 줄 서기" 장면 → 줄서기 개념 노출
     - 빈도: 직접 중재 세션 전/후 5분 이내 짧은 노출

  B. 루틴 시각 스케줄 보조 (Visual Schedule Support):
     - 치료 세션의 사회적 루틴을 사진/그림으로 시각화
     - 목적: 예측 가능성 증가 → 불안 감소 → 사회적 참여 기반 형성
     - 예: [놀이방 사진] → [선생님 사진] → [같이 놀기 그림] → [끝 그림]

  C. 사회적 관찰 앨범 (Social Observation Album):
     - 아동의 일상에서 사회적 장면을 사진으로 촬영하여 앨범 구성
     - 치료 세션에서 사진 보며 "여기 누가 있지?" 수준의 인식 활동
     - 학습지가 아닌 개별화 사진 자료 형태

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
GATE 3: "직접 중재 진행 중 → 학습지 전환 시점" — 전환 기준
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

다음 3개 모두 YES → Stage 1 학습지 본격 도입 가능:

  □ 성인에 대한 사회적 접근(social approach)이 자발적으로 발생
  □ 공동 주의가 관심사 외 대상으로도 확장 시작
  □ 간단한 사회적 루틴(인사, 주고받기)에서 자기 역할 인식

→ 전환 시 학습지 유형:
  - "사람이 있어요" 기초 사회적 인식 학습지 (사진 기반)
  - "같이 있어요" 물리적 근접 장면 학습지
  - "~주세요" 요구 스크립트 카드 (시각적 보조)
```

### Stage 2: 병행 놀이 (기능연령 3-4세)

| 하위 영역 | 목표 기술 | 예시 활동 |
|-----------|----------|----------|
| 물리적 근접 수용 | 같은 공간에서 다른 사람과 활동 | "옆에서 함께 그리기" 장면 |
| 물건 공유 공간 | 같은 재료를 사용하되 각자 활동 | 같은 블록통에서 각자 쌓기 |
| 관찰 학습 기초 | 다른 아이 활동 지켜보기 | "친구는 뭐 하고 있을까?" |
| 순서 대기 (2인) | 한 명이 할 때 기다리기 | "기다리기 → 내 차례" 2단계 시퀀스 |

**핵심 원리:** 병행 놀이는 "함께 놀기"가 아니라 "같은 공간에서 각자 놀기". 상호작용 기대 없음.

### Stage 3: 연합 놀이 / 기초 상호작용 (기능연령 4-5세)

| 하위 영역 | 목표 기술 | 예시 활동 |
|-----------|----------|----------|
| 인사하기 | 만남/헤어짐 시 적절한 인사 | 인사 순서맞추기 (3단계) |
| 간단한 놀이 요청 | "같이 놀자" 표현 | 말풍선 완성하기 |
| 물건 주고받기 | 요청-전달 상호작용 | "~주세요 / 여기 있어" 스크립트 |
| 순서 지키기 (3-4인) | 차례 기다리기 확장 | 순서판 + 순서 카드 |

**핵심 원리:** 명시적 스크립트(explicit script) 제공이 핵심. "이렇게 말하면 돼"를 시각적으로 보여줌.

### Stage 4: 협동 놀이 / 규칙 기반 상호작용 (기능연령 5-7세)

| 하위 영역 | 목표 기술 | 예시 활동 |
|-----------|----------|----------|
| 공동 목표 | 함께 하나의 결과물 만들기 | "함께 탑 쌓기" 순서맞추기 |
| 규칙 따르기 | 놀이/게임 규칙 이해 및 준수 | 규칙 카드 + 맞다/틀리다 판별 |
| 나누기/빌려주기 | 물건 공유 상호작용 | 나누기 시나리오 순서맞추기 |
| 승패 수용 | 이기거나 지는 상황 대처 | "이겼을 때/졌을 때" 행동 선택 |

**핵심 원리:** 규칙을 "아는 것"과 "지키는 것"은 다른 영역. 학습지는 "아는 것"을 담당. "지키는 것"은 직접 중재.

**지식-실행 갭(Knowledge-Execution Gap) 대응:**
```
아동이 규칙을 말로 설명할 수 있지만 실제로 못 지키는 경우:
→ 학습지 역할: 상황 인식 + 전략 시각화
  1. 상황 인식 학습지: "지금 무슨 상황이지?" (상황 판별)
  2. 전략 선택 학습지: "이때 어떻게 하면 좋을까?" (행동 선택지)
  3. 결과 예측 학습지: "이렇게 하면 어떻게 될까?" (결과 추론)
→ 직접 중재가 반드시 병행되어야 함을 명시
→ 학습지만으로 행동 변화를 기대하지 말 것
→ 치료사에게: "이 학습지는 인지적 준비(cognitive rehearsal)용입니다.
  실제 행동 연습은 구조화된 놀이 상황에서 직접 중재로 진행해주세요."
→ 심화: Section 11 '지식-실행 갭 브리지: 3단계 전이 프로토콜' 참조
```

### Stage 5: 관점 취하기 / 마음이론 (기능연령 7-9세)

| 하위 영역 | 목표 기술 | 예시 활동 |
|-----------|----------|----------|
| 타인 생각 추론 | 다른 사람의 생각 예측 | 생각풍선 채우기 |
| 의도 이해 | 실수 vs 고의 구별 | "일부러? 실수?" 판별 학습지 |
| 사회적 단서 읽기 | 비언어적 신호 해석 | 몸짓/표정 → 의도 매칭 |
| 예측과 결과 | 행동의 사회적 결과 예측 | "이렇게 하면?" 결과 추론 |
| 비유/농담 이해 기초 | 문자 그대로 vs 비유적 의미 구분 | "진짜 말 vs 농담 말" 판별 |

**핵심 원리:** 감정 영역의 Level 5(관점 취하기)와 겹침. 감정 추론이 선행되어야 사회적 관점 취하기가 가능.

**비유/농담/아이러니 이해 프로토콜:**
```
ASD 아동(특히 Level 1 고기능)의 문자적 해석(literal interpretation) 대응:

1단계 — 문자적 vs 비유적 구분 학습지:
  구조: 2분할 레이아웃
  왼쪽: 말풍선 + 문자 그대로 해석 이미지 (X 마크)
  오른쪽: 같은 말풍선 + 실제 의미 이미지 (O 마크)
  예: "배가 터지겠다" → 왼쪽: 배가 터지는 그림(X) / 오른쪽: 배 부른 아이(O)

2단계 — 맥락 단서 학습:
  구조: 상황 장면 + 말풍선 + "이 말은 무슨 뜻일까?" 질문
  맥락 단서 강조: 화자의 표정, 상황, 목소리 톤(시각적 표현)
  예: 웃으면서 "야, 진짜 잘한다~" → 표정(웃음) = 칭찬 or 놀림? 맥락으로 판단

3단계 — 농담 패턴 학습:
  한국 또래 문화 빈출 농담 유형:
  - 과장 ("백만 번째야!") → 실제 숫자가 아닌 "많다"는 뜻
  - 반어 ("아~ 진짜 좋겠다~" + 씁쓸한 표정) → 실제로는 안 좋음
  - 말장난 (동음이의어) → 같은 소리, 다른 뜻
  각 유형별 "단서 찾기" 활동: 표정/상황/말투에서 힌트 찾기

중요: 농담 "이해"와 농담 "사용"은 별개.
→ 학습지는 이해만 다룸. 사용은 직접 중재에서 자연스러운 맥락으로.
→ 이 영역은 사회성 + 언어 경계. 언어적 이해가 선행 조건.
  언어 기능연령 < 7세 → 비유/농담 학습지 부적절. 문자적 의사소통 먼저.

사회적 영향:
  "농담을 이해 못 해서 울거나 화내는" 상황 대처:
  Step 1: 감정 조절 (emotion.md) — "혼란스러울 때 멈추기" 전략
  Step 2: 확인 질문 스크립트 — "진짜야? 농담이야?" 말풍선 카드
  Step 3: 대처 행동 — "웃기면 같이 웃어도 돼, 모르겠으면 물어봐도 돼"
```

### Stage 6: 집단 역학 / 복잡한 사회적 상황 (기능연령 9세+)

| 하위 영역 | 목표 기술 | 예시 활동 |
|-----------|----------|----------|
| 집단 규범 이해 | 상황별 다른 행동 기준 | "학교에서 vs 집에서" 비교 |
| 간접적 사회 단서 | 은유, 아이러니, 빈정거림 이해 | "진짜로 하는 말? 아닌 말?" |
| 갈등 해결 전략 | 협상, 타협, 중재 | 갈등 시나리오 + 전략 선택 |
| 우정 유지 | 장기적 관계 기술 | 친구 관계 상황 순서맞추기 |
| 온라인 사회 규칙 | 디지털 공간의 사회적 규범 | 메신저/게임 상황 판별 (Section 12) |

**핵심 원리:** 추상적이고 맥락 의존적. 학습지로 인지적 틀(cognitive framework)을 제공하되, 일반화는 실생활에서만 가능.

### Stage 판별 질문

```
Step 1: 사회적 인식 수준 파악
  Q1. "다른 사람의 존재를 인식하고 반응하는가?"
      → NO → Stage 1 (사회적 인식 — 학습지 효과 제한적, 직접 중재 우선)
             → Section 1-1A 사회적 동기 부재 프로토콜 참조
      → YES ↓
  Q2. "다른 아이 옆에서 같은 공간을 공유할 수 있는가?"
      → NO → Stage 2 (병행 놀이)
      → YES ↓
  Q3. "간단한 인사, 놀이 요청 등 기초 상호작용이 가능한가?"
      → NO → Stage 3 (연합 놀이 / 기초 상호작용)
      → YES ↓
  Q4. "규칙이 있는 놀이에 참여하고 공동 목표를 이해하는가?"
      → NO → Stage 4 (협동 놀이 / 규칙 기반 상호작용)
      → YES ↓
  Q5. "다른 사람의 생각/의도를 추론할 수 있는가?"
      → NO → Stage 5 (관점 취하기 / 마음이론)
      → YES ↓
  Q6. "집단 규범, 간접적 의사소통, 갈등 해결이 가능한가?"
      → NO → Stage 6 (집단 역학)
      → YES → 유지/일반화 단계 (다양한 상황에서의 적용 연습)
```

### 중요: Stage 간 불일치 패턴

| 패턴 | 설명 | 대응 전략 |
|------|------|-----------|
| **규칙 암기 OK / 실행 NO** | 규칙을 말로 설명 가능하나 실제 상황에서 못 지킴 | Section 11 전이 프로토콜. 학습지→역할극→실제상황 3단계 |
| **1:1 OK / 그룹 NO** | 1:1 상호작용은 되지만 집단에서 무너짐 | Section 8-4 그룹 인원별 설계. 2인→3인→소그룹 점진 확장 |
| **성인과 OK / 또래 NO** | 성인과는 상호작용하나 또래에게 무관심 | 또래 특화 시나리오. 아동의 특별 관심사를 또래 상호작용 매개로 활용 |
| **인식 OK / 표현 NO** | 상황을 이해하나 적절한 말/행동을 못 함 | 말풍선+행동 화살표로 명시적 스크립트 제공 |
| **특정 상황 OK / 새 상황 NO** | 연습한 상황만 가능, 일반화 실패 | 같은 기술, 다른 시나리오로 다양한 변형 제공 |
| **단시간 OK / 장시간 NO** | 기술 습득은 됐으나 지속시간이 짧음 | Section 13 지속시간 기반 그레이딩 참조 |

## 2. ASD 특화 사회성 학습지 디자인 규칙

### 2-1. 명시적 규칙 표현 (Explicit Rule Visualization)

ASD 아동은 암묵적(implicit) 사회 규칙을 자연스럽게 습득하지 못합니다. 모든 규칙을 시각적으로 명시해야 합니다.

| 시각 도구 | 용도 | 디자인 규칙 |
|-----------|------|-------------|
| **생각풍선** (thought bubble) | 캐릭터의 내적 상태/생각 표현 | 구름 모양 테두리, 캐릭터 머리 위, 연결 원 3개 |
| **말풍선** (speech bubble) | 캐릭터가 말해야 할 것 | 둥근 사각형, 캐릭터 입 쪽 꼬리, 안에 한글 텍스트 |
| **행동 화살표** (action arrow) | 해야 할 행동의 방향/순서 | 굵은 초록색 화살표, 행동 주체→대상 방향 |
| **X 마크** (prohibition mark) | 하면 안 되는 행동 | 빨간 X 또는 빨간 동그라미+사선, 부적절 행동 위에 |
| **O 마크** (approval mark) | 해야 하는/적절한 행동 | 파란 O 또는 초록 체크, 적절 행동 위에 |
| **순서 번호** (step numbers) | 행동의 시간적 순서 | 큰 원 안에 숫자 ①②③, 각 장면 좌상단 |

### 2-2. 사회적 스크립트 (Social Scripts)

```
사회적 스크립트 구성 요소:
  1. 상황 설명: "언제/어디서" (시각적 배경)
  2. 상대방 행동: "친구가 ~함" (상대 캐릭터의 동작)
  3. 나의 생각: 생각풍선 "이럴 때는..." (인지적 준비)
  4. 나의 말: 말풍선 "~라고 말해요" (언어적 반응)
  5. 나의 행동: 행동 화살표 + 동작 이미지 (행동적 반응)
  6. 결과: 상대방 반응 (긍정적 피드백 장면)

스크립트 복잡도 조절:
  Stage 3: 요소 2-3개 (상황 + 말 + 결과)
  Stage 4: 요소 4개 (상황 + 생각 + 말 + 결과)
  Stage 5: 요소 5-6개 (상황 + 상대생각 + 내생각 + 말 + 행동 + 결과)
```

### 2-3. 적절/부적절 행동 대비 (Do / Don't Contrast)

```
레이아웃: 2분할 (좌/우 또는 상/하)

왼쪽/위: 부적절한 행동
  - 빨간 X 마크
  - 캐릭터가 부적절한 행동을 하는 장면
  - 상대방의 부정적 반응 (슬픈/화난 표정)
  - 캡션: "이렇게 하면..." (부정적 결과 암시)

오른쪽/아래: 적절한 행동
  - 초록 O/체크 마크
  - 캐릭터가 적절한 행동을 하는 장면
  - 상대방의 긍정적 반응 (행복한 표정)
  - 캡션: "이렇게 하면..." (긍정적 결과 암시)

주의사항:
  - 부적절 행동은 "위험한" 것이 아닌 "사회적으로 어색한" 수준만
  - 예: "장난감 빼앗기" (O) vs "때리기" (X — 학습지에 폭력 행동 묘사 금지)
  - 부적절 예시가 너무 매력적이면 안 됨 (모방 위험)
  - 적절 행동이 시각적으로 더 크고, 더 밝고, 더 매력적으로
```

### 2-4. 사회적 스크립트 카드 (Portable Script Cards)

```
목적: 실제 상황에서 휴대할 수 있는 미니 스크립트 카드
크기: 명함 크기 (90x50mm) 또는 라미네이팅 가능한 소형 카드
용도: 지식-실행 갭 브리지의 2단계(역할극) → 3단계(실제상황) 전이 보조

구성:
  앞면: 상황 그림 + "이때는..."
  뒷면: 말풍선 스크립트 + 행동 화살표 (1-3단계)

예시:
  앞면: [교실에서 친구 장난감이 갖고 싶은 장면]
  뒷면: ① 다가가기 → ② "빌려줄 수 있어?" → ③ 기다리기

사용법:
  - 치료 세션에서 역할극 시 참조
  - 학교/가정에서 실제 상황 발생 시 카드 꺼내 보기
  - 점진적 페이딩: 카드 보기 → 카드 없이 기억 → 자동화
```

## 3. 사회적 시나리오 5레벨 시퀀싱

사회적 기술은 반드시 아래 순서로 단계적으로 다룹니다.

### Level 1: 인사하기 (Greeting)

| 상황 | 스크립트 | Stage |
|------|---------|-------|
| 아침에 교실 들어갈 때 | 눈 마주침 → "안녕" → 가벼운 손 흔들기 | 3 |
| 친구 집에 갈 때 | 초인종 → 문 열리면 → "안녕, 놀러 왔어" | 3 |
| 헤어질 때 | 눈 마주침 → "안녕/잘 가" → 손 흔들기 | 3 |
| 어른에게 인사 | 눈 마주침 → 허리 인사 → "안녕하세요" | 3-4 |
| 처음 만나는 사람 | 눈 마주침 → "안녕하세요, 저는 ~예요" | 4 |

**ASD 특화:** 눈 마주침은 "상대방 코/이마 근처 보기"로 대체 가능. 강제적 눈 맞춤 요구 금지.

### Level 2: 놀이 요청 (Play Request)

| 상황 | 스크립트 | Stage |
|------|---------|-------|
| 같이 놀고 싶을 때 | 다가가기 → "같이 놀아도 돼?" → 대답 기다리기 | 3-4 |
| 놀이에 끼고 싶을 때 | 가까이 가기 → 관찰하기 → "나도 해도 돼?" | 4 |
| 거절당했을 때 | "알겠어" → 다른 놀이 찾기 (또는 나중에 다시 물어보기) | 4-5 |
| 놀이 제안하기 | "~하고 싶은데, 같이 할래?" → 상대 의견 듣기 | 4-5 |

### Level 3: 나누기와 교대 (Sharing & Turn-Taking)

| 상황 | 스크립트 | Stage |
|------|---------|-------|
| 장난감 함께 쓰기 | "먼저 해, 다 하면 나 줘" / "같이 쓰자" | 4 |
| 간식 나누기 | "하나 줄까?" → 건네기 → "고마워" 듣기 | 3-4 |
| 순서 기다리기 | "지금은 ~차례야" → 기다리기 → "이제 내 차례!" | 3-4 |
| 빌려주기/빌리기 | "잠깐 빌려줄 수 있어?" → "응/나중에" → 수용 | 4-5 |

**빌리기 사회 스크립트 상세 (하위 기술 분해):**
```
"빌리기"는 복합 사회 기술. 하위 기술 분해:

전제 조건:
  - 소유권 개념: "이건 친구 거야" (인지)
  - 요구 표현: "~해 줘" / "~줄 수 있어?" (언어)
  - 기다리기: 상대 반응을 기다림 (자기 조절)
  - 수용: "응" 또는 "안 돼" 모두 받아들이기 (감정 조절)

4단계 순서맞추기:
  ① [그림: 친구가 장난감 가지고 놀고 있음. 아동이 바라봄]
     생각풍선: "나도 하고 싶다"
  ② [그림: 아동이 친구에게 다가감]
     행동 화살표: 아동 → 친구 방향
  ③ [그림: 아동이 말함]
     말풍선: "빌려줄 수 있어?" (또는 기능연령 3-4세: "빌려줘")
     행동: 손 내밀기
  ④a [그림: 친구가 건네줌] (긍정 결과)
     말풍선: "응, 여기!"
     아동 반응: "고마워!"
  ④b [그림: 친구가 아직 쓰고 있음] (거절 결과)
     말풍선: "지금은 안 돼, 나중에"
     아동 반응: "알겠어" (다른 놀이 찾기)

주의: ④a와 ④b 두 가지 결과를 모두 보여주되,
  거절(④b)도 자연스러운 반응임을 시각적으로 명시 (X 마크 없음).
  "거절당해도 괜찮아" 메시지를 별도 칸에 표시.
```

### Level 4: 갈등 해결 (Conflict Resolution)

| 상황 | 스크립트 | Stage |
|------|---------|-------|
| 같은 장난감을 원할 때 | 상황 인식 → "먼저/나중에" 또는 "같이" 협상 | 4-5 |
| 친구가 규칙 안 지킬 때 | "우리 규칙은 ~인데" → 설명 → 선생님 도움 요청 | 5 |
| 실수로 피해를 줬을 때 | 상황 인식 → "미안해" → 보상 행동 | 4-5 |
| 오해가 생겼을 때 | "나는 ~인 줄 알았어" → 상대 이야기 듣기 | 5-6 |

**ASD 특화:** 갈등 해결은 감정 조절(emotion regulation)이 선행되어야 함. 감정이 격앙된 상태에서의 갈등 해결은 학습지 범위 밖.

### Level 5: 공감과 도움 (Empathy & Helping)

| 상황 | 스크립트 | Stage |
|------|---------|-------|
| 친구가 울고 있을 때 | 감정 인식 → 다가가기 → "괜찮아?" → 기다리기 | 5 |
| 친구가 도움이 필요할 때 | 상황 관찰 → "도와줄까?" → 도움 행동 | 5 |
| 친구에게 좋은 일이 있을 때 | 감정 인식 → "잘했다! / 축하해!" | 5 |
| 친구가 혼자 있을 때 | 관찰 → 다가가기 → "같이 놀래?" | 5-6 |

**공감 학습지 설계 핵심:**
```
공감 = 감정 인식(emotion) + 사회적 행동(social)
→ 두 도메인이 동시에 필요한 복합 활동

필수 선행 조건 확인:
  1. 감정 영역 Level 3+ (상황 기반 감정 추론 가능) — emotion.md 참조
  2. 사회성 Stage 4+ (기초 상호작용 스크립트 습득)

둘 다 충족 시:
  → 공감 학습지 진행 가능
  → 구조: 상황 장면 + 감정 인식 질문 + 행동 선택 질문 (2단계)
    Step 1: "이 친구 기분은 어떨까?" (감정 추론)
    Step 2: "이때 어떻게 하면 좋을까?" (사회적 행동 선택)

```

## 추가: 고독 선호 vs 능력 부족 감별

3가지 가설:
- A 진정한 고독 선호: 또래 접근 시 거부 안 하지만 적극 참여도 안 함. 혼자 놀이 시 안정.
- B 학습된 회피: 또래 접근 시 긴장/회피. 과거 부정적 경험. 안전환경에서 성인과는 OK.
- C 능력 부족 배제: 접근 시도하지만 부적절. 거부당한 후 혼자 전환. 어른 중재 시 가능.

자기보고 신뢰도: 기능연령 5세 미만 → 행동 관찰 우선

## 추가: 사회적 접근 전략 (거리/타이밍/강도)

하위기술 1 — 물리적 거리: "팔 하나 거리" 규칙, 동심원 다이어그램
하위기술 2 — 접근 타이밍: 관찰→판단(초록/노랑/빨강 신호등)→접근
하위기술 3 — 관심 표현 강도: "3문장→1질문" 규칙, 상대 반응 읽기

## 추가: 선의의 거짓말 프로토콜

정의 구분: 선의의 거짓말(감정보호) ≠ 농담(재미) ≠ 속이기(자기이익)
전제: 1차 마음이론 + 감정 예측 + 인지적 유연성 → 3/4+ 시 학습 가능
핵심: "거짓말"이 아닌 "배려하는 말" 프레이밍
한국 문화: 선의의 거짓말 = "예의"로 기대됨

## 추가: 반복 질문 대응

기능 분석 필수 (치료사 먼저 판별):
- A 불안: 답 듣고 잠시 안정 → 시각적 스케줄
- B 대화개시 부재: 답에 무관심 → "말 걸기 카드" 3종
- C 감각추구: 혼잣말로도 반복 → OT-SI 의뢰
- D 정보처리: 복잡설명 후 반복 → 언어 단순화

사회적 영향 최소화: "질문 카운터" 시각도구 + 대체 스크립트 카드
