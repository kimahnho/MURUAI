# MOTOR DOMAIN (소근육 발달) -- Specialist Reference

> **역할**: 이 파일을 읽은 AI 에이전트는 한국의 소근육/작업치료 전문가처럼 사고하고 응답한다.
> 대상: ASD 및 발달장애 아동 (만 0-12세).
> 목적: 치료 활동지(학습지) 생성 시 임상적으로 정확한 판단을 내리기 위한 지식 기반.
>
> **기능연령 계산**: SKILL.md 진단 매핑 테이블 적용.
> ASD L2 = 실제나이 - 1~2세, ASD L3 = 실제나이 - 2~3세, ID Moderate = 실제나이 / 2.
> 기능연령 기준으로 아래 이정표 참조.
>
> **7세+ 소근육**: 필기 유창성(속도+가독성), 복잡 도형 자르기, 도구 정교 사용.
> 8세+ 기본 쓰기 가능하면 학업 지원 영역으로 전환 고려.

## 1. 소근육 발달 이정표 (Fine Motor Milestones)

| 연령 | 잡기 (Grasp) | 그리기 (Drawing) | 조작 (Manipulation) | 자조 (Self-Care) |
|------|-------------|-----------------|--------------------|-----------------|
| 0-6개월 | 반사적 움켜쥠(reflexive grasp) → 의도적 잡기 출현 | 해당 없음 | 딸랑이 흔들기, 입으로 탐색 | 젖병에 손 가져가기 |
| 6-12개월 | 갈퀴잡기(raking) → 열등집기(inferior pincer) | 해당 없음 | 물건 옮겨잡기, 컵에 블록 넣기/빼기 | 손가락 음식 먹기, 컵 잡기 시도 |
| 1-2세 | 손바닥 잡기(palmar) → 엄지-검지 집기(pincer) 출현 | 끄적임(scribbling), 점 찍기 | 블록 2-3개 쌓기, 큰 구슬 넣기 | 숟가락 쥐기 시도 |
| 2-3세 | 디지털 잡기(digital pronate) 출현 | 수직선, 원형 끄적임 | 책장 넘기기, 큰 구슬 끼우기, 3-4개 블록 | 양말 벗기, 지퍼 잡기 |
| 3-4세 | 3점 잡기(tripod) 미성숙 형태 출현 | 수평선, 원, 가위로 직선 자르기 | 작은 구슬 끼우기, 점토 공 만들기 | 단추 풀기 시도 |
| 4-5세 | 3점 잡기 안정화 | 십자(+), 사각형, 곡선 자르기, 색칠 범위 내 | 젓가락 시도, 접기(반으로) | 단추 잠그기, 신발 신기 |
| 5-6세 | 성숙 3점 잡기(dynamic tripod) | 삼각형, 대각선, 이름 쓰기, 신발끈 시작 | 작은 물체 정밀 조작 | 지퍼 완전 조작 |
| 6-7세 | 성인형 연필 잡기 | 유창한 글씨, 복잡 도형 자르기, 다이아몬드 | 정교한 손가락 분리 운동 | 신발끈 묶기 |

### 1-1. 잡기 발달 단계 (Grasp Development Hierarchy)

```
[미성숙] ──────────────────────────────────────────────► [성숙]

1. 손바닥 잡기 (Palmar Supinate)    : 주먹으로 움켜쥠, 어깨 움직임
2. 디지털 잡기 (Digital Pronate)     : 손가락으로 잡되 손목 고정 안 됨
3. 미성숙 3점 잡기 (Static Tripod)  : 엄지+검지+중지, 손가락 움직임 적음
4. 성숙 3점 잡기 (Dynamic Tripod)   : 손가락 끝으로 정밀 조절, 유연한 움직임
```

### 1-2. 잡기 이전 단계 활동 (Pre-Grasp Activities)

> **중요**: 연필 잡기가 아직 불가능한 아동에게는 아래 활동이 선행되어야 한다.

| 활동 | 목표 근육/기능 | 예시 자료 |
|------|--------------|----------|
| 손가락 그림 그리기 (finger painting) | 손바닥 인식, 촉각 탐색 | 핑거페인트, 쉐이빙크림 |
| 점토/플레이도 누르기, 굴리기 | 손 내재근 강화, 양손 협응 | 점토, 밀가루 반죽 |
| 종이 찢기/구기기 | 양손 협응, 손가락 분리 | 신문지, 색종이 |
| 큰 크레용으로 끄적이기 | 손바닥 잡기 → 잡기 발달 유도 | 뭉툭한 크레용 (직경 3cm+) |
| 스티커 떼어 붙이기 | 엄지-검지 대립, pincer 발달 | 큰 스티커 (직경 3cm+) |
| 스폰지 짜기 | 손 전체 힘 조절 | 물놀이용 스폰지 |
| 집게 활동 | pincer 강화, 잡기 전이 | 빨래집게, 큰 핀셋 |
| 구슬 넣기 (큰 구멍) | 눈-손 협응 기초 | 구멍 직경 3cm+ 보드 |

**판단 기준**: 아동이 큰 크레용을 손바닥으로 잡고 2-3초 유지할 수 있을 때 → 선 긋기 활동 도입 가능.

## 1A. 극저연령 프로토콜 (0-2세 Early Intervention Protocol)

> **핵심 원칙**: 0-2세 아동에게는 학습지(워크시트) 기반 활동이 발달적으로 부적절하다. 이 연령대는 실물 감각활동(hands-on sensory activities)이 우선이다.

### 1A-1. 왜 학습지가 부적절한가

```
[0-2세 발달 특성]
- 감각운동기(Piaget): 실물 조작을 통한 학습이 주 경로
- 주의 집중 시간: 1-3분 (과제 지속 불가)
- 도구 사용: 크레용 잡기조차 불안정 (palmar grasp 초기)
- 2D 이해 한계: 종이 위 그림과 실물의 연결이 미성숙
- 앉기 자세: 독립적 상체 안정성 불완전 (특히 12-18개월)

→ 결론: 20개월 발달지연 아동에게 학습지 제공 = 부적절. 실물 감각활동 우선.
```

### 1A-2. 0-2세 권장 활동 목록 (학습지 대체)

| 연령 | 활동 | 발달 목표 | 필요 자료 |
|------|------|----------|----------|
| 0-6개월 | 딸랑이 흔들기/쥐기 | 반사적 잡기 → 의도적 잡기 전환 | 가벼운 딸랑이, 촉감 장난감 |
| 6-12개월 | 용기에 넣기/빼기 | 놓기(release) 발달, 원인-결과 인지 | 큰 용기, 블록 |
| 6-12개월 | 드럼 치기/두드리기 | 양손 사용, 힘 조절 시작 | 작은 드럼, 냄비 |
| 12-18개월 | 큰 블록 쌓기 (2-3개) | 눈-손 협응, 놓기 조절 | 직경 5cm+ 블록 |
| 12-18개월 | 핑거페인팅 | 촉각 탐색, 손 인식, 흔적 남기기 경험 | 무독성 핑거페인트, 큰 종이/트레이 |
| 18-24개월 | 큰 크레용 끄적이기 | 도구 잡기 시작, 끄적임(scribble) 출현 | 뭉툭한 크레용(직경 3cm+), 큰 종이(A3+) |
| 18-24개월 | 모래/쌀 놀이 (퍼내기) | 도구 사용 시작, 손목 회전 | 모래/쌀, 작은 컵, 숟가락 |
| 18-24개월 | 큰 구슬 끼우기 | pincer grasp 강화, 눈-손 협응 | 구슬 직경 4cm+, 굵은 끈 |
| 18-24개월 | 뚜껑 열기/닫기 | 양손 협응, 손목 회전 | 다양한 크기 용기 |
| 12-24개월 | 물놀이 (붓기/짜기) | 손 전체 힘 조절, 감각 탐색 | 물, 스폰지, 컵 |

### 1A-3. 극저연령 의사결정 흐름

```
Q: 아동 생활연령 또는 발달연령이 0-2세인가?
├─ YES
│   Q: 발달연령이 12개월 미만인가?
│   ├─ YES → 실물 감각 탐색 활동만 제공 (딸랑이, 넣기/빼기, 촉감 놀이)
│   │         ⚠ 학습지 절대 제공하지 않음
│   └─ NO (12-24개월)
│       Q: 큰 크레용을 손바닥으로 잡을 수 있는가?
│       ├─ NO → 잡기 이전 활동 (핑거페인팅, 점토, 모래놀이)
│       │       ⚠ 학습지 부적절 → 실물 활동만
│       └─ YES → 큰 종이(A3+)에 자유 끄적이기 허용
│               → 선 따라 긋기 학습지는 아직 부적절
│               → 끄적임에서 방향성 출현 시까지 대기
└─ NO → 일반 프로토콜 (섹션 2 이후)

⚠ "20개월인데 학습지 써도 될까요?" → 부적절. 실물 감각활동 우선.
   대안: 핑거페인팅, 모래글씨, 큰 크레용 자유 끄적이기 (A3 이상 큰 종이)
```

### 1A-4. 극저연령 끄적이기 발달 판단

```
[끄적이기 발달 단계 (12-30개월)]

1단계: 무질서 끄적이기 (Disordered scribble, ~12-16개월)
  - 팔 전체 움직임, 무작위 방향
  - ✅ 정상 발달. "끄적이기만 해요" = 이 단계에서 정상
  - 다음 목표: 끄적이는 시간 늘리기, 다양한 도구 경험

2단계: 조절된 끄적이기 (Controlled scribble, ~16-20개월)
  - 반복 동작 출현 (왔다갔다, 동그라미 비슷한 움직임)
  - 팔 → 손목 움직임으로 전환 시작
  - 다음 목표: 방향성 있는 움직임 격려

3단계: 이름 붙이는 끄적이기 (Named scribble, ~20-30개월)
  - "이거 엄마야" 등 끄적인 것에 의미 부여
  - 원형/수직 방향성 출현
  - 다음 목표: 수직선 따라 긋기(L1) 도입 시도 가능

→ 2.5세(30개월)까지 끄적이기만 하는 것은 정상 범위 내.
→ 30개월 이후에도 1단계(무질서) 수준이면 발달 지연 가능 → 평가 권장.
```

## 2. 쓰기 전 기술 위계 (Pre-Writing Skills Hierarchy)

```
[1단계]        [2단계]       [3단계]       [4단계]        [5단계]
끄적임    →   수직선(ㅣ)  →  수평선(ㅡ)  →   원(O)    →   십자(+)
(scribble)    (vertical)   (horizontal)   (circle)     (cross)

[6단계]        [7단계]       [8단계]       [9단계]        [10단계]
우대각선(/)  →  사각형(□)  →  좌대각선(\) →  삼각형(△)  →  다이아몬드(◇)
(R diagonal)   (square)    (L diagonal)   (triangle)    (diamond)
```

### 각 단계별 선행 조건 및 전형적 달성 연령

| 단계 | 도형 | 전형 연령 | 선행 조건 | 실패 시 확인 사항 |
|------|------|---------|----------|-----------------|
| 1 | 끄적임 | 12-18개월 | 도구 잡기 가능 | 잡기 발달 확인 |
| 2 | 수직선 | 2세 | 끄적임에서 방향성 출현 | 어깨 안정성, 시작-끝 개념 |
| 3 | 수평선 | 2.5세 | 수직선 가능 | 손목 회전, 좌→우 방향 |
| 4 | 원 | 3세 | 수직+수평선 | 연속 곡선 움직임, 시작=끝 개념 |
| 5 | 십자 | 3.5-4세 | 수직+수평선 교차 | 선 교차점 인지, 계획 능력 |
| 6 | 우대각선 | 4-4.5세 | 수직+수평선 안정 | 대각선 방향 인지 (시각-공간) |
| 7 | 사각형 | 4.5세 | 수평+수직선+방향전환 | 4번 멈추고 방향 바꾸기 |
| 8 | 좌대각선 | 5세 | 우대각선 | 비우세측 대각선 |
| 9 | 삼각형 | 5-5.5세 | 대각선+방향전환 | 3번 꺾기, 각도 인지 |
| 10 | 다이아몬드 | 6-7세 | 삼각형+좌우대각선 | 4개 대각선 연결 |

### 2-1. 십자(+) 그리기 실패 분석 및 중재 (Cross Drawing Failure Analysis)

> 십자(+)는 "두 선의 교차"라는 계획 능력이 필요한 중요한 전환점이다.

```
[십자 그리기 실패 유형별 분석]

유형 1: 두 선이 만나지 않음 (교차 실패)
  - 원인: 시각-운동 협응의 "교차점 조준(intersection targeting)" 미발달
  - 하위 기술 분해:
    a) 첫 번째 선을 긋고 → 그 선의 중간 지점을 시각적으로 파악
    b) 두 번째 선의 시작점을 첫 번째 선 위에 정확히 배치
    c) 첫 번째 선을 가로질러 계속 진행
  - 중재:
    Step 1: 수직선 위에 큰 점(●) 표시 → "이 점에서 시작해서 옆으로"
    Step 2: 교차점에 별(★) 표시 → "별을 지나가게"
    Step 3: 교차점 마커 크기 점차 축소
    Step 4: 마커 없이 십자 그리기

유형 2: 선이 비스듬하게 만남 (수직/수평 유지 실패)
  - 원인: 수직선/수평선 안정성 부족
  - 중재: 수직선, 수평선 개별 연습 강화 (L2-L3 반복)

유형 3: T자나 L자만 그림 (계획 실패)
  - 원인: "교차"라는 개념 미이해
  - 중재:
    Step 1: 완성된 십자 위에 덧그리기 (tracing)
    Step 2: 수직선만 그려진 상태에서 수평선 추가하기
    Step 3: 빈 공간에 십자 그리기
```

## 3. 선 따라 긋기 활동 레벨 (Line Tracing Activity Levels)

### 레벨 체계

| 레벨 | 선 유형 | 선 폭 | 가이드 | 적합 대상 |
|------|--------|-------|--------|----------|
| L1 | 굵은 직선 | 2cm | 시작점(별★), 끝점(별★), 방향 화살표 | 끄적임→직선 전이기 |
| L2 | 보통 직선 | 1cm | 시작점(큰 점●), 끝점(큰 점●) | 직선 긋기 연습 |
| L3 | 가는 직선 | 0.5cm | 시작점(작은 점), 끝점(작은 점) | 직선 숙달기 |
| L4 | 완만한 곡선 (넓은 S) | 1.5cm | 시작/끝 마커, 점선 가이드 | 곡선 도입 |
| L5 | 지그재그, 꺾은선 | 1cm | 꺾이는 점에 점 표시 | 방향 전환 연습 |
| L6 | 복잡한 곡선, 나선 | 0.5cm | 시작점만 | 곡선 숙달기 |
| L7 | 한글 자모 따라쓰기 (기본 자음) | 획 폭 0.5cm, 글자 크기 5cm+ | 획순 번호, 시작 화살표 | 쓰기 준비 단계 |
| L8 | 한글 음절 따라쓰기, 미로 | 글자 크기 3-4cm | 획순만 | 쓰기 실행 단계 |

### 레벨 선택 의사결정 흐름

```
Q: 아동이 크레용/연필을 잡을 수 있는가?
├─ NO → 잡기 이전 활동 (섹션 1-2)
└─ YES
   Q: 끄적임에서 방향성이 보이는가?
   ├─ NO → L1 (굵은 직선, 최대 가이드)
   └─ YES
      Q: 직선(수직/수평)을 따라 그을 수 있는가?
      ├─ NO → L1-L2
      └─ YES
         Q: 원/곡선을 따라 그을 수 있는가?
         ├─ NO → L3-L4
         └─ YES
            Q: 방향 전환(꺾기)이 가능한가?
            ├─ NO → L5
            └─ YES
               Q: 목표가 한글 쓰기인가?
               ├─ YES → L7-L8
               └─ NO → L6 (복잡 곡선/나선)
```

## 4. 감각과민 조절 가이드 (Sensory Sensitivity Adjustments)

### 4-1. 시각과민 (Visual Hypersensitivity)

| 요소 | 일반 자료 | 감각과민용 자료 |
|------|----------|---------------|
| 선 색상 | 검정 (#000000) | 회색 (#999999) 또는 진한 파스텔 (#6B8E9B) |
| 배경색 | 흰색 (#FFFFFF) | 연한 크림 (#FFF8E7) 또는 연한 회색 (#F5F5F0) |
| 선 스타일 | 실선 (solid) | 점선 (dotted) 또는 파선 (dashed) |
| 선 폭 | 레벨 기준 | 레벨 기준 + 0.5cm 추가 (여유 공간) |
| 시작/끝 마커 | 밝은 색 별/점 | 연한 파란색 (#A8C8D8) 점, 크기 키움 |
| 이미지/캐릭터 | 컬러풀 | 단색 라인아트 또는 제거 |
| 페이지 여백 | 보통 | 넓게 (상하좌우 3cm+) |

### 4-2. 피해야 할 시각 요소

- **빨강/주황 계열** (#FF0000 ~ #FF6600): 각성 과잉 유발, 감각과민 아동에게 불안 촉발
- **높은 대비** (흑백 강한 대비): 시각 자극 과부하
- **빽빽한 패턴 반복**: 시각 피로, 집중 방해
- **번쩍이는/반짝이는 효과**: 디지털 자료에서 애니메이션 제거
- **작은 글씨/마커**: 눈 피로 유발

### 4-3. 촉각과민 (Tactile Hypersensitivity) -- 학습지 관련

- 종이 재질: 매끈한 코팅지 선호 (거친 종이 거부 가능)
- 도구: 부드러운 그립의 연필/크레용 추천
- 스티커: 재질 민감도 확인 (끈적임 거부 아동 있음)

### 4-4. 촉각과민 도구 거부 프로토콜 (Tool Refusal Due to Tactile Defensiveness)

> **핵심 상황**: 연필/크레용 잡기 자체를 거부하는 아동. 촉각 방어(tactile defensiveness)로 인해 도구를 손에 쥐는 것 자체가 불쾌한 경우.

```
[도구 거부 감별진단]

거부 원인 확인:
├─ 촉각과민 (도구 재질/촉감이 싫음) → 본 프로토콜 적용
├─ 잡기 근력 부족 (쥘 수 없어서) → 섹션 1-2 잡기 이전 활동
├─ 과제 거부 (하기 싫어서) → 동기부여/행동 전략 (motor 영역 외)
└─ 경험 부족 (도구를 처음 봄) → 탐색 시간 제공 후 재평가
```

#### 대체 도구 위계 (도구 없음 → 도구 사용)

| 단계 | 도구/매체 | 촉각 부하 | 활동 예시 | 목표 |
|------|----------|----------|----------|------|
| 0 | 도구 없음 - 손가락 | 최저 | 모래 위에 손가락으로 그리기, 쉐이빙크림 위 그리기 | 그리기 경험 자체, 흔적 남기기 재미 |
| 1 | 손가락 페인팅 | 낮음 | 큰 종이에 핑거페인트로 자유 그리기 | 손-매체 접촉 허용, 표현 경험 |
| 2 | 모래 글씨/트레이 | 낮음 | 모래 트레이에 손가락으로 선/모양 | 선 긋기 개념 (도구 없이) |
| 3 | 뭉툭한 크레용 (직경 3cm+) | 중간 | 큰 종이에 자유 끄적이기 | 도구 잡기 경험, 짧은 시간 허용 |
| 4 | 굵은 마커 (캡 포함 직경 2cm+) | 중간 | 점 찍기, 짧은 선 | 적은 힘으로 선 보임 → 성공 경험 |
| 5 | 부드러운 그립 연필 (그립 어댑터) | 중간-높음 | L1 선 따라 긋기 | 표준 도구로 전환 시작 |
| 6 | 일반 크레용/연필 | 높음 | 일반 활동 | 표준 도구 사용 |

#### 점진적 도구 전환 프로토콜

```
[전환 원칙]
1. 현재 허용하는 매체에서 시작 (절대 강요하지 않음)
2. 한 단계에서 3회 이상 거부 없이 사용 → 다음 단계 시도
3. 다음 단계 거부 시 → 즉시 이전 단계로 복귀 (좌절 방지)
4. 감각 식단(sensory diet)과 병행 권장 (OT 감각통합 치료)
5. 세션 시작 시 선호 도구로 워밍업 → 중간에 다음 단계 도구 잠깐 시도 → 다시 선호 도구로 마무리

[구체적 전환 전략]
모래 글씨 → 손가락 페인팅: "모래에서 했던 것처럼 여기에도 해볼까?"
손가락 페인팅 → 뭉툭한 크레용: 크레용을 페인트에 살짝 찍어 사용 (친숙화)
크레용 → 마커: 크레용과 마커 둘 다 제공, 선택하게 함
마커 → 연필: 그립 어댑터 + 부드러운 4B 연필부터

[치료사 표현 예시]
"오늘은 손가락으로 그려볼까? 크레용은 테이블에 놓아둘게, 만져보고 싶으면 만져봐."
→ 압력 없이 노출(exposure) 제공
```

## 5. 시각-운동 협응 활동 (Visual-Motor Coordination)

### 5-1. 점 잇기 (Dot-to-Dot)

| 난이도 | 점 수 | 점 간격 | 연결선 유형 | 숫자/순서 표시 |
|--------|-------|--------|-----------|--------------|
| 입문 | 3-5개 | 5cm+ | 직선만 | 큰 숫자 (2cm), 색상 구분 |
| 초급 | 5-10개 | 3-5cm | 직선 | 숫자 (1.5cm) |
| 중급 | 10-20개 | 2-3cm | 직선+완만곡선 | 숫자 (1cm) |
| 상급 | 20개+ | 1-2cm | 다양한 방향 | 작은 숫자 |

#### 5-1A. 점잇기 실패 분석: 순서 유지 문제

> 점잇기는 소근육 + 숫자 인지 + 시각 추적 + 작업 기억의 복합 과제이다.

```
[점잇기에서 "5개부터 순서를 잃어버린다" 분석]

전제조건 확인:
1. 숫자 인지: 해당 숫자까지 순서를 알고 있는가?
   - 1-3은 알지만 4-5부터 모름 → 인지(숫자) 문제, motor 문제 아님
   - 숫자는 아는데 순서를 건너뜀 → 시각 스캐닝/작업 기억 문제

2. 시각 추적(visual scanning):
   - 3개: 시야 내 한눈에 보임 → 스캐닝 불필요
   - 5개+: 다음 숫자를 "찾아야" 함 → 시각 스캐닝 능력 필요
   - 찾지 못하면 → 시각 스캐닝 훈련 선행

3. 작업 기억(working memory):
   - "지금 몇 번까지 했지?" 기억 유지 필요
   - 3개: 부하 낮음, 5개+: 부하 증가

[중재 전략]
- 숫자 모름 → 숫자 인지 과제 선행 (인지 영역), 점잇기는 색상 순서로 대체
- 시각 스캐닝 약함 → 점 사이 거리 줄이기, 다음 점에 색상 힌트
- 작업 기억 약함 → 완료한 점에 스티커 붙이기 (시각 마커), 연결선 색 유지
- 복합 → 점 수 3개 유지하면서 다양한 모양 만들기 (소근육 초점 유지)

[점잇기 vs 인지 경계 판단]
점잇기 실패 = 무조건 소근육 문제? → NO
"점까지 선은 잘 긋는데 다음 점을 못 찾는다" → 인지/시각 문제
"다음 점은 찾는데 선이 점에 도달하지 못한다" → 소근육/시각-운동 문제
→ 어디서 실패하는지 관찰이 핵심
```

### 5-2. 미로 (Maze)

| 난이도 | 통로 폭 | 분기점 | 막다른 길 | 크기 |
|--------|--------|--------|----------|------|
| 입문 | 3cm+ | 1-2개 | 없음 | A5 크기 |
| 초급 | 2cm | 2-3개 | 1개 | A4 크기 |
| 중급 | 1.5cm | 3-5개 | 2-3개 | A4 크기 |
| 상급 | 1cm | 5개+ | 다수 | A4 크기 |

#### 5-2A. 미로 활동 하위 기술 분석 및 벽 무시 문제

> 미로는 단순 소근육이 아니라 시각 추적 + 계획(planning) + 억제(inhibition) + 소근육의 복합 과제이다.

```
[미로의 구성 하위 기술]

1. 시각 추적 (Visual Tracking)
   - 통로를 눈으로 먼저 따라가며 경로 파악
   - 실패 시: 연필을 대고 바로 출발 → 막다른 길에서 당황

2. 계획 (Planning/Executive Function)
   - 분기점에서 어느 쪽이 맞는지 예측
   - 실패 시: 무작위 방향 선택, 반복적 막다른 길

3. 억제 (Inhibition)
   - 벽을 보고 "멈추는" 능력
   - 직선으로 가고 싶은 충동을 억제
   - 실패 시: 벽을 무시하고 직선으로 통과

4. 소근육 조절 (Fine Motor Control)
   - 통로 안에서 선을 유지하는 능력
   - 실패 시: 통로를 벗어남 (선 폭 관리 불가)

[벽 무시 행동 분석: "직선으로 가요"]

원인 분류:
├─ 억제 기능 미성숙/결함 (가능성 높음)
│   - 벽의 의미를 알지만 멈추지 못함
│   - ADHD 또는 충동성 높은 아동에서 빈번
│   - 확인 방법: "여기 벽이야, 지나가면 안 돼" → 알지만 반복
│
├─ 규칙 이해 부족
│   - 벽 = 지나갈 수 없다는 규칙 자체를 모름
│   - 확인 방법: "벽은 뭘까?" 질문 → 모른다고 하면 이것
│   - 연령: 3-4세에서 정상적으로 가능
│
├─ 시각 변별 어려움
│   - 통로와 벽의 경계를 시각적으로 구분 못함
│   - 확인 방법: 통로를 색칠(연한 색)하면 나아지는지 확인
│
└─ 과제 동기/관심 부족
    - 미로 자체에 흥미 없음
    - 확인 방법: 관심 주제 미로(좋아하는 캐릭터)로 변경

[중재 전략: 벽 무시]

단계 1: 물리적 미로 (3D) 먼저
  - 실물 미로(블록, 상자)에서 장난감 캐릭터 이동시키기
  - 벽에 부딪히면 "쿵! 못 가!" → 신체 경험으로 규칙 학습

단계 2: 손가락 미로 (연필 없이)
  - 큰 미로를 출력하고 손가락으로 따라가기
  - 벽 터치 시 "삐!" 소리 → 게임화

단계 3: 굵은 통로 미로 + 벽 강조
  - 통로 폭 3cm+, 벽을 굵은 빨간선으로
  - 벽에 "X" 표시 또는 벽돌 무늬

단계 4: 점진적 벽 축소, 통로 좁히기
  - 벽 색상을 점차 일반(검정)으로
  - 통로 폭 점차 줄이기

[ADHD 가능성 고려]
미로에서 벽 무시가 반복되고 + 다른 과제에서도 충동성 → ADHD 선별 권장
→ 그러나 6세 미만은 정상 발달에서도 억제 미성숙 가능
→ 다른 영역(주의력, 과잉행동)과 종합 판단 필요
```

### 5-3. 따라 그리기 / 모양 복사 (Copy Patterns)

| 난이도 | 활동 | 보조 | 선행 기술 |
|--------|------|------|----------|
| 입문 | 동일 모양 위에 덧그리기 (tracing) | 점선 가이드 | 잡기 가능 |
| 초급 | 바로 옆에 같은 모양 그리기 | 격자 가이드 | 직선/원 가능 |
| 중급 | 아래에 같은 모양 그리기 | 격자만 | 기본 도형 가능 |
| 상급 | 보고 빈 공간에 그리기 | 없음 | 복합 도형 가능 |

### 5-4. 좌우 대칭 그리기 (Symmetry Drawing)

> 일반 선 따라 긋기와는 **별도 활동 유형**. 시각-공간 인지와 소근육을 동시에 훈련.

| 난이도 | 활동 설명 | 격자 | 대칭축 |
|--------|----------|------|--------|
| 입문 | 대칭축 왼쪽 완성 → 오른쪽 점선 따라 그리기 | 큰 격자 (2cm) | 굵은 세로선, 색상 표시 |
| 초급 | 왼쪽 완성 → 오른쪽 격자 위에 그리기 | 중간 격자 (1.5cm) | 세로선 |
| 중급 | 왼쪽 완성 → 오른쪽 빈 공간 그리기 | 없음 | 점선 축 |
| 상급 | 복잡 패턴 대칭 (곡선 포함) | 없음 | 가는 점선 |

**패턴 유형**: 직선 조합 → 직선+곡선 → 곡선 중심 → 복합 도형
**주의**: ASD 아동은 대칭에 강점을 보이는 경우가 많음 (시스테마이징 성향). 흥미 유발에 효과적이나, 반복 고착(fixation)에 주의.

| ㅡ | 수평선 1획 | 수평선 | ★ |
| ㅣ | 수직선 1획 | 수직선 | ★ |
| ㅐ | ㅏ + 수직선 | ㅏ 가능 | ★★ |
| ㅔ | ㅓ + 수직선 | ㅓ 가능 | ★★ |

### 6-4. 쓰기 준비도 평가 체크리스트

한글 쓰기 활동을 시작하기 전에 반드시 확인:

```
□ 연필 잡기: 최소 미성숙 3점 잡기(static tripod) 이상
□ 수직선: 위→아래 직선 긋기 가능 (5cm 이상)
□ 수평선: 좌→우 직선 긋기 가능 (5cm 이상)
□ 방향 전환: 선을 긋다가 꺾을 수 있음 (ㄱ의 전제)
□ 모방 능력: 시범을 보면 따라할 수 있음
□ 시각 변별: ㄱ과 ㄴ의 차이를 인식할 수 있음

→ 6개 중 4개 이상 충족 시 자모 따라쓰기 도입 가능
→ 미충족 시 선 따라 긋기(L1-L5)에서 더 연습
```

### 6-5. 한글 자모 의사결정 자동 분기 (Hangul Letter Request Decision Flow)

> 치료사가 특정 자모 따라쓰기를 요청할 때, 선행 자모 확인 → 가능/대안 자동 분기

```
[자모 요청 시 자동 분기 프로토콜]

치료사: "ㅎ 따라쓰기 활동지 만들어주세요"

Step 1: ㅎ의 선행 자모 확인
  ㅎ (순서 13) → 선행 필수: ㅇ(3), 수평선, 점(수직 짧은선)

Step 2: 선행 자모 달성 여부 확인
  Q: 아동이 ㅇ(원)을 쓸 수 있는가?
  ├─ YES → Step 3으로
  └─ NO → ⚠ "ㅎ 쓰기 전에 ㅇ(원 그리기)부터 연습이 필요합니다"
          → 대안: ㅇ 따라쓰기 활동지 제공
          → 추가 확인: 원 그리기 자체가 안 되면 → 원 이전 단계(L4 곡선)

  Q: 수평선을 안정적으로 그을 수 있는가?
  ├─ YES → Step 3으로
  └─ NO → ⚠ "ㅎ는 수평선이 필요합니다. 수평선 연습부터"
          → 대안: L2-L3 수평선 따라 긋기

Step 3: ㅎ 활동지 제공 가능 판단
  → 선행 조건 충족 → ㅎ 따라쓰기 활동지 생성
  → 부분 충족 → 충족된 선행 자모 + 부족한 기술 연습 복합 활동지
```

```
[전체 자모 선행 체크 매트릭스]

요청 자모 → 필수 확인 사항 → 미충족 시 대안

ㄴ(1순위) → 수직선+수평선+꺾기 → L5(꺾은선) 연습
ㄱ(2순위) → 수평선+수직선+꺾기 → L5(꺾은선) 연습
ㅇ(3순위) → 원 그리기(L4) → L4(곡선) 연습
ㄷ(4순위) → ㄴ 가능 확인 → ㄴ 따라쓰기
ㅁ(5순위) → 사각형 그리기 → 사각형 따라 그리기
ㅋ(6순위) → ㄱ 가능 확인 → ㄱ 따라쓰기
ㅅ(7순위) → 양쪽 대각선 → L6(대각선) 연습
ㅂ(8순위) → ㅁ 가능 확인 → ㅁ 따라쓰기
ㅈ(9순위) → ㅅ 가능 확인 → ㅅ 따라쓰기
ㅊ(10순위) → ㅈ 가능 확인 → ㅈ 따라쓰기
ㅌ(11순위) → ㄷ 가능 확인 → ㄷ 따라쓰기
ㅍ(12순위) → ㅁ+수평선 확인 → ㅁ 따라쓰기
ㅎ(13순위) → ㅇ+수평선 확인 → ㅇ 따라쓰기
ㄹ(14순위) → ㄱ+ㄴ 능숙 확인 → ㄱ,ㄴ 반복 연습

→ 선행 자모가 미충족이면 무조건 대안 제공.
→ "아직 ㅎ는 어렵습니다" + 이유 + 대안 활동을 함께 출력.
→ 치료사가 "그래도 ㅎ로 해주세요" → 최대 보조(점선, 큰 글자, 시작점) + 선행 기술 연습 병행 권장 안내.
```

## 7. 복합 활동 설계 가이드 (Cross-Domain Activity Design)

### 7-1. 소근육 + 인지 복합 활동

> 하나의 학습지에 소근육 과제와 인지 과제를 통합할 때의 원칙.

**설계 원칙**:
1. **주 과제(primary)** 하나 + **부 과제(secondary)** 하나만 결합
2. 주 과제 난이도를 **1단계 낮춤** (인지 부하 고려)
3. 두 과제가 자연스럽게 연결되어야 함 (억지 결합 금지)

**복합 활동 유형**:

| 소근육 과제 | + 인지 과제 | 활동 예시 |
|------------|-----------|----------|
| 선 긋기 | 같은 것 찾기 (매칭) | 왼쪽 그림에서 오른쪽 같은 그림으로 선 긋기 |
| 선 긋기 | 분류 (카테고리) | 과일은 빨간 바구니로, 동물은 파란 바구니로 선 긋기 |
| 색칠하기 | 숫자 인지 | 숫자별 지정 색으로 칠하기 (색칠 by number) |
| 미로 | 순서 (시퀀싱) | 미로 안에 순서대로 지나가야 할 그림 배치 |
| 점 잇기 | 수 세기 | 숫자 순서대로 점 잇기 |
| 오리기 | 모양 변별 | 같은 모양만 골라 오리기 |
| 따라 그리기 | 패턴 완성 | 반복 패턴에서 빠진 부분 그려넣기 |

**주의사항**:
- 인지 과제가 소근육을 방해하면 안 됨 (예: 계산하면서 가는 선 긋기 = 과부하)
- ASD 아동의 경우 시각적 단서를 충분히 제공
- 복합 활동은 단일 활동을 각각 수행할 수 있을 때만 도입

### 7-2. 소근육 + 언어 복합

| 소근육 과제 | + 언어 과제 | 활동 예시 |
|------------|-----------|----------|
| 선 긋기 | 어휘 | 그림 이름 말하면서 해당 그림으로 선 긋기 |
| 자모 따라쓰기 | 음운 인식 | ㄱ 쓰고 → ㄱ으로 시작하는 그림 찾기 |
| 색칠하기 | 지시 따르기 | "사과를 빨간색으로 칠하세요" 문장 이해 |

## 8. 가위질 발달 단계 (Scissor Skills Development)

### 8-1. 가위질 전제조건

```
□ 양손 협응: 한 손은 종이, 한 손은 가위
□ 손가락 분리 운동: 엄지-검지-중지를 독립 움직임
□ 잡기 힘: 가위를 열고 닫는 힘 (스프링 가위로 시작 가능)
□ 눈-손 협응: 선을 보며 손을 움직이기
```

### 8-2. 단계별 가위질 활동

| 단계 | 활동 | 자료 사양 | 보조 |
|------|------|---------|------|
| 0 | 종이 찢기 (가위 전) | 얇은 종이 | 없음 |
| 1 | 한 번 자르기 (snip) | 폭 2cm 종이 띠 | 굵은 선 표시 |
| 2 | 직선 자르기 (짧은) | 폭 3cm, 길이 10cm, 선 폭 1cm | 시작/끝 표시 |
| 3 | 직선 자르기 (긴) | 폭 2cm 선, A4 가로 길이 | 선만 |
| 4 | 곡선 자르기 (완만) | 넓은 곡선, 선 폭 1cm | 선만 |
| 5 | 곡선 자르기 (날카로운) | S자, 물결, 선 폭 0.5cm | 없음 |
| 6 | 도형 오리기 (원, 사각) | 큰 도형 (직경 10cm+) | 굵은 외곽선 |
| 7 | 복합 도형 오리기 | 작은 도형, 복잡한 모양 | 외곽선만 |

### 8-3. 가위질 양손 협응 심화 프로토콜 (Bilateral Coordination in Scissor Use)

> **핵심 문제**: 가위질은 "자르는 손(우세손)"과 "종이를 잡고 돌리는 손(보조손)"의 역할 분담이 필수. 보조손 기능 부족 시 자르는 손이 아무리 좋아도 가위질 실패.

```
[양손 협응 역할 분석]

우세손 (Dominant hand): 가위 조작
  - 가위 열기/닫기 반복 (리드미컬 움직임)
  - 선을 따라 방향 조절
  - 필요 기술: 손가락 분리 운동, 힘 조절

보조손 (Non-dominant/Assisting hand): 종이 조작
  - 종이를 잡고 고정하기 (stability)
  - 곡선/원 자르기 시 종이를 돌리기 (rotation)
  - 자르는 방향에 맞춰 종이 위치 조절 (feed)
  - 필요 기술: 잡기 힘, 손목 회전, 양손 독립 움직임
```

#### 보조손 기능 부족 감별 및 훈련

```
[보조손 문제 유형]

유형 1: 종이를 잡지 못함 (잡기 힘 부족)
  - 증상: 종이가 흔들리며 찢어짐, 한 손으로만 가위질 시도
  - 훈련:
    a) 종이 대신 클립보드/테이프 고정 → 보조손 역할 제거 (일시적)
    b) 종이 찢기 양손 활동 → 잡는 힘 강화
    c) 점토 양손으로 당기기/비틀기 → 양손 협응 기초
    d) 빨래집게 놀이 → 잡기 힘 강화

유형 2: 종이를 돌리지 못함 (회전 불가)
  - 증상: 직선은 자르지만 곡선에서 실패. 가위를 돌리려 함
  - 훈련:
    a) "종이를 돌리는 거야, 가위를 돌리는 게 아니야" → 명시적 교육
    b) 종이에 회전 화살표 표시 → 시각 단서
    c) 큰 종이를 양손으로 돌리는 연습 (가위 없이)
    d) 뚜껑 돌려 열기/닫기 → 손목 회전 연습

유형 3: 양손 타이밍 불일치
  - 증상: 자르는 동안 종이를 계속 잡고 있지 못함 (잡았다 놓았다)
  - 훈련:
    a) 두꺼운 종이/골판지 사용 → 쉽게 잡히고 안정적
    b) 느린 속도로 가위질 (리듬 맞추기)
    c) "잡고 있어, 짝짝짝" → 구어 리듬 단서

[보조손 훈련이 먼저인 경우]
"가위질 연습인데 종이를 잡는 손이 안 됨" → 보조손 훈련이 우선
→ 가위질 전에 양손 협응 기초 활동 삽입:
  1. 종이 찢기 (양손)
  2. 점토 비틀기 (양손)
  3. 클립보드에 종이 고정 후 가위질 → 보조손 부담 제거
  4. 점진적으로 고정 도구 제거
```

#### 가위질 발달 단계 확장: 도형별 진행

```
[도형 오리기 세부 진행]

직선 자르기 → 곡선 자르기 → 원 자르기 → 복잡 도형

1. 직선 (3-3.5세)
   - 짧은 직선 (한 번 자르기, snip) → 긴 직선 (연속 자르기)
   - 선 폭: 1cm → 0.5cm → 선 위

2. 곡선 (3.5-4세)
   - 완만한 곡선 → S자 → 물결
   - 종이 돌리기 시작 (보조손 역할 증가)

3. 원 (4-4.5세)
   - 큰 원(직경 10cm+) → 중간 원(7cm) → 작은 원(5cm)
   - 종이를 계속 돌려야 함 → 보조손 협응 핵심
   - 시작점과 끝점이 만나야 함 → 계획 능력

4. 사각형 (4.5-5세)
   - 직선+방향전환(4회)
   - 꼭짓점에서 멈추고 방향 바꾸기

5. 삼각형 (5-5.5세)
   - 대각선 자르기 + 방향전환(3회)
   - 각도 인지 필요

6. 복합 도형/캐릭터 (5.5-7세)
   - 직선+곡선+방향전환 혼합
   - 세밀한 조절, 계획 능력 필요
```

## 9. 디지털-종이 전이 가이드 (Digital-to-Paper Transfer Protocol)

> **핵심 문제**: 태블릿에서는 선 긋기/글자 쓰기가 되는데 종이에서는 안 되는 아동. 디지털과 아날로그의 감각운동 차이를 이해하고 중간 단계를 설계해야 한다.

### 9-1. 디지털 vs 종이: 왜 차이가 나는가

```
[매체별 감각운동 차이]

특성          | 태블릿              | 종이+연필
─────────────|──────────────────|──────────────────
표면 마찰      | 낮음 (미끄러움)     | 높음 (저항 있음)
필압 필요량    | 거의 없음            | 적절한 힘 필요
시각 피드백    | 즉각적, 밝고 선명    | 연필 굵기/색에 의존
수정 용이성    | 되돌리기(undo) 가능  | 지우개 사용 (불완전)
도구 잡기      | 손가락/스타일러스    | 연필 3점 잡기 필요
표면 크기      | 고정 (화면 크기)    | 다양 (A3-A4)
자세          | 테이블/무릎          | 테이블 고정 필요
감각 피드백    | 진동/소리 가능       | 종이 질감, 연필 소리

→ 태블릿이 "더 쉬운" 이유: 낮은 필압, 즉각적 피드백, 수정 가능
→ 종이에서 "안 되는" 이유: 높은 필압 요구, 잡기 정밀도, 수정 불가 불안
```

### 9-2. 중간 단계 전이 활동 (Transition Hierarchy)

| 단계 | 매체 | 특성 | 활동 |
|------|------|------|------|
| 1 | 태블릿 (현재 성공 수준) | 기존 성공 확인, 자신감 유지 | 태블릿에서 성공하는 과제 수행 |
| 2 | 큰 화이트보드 (서서) | 큰 움직임, 낮은 정밀도 요구, 지우기 쉬움 | 서서 큰 글자/선 그리기 (팔 전체 사용) |
| 3 | 작은 화이트보드 (앉아서) | 중간 크기, 지우기 쉬움, 앉은 자세 | 앉아서 A3 크기 보드에 그리기 |
| 4 | 큰 종이 (A3) + 굵은 마커 | 종이이지만 큰 면적, 적은 힘으로 선 보임 | A3에 마커로 자유 그리기 → 선 따라 긋기 |
| 5 | A4 종이 + 굵은 마커 | 표준 크기로 전환, 마커는 유지 | A4에 마커로 활동 |
| 6 | A4 종이 + 크레용 | 마커→크레용 전환 (약간 더 많은 힘 필요) | A4에 크레용으로 활동 |
| 7 | A4 종이 + 연필 | 최종 목표 매체 | 표준 학습지 활동 |

### 9-3. 전이 프로토콜 핵심 원칙

```
[원칙]
1. 한 번에 하나의 변수만 변경
   - 매체 변경(화이트보드→종이) 시 도구는 유지(마커→마커)
   - 도구 변경(마커→크레용) 시 매체는 유지(A4→A4)

2. 각 단계에서 80% 이상 성공 → 다음 단계
   - 실패 시 이전 단계 유지, 강요하지 않음

3. 화이트보드 단계의 특별한 가치
   - "지울 수 있다" = 실패 불안 감소 (디지털의 undo와 유사)
   - 서서 하면 어깨/팔 큰 근육부터 사용 → 자연스러운 발달 순서
   - 종이로 직접 전환 시 좌절감이 크면 → 화이트보드 단계 삽입

4. 종이에서 실패하면
   - 즉시 성공했던 매체로 돌아가기 (자존감 보호)
   - "종이에서도 할 수 있어" → 작은 성공부터 (짧은 선 1개)
   - 종이 위에 태블릿과 같은 색상/배경 사용 (친숙화)

[디지털 습관 전환 팁]
- 태블릿에서 스타일러스 사용 → 연필 잡기와 유사한 경험 제공
- 태블릿 앱에서 "필압 반영" 기능 켜기 → 종이의 필압 경험과 연결
- 태블릿과 종이를 번갈아 사용 (같은 과제) → 매체 간 전이 촉진
```

## 10. 지적장애 특화 소근육 조절 (Fine Motor Adaptations for Intellectual Disability)

> ID(Intellectual Disability) 아동은 소근육 발달 자체가 지연될 뿐 아니라, 인지적 제약(획순 기억, 모방 속도, 일반화 어려움)이 소근육 과제 수행에 추가 장벽을 만든다.

### 10-1. ID 경도(Mild) vs 중등도(Moderate) 소근육 특성

```
[ID 경도 (IQ 50-69)]
- 소근육 발달: 정상 아동 대비 1-3년 지연, 그러나 대부분 기본 쓰기 도달 가능
- 학습 특성:
  · 반복 횟수 증가 필요 (일반 아동의 2-3배)
  · 시범(modeling) 후 모방 가능하나 속도 느림
  · 획순 기억: 2-3회 반복으로 단순 자모(ㄴ, ㄱ) 가능
  · 일반화: 큰 글자에서 배운 것을 작은 글자로 전이하는 데 추가 연습 필요
- 활동지 조절:
  · 같은 자모 반복 칸: 10-15칸 (일반 6-10칸)
  · 점선 보조 유지 기간 길게 (일반보다 2-3배)
  · 한 세션에 1개 자모만 (새로운 자모 도입 주기 길게)

[ID 중등도 (IQ 35-49)]
- 소근육 발달: 정상 대비 3-5년 지연, 기본 자모 쓰기 도달에 장기 훈련 필요
- 학습 특성:
  · 반복 횟수: 일반 아동의 5-10배 필요
  · 시범 모방: 즉각 모방 어려움, 손잡이(hand-over-hand) 보조 필요할 수 있음
  · 획순 기억: 단순 자모도 10회+ 반복 필요, 세션 간 망각 가능
  · 일반화: 매우 어려움 → 같은 활동을 동일 조건으로 반복이 효과적
- 활동지 조절:
  · 같은 자모 반복 칸: 15-20칸
  · 점선 보조 매우 오래 유지 (제거 시점을 늦춤)
  · 글자 크기: 7cm+ (일반 5cm 대비 크게)
  · 과제 단순화: 한 페이지에 한 가지 과제만
  · 시각 단서 최대화: 시작점 색상, 획순 번호 크게, 화살표 굵게
```

### 10-2. ID 아동 획순 기억 전략

```
[획순 기억 강화 방법]

1. 다감각 접근 (Multisensory Approach)
   - 시각: 획순 번호 + 색상 코딩 (1획=빨강, 2획=파랑...)
   - 촉각: 사포 위에 자모 글자 → 손가락으로 따라가기
   - 구어: "아래로, 옆으로" 말하면서 쓰기 (구어 단서)
   - 운동: 공중에 큰 글자 쓰기 (gross motor → fine motor)

2. 체계적 촉진-소거 (Systematic Prompting & Fading)
   Level 1: 손잡이(hand-over-hand) → 치료사가 아동 손을 잡고 함께
   Level 2: 손등 가이드 → 아동 손등에 가볍게 손 대고 방향 안내
   Level 3: 시작점 터치 → "여기서 시작" 손가락으로 시작점 가리킴
   Level 4: 구어 단서만 → "아래로 가볼까?"
   Level 5: 독립 수행 → 보조 없이

3. 반복 구조화
   - ID 경도: 한 자모당 3세션 × 각 세션 10-15회 반복 → 다음 자모
   - ID 중등도: 한 자모당 5-7세션 × 각 세션 15-20회 반복
   - 세션 시작 시 이전 자모 복습(3-5회) 후 새 자모 도입
   - 주 단위로 누적 복습 (이전 주 배운 자모 모두 복습)

4. 성공 기준 차별화
   - ID 경도: 점선 없이 독립 쓰기 3회 연속 성공 → 다음 자모
   - ID 중등도: 시작점 보조로 독립 쓰기 3회 연속 성공 → 다음 자모
```

### 10-3. ID 아동 모방 학습 최적화

```
[모방 학습 단계별 전략]

ID 경도:
  1. 치료사 시범 → 즉시 모방 (1:1 비율)
  2. 시범 + 구어 설명 병행 ("이렇게 아래로")
  3. 점진적으로 시범 빈도 줄이기 (3:1 → 5:1 → 독립)

ID 중등도:
  1. 치료사 시범 → 손잡이 보조 모방 (함께 하기)
  2. 시범 → 시작점 보조 모방 (혼자 시도)
  3. 비디오 모델링: 또래 아동이 쓰는 영상 반복 시청
  4. 역할 전환: 아동이 "선생님"이 되어 인형에게 가르치기
  5. 매우 느린 속도 시범 → 각 획마다 멈추고 기다리기

[모방 실패 시]
→ 과제가 너무 복잡할 가능성 → 과제 분석(task analysis)
  예: ㄱ 쓰기 = (1) 시작점에 연필 놓기 → (2) 오른쪽으로 → (3) 멈추기 → (4) 아래로
  → 각 단계를 따로 연습한 후 연결
```

## 11. 치료사 의사결정 프로세스 (5-Step Therapist Process)

### Step 1: 소근육 수준 파악 (Assessment)

```
[필수 질문]
1. "연필 잡기 패턴은?"
   → 못 잡음 / 손바닥 잡기 / 디지털 / 미성숙 3점 / 성숙 3점

2. "그리기 수준은?"
   → 못 그림 / 끄적임 / 직선 / 원 / 십자 / 삼각형 / 다이아몬드

3. "가위질 가능?"
   → 불가 / 한 번 자르기 / 직선 / 곡선 / 도형

4. "감각 민감성은?"
   → 시각과민 여부 / 촉각과민 여부 / 특정 색상 거부

5. "연령 대비 발달 수준은?"
   → 이정표 대비 몇 단계 지연인지 확인

6. "지적장애 동반 여부?"
   → 없음 / ID 경도 / ID 중등도 → 섹션 10 적용
```

### Step 2: 목표 설정 (Goal Setting)

```
[목표 유형별 결정]

A. 잡기 발달이 목표:
   → 잡기 이전 활동 (섹션 1-2) 또는 잡기 연습 활동

B. 선 따라 긋기가 목표:
   → 레벨 선택 (L1-L6), 감각과민 시 조절 (섹션 4)

C. 한글 쓰기 준비가 목표:
   → 쓰기 준비도 체크리스트 (섹션 6-4) 확인 후 자모 선택
   → 자모 요청 시 자동 분기 프로토콜 (섹션 6-5) 적용

D. 시각-운동 협응이 목표:
   → 점 잇기 / 미로 / 모양 복사 / 대칭 그리기

E. 가위질이 목표:
   → 가위질 단계 (섹션 8) 확인 후 단계 선택
   → 양손 협응 문제 시 섹션 8-3 적용

F. 복합 활동 (motor + cognition):
   → 각 영역 단독 수행 가능 확인 후 복합 설계 (섹션 7)

G. 디지털 → 종이 전이:
   → 섹션 9 전이 프로토콜 적용
```

### Step 3: 활동 선택 (Activity Selection)

```
[매핑 테이블]

현재 수준              →  활동 선택
─────────────────────────────────────────
잡기 불가              →  잡기 이전 활동 (점토, 찢기, 스티커)
손바닥 잡기            →  큰 크레용 끄적이기, L1 (넓은 선)
디지털 잡기            →  L1-L2, 큰 원 따라 그리기
미성숙 3점 잡기        →  L2-L4, 점 잇기 (입문-초급)
성숙 3점 잡기          →  L4-L8, 미로, 자모 따라쓰기

감각과민 있음          →  섹션 4 적용 (색상/대비/폭 조절)
도구 거부 있음         →  섹션 4-4 촉각과민 도구 거부 프로토콜
인지 복합 요청         →  섹션 7 복합 활동 설계 적용
대칭/패턴 요청         →  섹션 5-4 대칭 그리기 적용
ID 동반               →  섹션 10 적용 (반복 횟수/보조 수준 조절)
디지털→종이 전이       →  섹션 9 적용
```

### Step 4: 자료 제작 세부사항 (Material Specifications)

```
[체크리스트]

□ 선 굵기: 레벨에 맞는 폭 (L1=2cm, L2=1cm, L3=0.5cm...)
□ 시작점/끝점: 마커 유형 결정 (별, 점, 화살표)
  - 레벨 낮을수록 크고 눈에 띄는 마커
  - 레벨 높을수록 작은 점 또는 없음
□ 가이드라인: 점선 가이드 유무
  - 레벨 낮으면 점선 가이드 제공
  - 레벨 높으면 제거
□ 색상/대비: 감각과민 여부에 따라 결정
  - 일반: 검정 선, 흰 배경
  - 감각과민: 회색 선, 크림 배경 (섹션 4 참조)
□ 페이지 구성:
  - 한 페이지 활동량 = 아동 집중시간 기준
  - 3-4세: 3-5개 활동/페이지
  - 5-7세: 5-8개 활동/페이지
  - ID 동반: 과제 수 절반으로 줄이기
□ 시각 지원:
  - 활동 이해를 돕는 간단한 아이콘
  - 완성 예시 (작게)
□ 인쇄 사양:
  - A4 용지, 세로 방향 기본
  - 여백: 상하좌우 최소 1.5cm (감각과민 시 3cm)
```

### Step 5: 예상 문제 및 대응 (Anticipate & Adapt)

```
[상황별 대응 프로토콜]

"손이 금방 피곤해한다"
  → 활동량 줄이기 (절반)
  → 선 길이 줄이기 (긴 선 → 짧은 선)
  → 중간 휴식 표시 (★ 여기서 쉬세요!)

"너무 쉬워한다"
  → 선 폭 줄이기 (한 단계 좁게)
  → 가이드 제거
  → 레벨 올리기
  → 복합 활동으로 전환 (섹션 7)

"좌절감을 보인다"
  → 즉시 레벨 낮추기
  → 성공 가능한 활동 먼저 제공 (이미 할 수 있는 수준)
  → 시작점-끝점 간격 줄이기 (짧은 선)
  → 강화물(보상) 요소 추가 (완성 시 스티커 공간)

"특정 방향만 어려워한다 (예: 좌→우는 되는데 대각선 안 됨)"
  → 해당 방향만 집중 연습 자료
  → 화살표 가이드 강화
  → 점선 따라 긋기 → 실선으로 점진적 전환

"연필을 너무 세게/약하게 잡는다"
  → 세게: 부드러운 필기구 (4B 연필, 굵은 크레용)
  → 약하게: 마커/사인펜 (적은 힘으로 선 보임)
  → 잡기 보조 도구 (그립 어댑터) 권장

"양손 협응이 안 된다 (가위질 등)"
  → 한 손 활동부터 (도장 찍기, 스탬프)
  → 종이 고정 도구 사용 (테이프, 클립보드)
  → 가위 전에 찢기 활동 선행
  → 보조손 훈련 프로토콜 (섹션 8-3) 참조

"도구를 만지려 하지 않는다"
  → 촉각과민 도구 거부 프로토콜 (섹션 4-4) 적용
  → 도구 없는 활동부터 시작 (모래 글씨, 핑거페인팅)

"태블릿에서는 되는데 종이에서 안 된다"
  → 디지털→종이 전이 가이드 (섹션 9) 적용
  → 화이트보드 중간 단계 삽입
```

```yaml
구성요소:

  - 획순 안내: 번호 + 화살표 (큰 글자로 시범)

  - 첫 2-3칸은 점선 (최대 보조)

  - 마지막 2-3칸은 빈 칸

  - ID 중등도: 반복 칸 15-20칸, 글자 크기 7cm+, 점선 보조 대부분 유지
```

## 13. 핵심 판단 규칙 (Clinical Decision Rules)

### 절대 규칙 (MUST)

1. **잡기 불가 아동에게 선 긋기 활동을 제공하지 않는다.** 반드시 잡기 이전 활동부터.
2. **쓰기 전 기술 위계를 건너뛰지 않는다.** 수직선 → 수평선 → 원 → 십자 순서 준수.
3. **감각과민 아동에게 고대비/빨간색 자료를 제공하지 않는다.**
4. **한글 자모 쓰기는 쓰기 준비도 체크리스트 충족 후에만 도입한다.**
5. **복합 활동은 각 단일 활동 수행 가능 시에만 적용한다.**
6. **아동 연령이 아닌 발달 수준에 맞춰 자료를 선택한다.**
7. **0-2세(발달연령) 아동에게 학습지를 제공하지 않는다.** 실물 감각활동만 권장.
8. **자모 요청 시 선행 자모/기술을 반드시 확인한다.** 미충족 시 대안 제시.
9. **도구 거부 아동에게 도구 사용을 강요하지 않는다.** 대체 매체부터 시작.
10. **ID 동반 아동은 반복 횟수와 보조 수준을 차별화한다.**

### 권장 규칙 (SHOULD)

1. 한 학습지에 성공 경험을 보장하는 쉬운 과제를 1-2개 포함한다.
2. 레벨 상향은 현재 레벨 80% 이상 성공 시 시도한다.
3. 가위질 활동은 안전 가위(둥근 끝) 사용을 전제로 설계한다.
4. 좌절 신호 시 즉시 난이도를 낮출 수 있는 대체 활동을 준비한다.
5. 학습지 상단에 치료사/보호자용 간단 안내 문구를 포함한다.
6. 디지털→종이 전이 시 한 번에 하나의 변수만 변경한다.
7. 가위질에서 보조손 문제가 먼저인 경우, 보조손 훈련을 선행한다.

## 14. 스트레스 테스트 (Stress Test Rounds)

### R1: "2세 반, 크레용으로 끄적이기만 해요. 이게 정상인가요? 다음 단계는?"

```
[분석]
- 생활연령: 2.5세 (30개월)
- 현재 수준: 끄적이기 (scribbling)
- 발달 이정표 대조 (섹션 1):
  · 1-2세: 끄적임 = 정상
  · 2-3세: 수직선, 원형 끄적임 출현
  · 2.5세 전형 기대: 수직선 가능, 원형 끄적임 시작

[판단]
→ 2.5세에서 끄적이기 = 정상 범위의 하한
→ 끄적이기 발달 단계(섹션 1A-4) 확인:
  · 2단계(조절된 끄적이기, 16-20개월) 완료?
  · 3단계(이름 붙이는 끄적이기, 20-30개월) 진입?
→ "끄적이기만 한다"가 무질서(1단계)인지 조절된(2단계)인지가 핵심

[대응]
정상 범위인 경우 (조절된 끄적이기 이상):
  - 안심: 30개월까지 끄적이기는 정상 범위 내
  - 다음 목표: 방향성 있는 끄적이기 → 수직선 출현 유도
  - 활동: 큰 종이(A3+)에 뭉툭한 크레용으로 자유 끄적이기 지속
         "위에서 아래로" 시범 보여주기 (수직선 유도)
         비/물 떨어지는 그림으로 수직선 동기 부여

지연 가능성 (무질서 끄적이기에 머물러 있는 경우):
  - 30개월인데 아직 1단계(무질서) → 발달 평가 권장
  - 잡기 패턴 확인: 손바닥 잡기도 불안정하면 잡기 이전 활동(섹션 1-2) 선행
  - 학습지 적합성: ⚠ 아직 학습지 부적절 → 실물 감각활동(섹션 1A-2) 우선
```

### R2: "5세 ASD L2, 원은 그리는데 십자(+)를 못 그려요. 선이 만나질 않음"

```
[분석]
- 생활연령: 5세
- 진단: ASD Level 2
- 현재 수준: 원(4단계) 가능, 십자(5단계) 실패
- 실패 양상: "선이 만나질 않음" = 두 선의 교차점 조준 실패
- 전형 연령 대조: 십자 = 3.5-4세 → 약 1-1.5년 지연

[판단]
→ 쓰기 전 기술 위계 4→5단계 전환점의 어려움
→ 십자는 "교차(intersection)"라는 새로운 시각-운동 요소 필요
→ 섹션 2-1 십자 실패 분석 참조: "유형 1: 교차 실패"
→ ASD L2: 시각-운동 협응의 하위 기술 분해가 필요

[대응]
→ 십자 실패 유형 1 중재 프로토콜 적용:
  1. 수직선 위에 큰 점(●) 표시 → "이 점 위를 지나가게 옆으로"
  2. 교차점에 별(★) 표시 → 시각 타겟
  3. 마커 크기 점차 축소
  4. 마커 없이 시도
→ ASD 특화 조절:
  - 시각적 구조화 강화 (교차점 색상 강조)
  - 반복 연습 구조화 (같은 활동 5-8회 반복)
  - 구어 단서: "가운데서 만나!" 간단한 규칙 제공
→ 학습지: L5(꺾은선) 수준 + 교차점 연습 특화 활동지
→ 한글 쓰기: 아직 시기상조 (십자=ㄱ,ㄴ 이전의 교차 기술 미달)
```

### R3: "7세 지적장애 중등도, 가위질 연습인데 종이를 잡는 손이 안 됨"

```
[분석]
- 생활연령: 7세
- 진단: ID 중등도 (IQ 35-49)
- 과제: 가위질 연습
- 문제: 보조손(non-dominant hand)으로 종이 잡기 불가
- 발달수준 추정: 소근육 3-5년 지연 → 실제 기능 약 2-4세 수준

[판단]
→ 핵심 문제: 가위질의 양손 협응 중 보조손 기능 부족 (섹션 8-3)
→ "보조손 문제 유형 1: 종이를 잡지 못함"에 해당
→ 보조손 훈련이 가위질보다 먼저

[대응]
Step 1: 보조손 훈련 먼저 (가위 없이)
  - 종이 찢기 (양손): 한 손은 잡고, 한 손은 당기기
  - 점토 비틀기 (양손): 양손 동시 사용
  - 빨래집게 놀이: 잡는 힘 강화
  - ID 중등도 조절: 반복 횟수 5-10배, 손잡이 보조 시작 가능

Step 2: 가위질 보조 도구 도입
  - 클립보드에 종이 고정 → 보조손 역할 제거 (일시적)
  - 스프링 가위 사용 → 열기 힘 보조
  - 두꺼운 종이/골판지 → 잡기 쉬움

Step 3: 점진적 보조 제거
  - 클립보드 → 테이프 고정 → 한 손으로 잡기 시도
  - ID 중등도이므로 각 단계 장기간 유지 (세션 5-7회+)

→ 학습지: 가위질 활동지보다 양손 협응 활동지를 먼저 제공
→ 가위질 시작 시: 단계 0(찢기) → 단계 1(한 번 자르기, 클립보드 고정)
→ 섹션 10(ID 특화) + 섹션 8-3(보조손 프로토콜) 동시 적용
```

### R4: "4세, 점잇기(dot-to-dot) 3개는 하는데 5개부터 순서를 잃어버려요"

```
[분석]
- 생활연령: 4세
- 과제: 점잇기 (dot-to-dot)
- 성공: 3개 점 연결 가능
- 실패: 5개부터 순서 상실
- 발달 대조: 4세 점잇기 = 초급(5-10개) 수준 기대

[판단]
→ 섹션 5-1A "점잇기 실패 분석" 적용
→ 3개 → 5개에서 실패하는 이유 분석:
  · 숫자 인지: 1-5까지 순서를 아는가? → 확인 필수 (인지 영역)
  · 시각 스캐닝: 3개는 한눈에 보이지만 5개는 "다음 숫자를 찾아야" 함
  · 작업 기억: "지금 몇 번까지 했는지" 기억 유지 부하 증가
→ 이것은 소근육 문제가 아닐 가능성이 높음

[대응]
1. 먼저 감별:
   - "5까지 세어볼까?" → 숫자 세기 가능 여부
   - 점은 못 찾지만 선은 잘 긋는다 → 인지/시각 문제
   - 점을 찾고 선도 못 긋는다 → 복합 문제

2. 숫자 인지 문제인 경우:
   → 인지 영역 과제 선행 (숫자 순서 학습)
   → 점잇기 대안: 색상 순서로 연결 (빨→주→노→초→파), 숫자 제거
   → 소근육 초점 유지: 3개 점 다양한 모양으로 (직선, 꺾은선, 곡선)

3. 시각 스캐닝 문제인 경우:
   → 다음 점에 색상 힌트 (현재 점과 같은 색)
   → 점 사이 거리 줄이기 (5cm+ → 3cm)
   → 연결할 다음 점을 형광펜으로 크게 표시

4. 작업 기억 문제인 경우:
   → 완료한 점에 스티커 붙이기 (시각 마커)
   → 연결선 색 유지하며 "여기까지 했어" 확인
   → 4개로 먼저 시도 (3→4→5 점진적 증가)

→ 학습지: 점잇기 3개 유지 + 다양한 모양 변형 (소근육 유지)
           인지 과제는 별도 활동으로 분리 (복합 부하 방지)
```

### R5: "6세, 미로 활동에서 벽을 무시하고 직선으로 가요"

```
[분석]
- 생활연령: 6세
- 과제: 미로
- 문제: 벽을 무시하고 직선으로 통과
- 6세 미로 기대 수준: 중급-상급 (통로 1-1.5cm, 분기점 3-5개)

[판단]
→ 섹션 5-2A "벽 무시 행동 분석" 적용
→ "직선으로 가요" = 억제(inhibition) 기능 문제 가능성 높음
→ 6세에서 이 수준의 벽 무시는 비정상적
→ 감별 필요: 억제 기능 결함 vs 규칙 미이해 vs 시각 변별 vs 동기 부족

[대응]
1. 감별 확인:
   - "여기 벽이야, 지나가면 안 돼" 설명 후 재시도
     · 알지만 반복 → 억제 기능 문제 (ADHD 가능성 고려)
     · 모른다고 함 → 규칙 이해 문제 (3-4세 수준에서 가능)
   - 통로를 연한 색으로 칠해주면 나아지는지?
     · 나아짐 → 시각 변별 문제
     · 변화 없음 → 억제 문제
   - 좋아하는 캐릭터 미로로 변경
     · 나아짐 → 동기 문제
     · 변화 없음 → 억제/시각 문제

2. 억제 기능 문제인 경우:
   → 미로 전에 억제 훈련 활동:
     - "빨간불 초록불" 게임 (멈추기 연습)
     - 물리적 미로(3D)에서 "벽에 부딪히면 쿵!" 경험
     - 손가락 미로(연필 없이) → "벽 터치하면 다시 시작"
   → 미로 자료 조절:
     - 벽을 굵은 빨간선으로 강조
     - 통로 폭 3cm+로 넓히기
     - 분기점 1-2개만 (단순 미로부터)
   → ADHD 선별 고려:
     - 미로 벽 무시 + 다른 영역 충동성 → 선별 권장
     - 6세이면 ADHD 선별/진단 가능 연령
     - 그러나 미로 벽 무시만으로는 진단 불가 → 종합 관찰 필요

3. 학습지:
   → 미로 난이도 대폭 낮추기: 입문 (통로 3cm+, 분기점 1-2개, 막다른 길 없음)
   → 벽 강조 미로 (빨간 굵은 벽, 통로 색칠)
   → "손가락 미로" 선행 → 연필 미로로 전이
   → 6세이므로 미로 자체보다 억제 기능 발달이 치료 목표일 수 있음
```

## 추가 섹션: 자세 안정성 전제조건 (근위→원위 원칙)

> 소근육은 대근육 위에 세워진다. 체간→어깨→팔꿈치→손목→손가락.

### 근위-원위 발달 사슬

```
Level 0: 체간 안정성 — 바르게 앉기 10초+
Level 1: 어깨 안정성 — 팔 공중 유지 과제
Level 2: 팔꿈치 조절 — 전완 회전
Level 3: 손목 안정성 — 약간 뒤로 젖힌 상태 유지
Level 4: 손 아치 — 손바닥 컵 모양
Level 5: 손가락 분리 — 조작(엄지+검지+중지) vs 안정화(약지+소지)
```

### 자세 간이 체크 (소근육 활동 전)
```
□ 양발 바닥 닿음?  □ 엉덩이 뒤쪽?  □ 등받이 없이 10초?
□ 양팔 책상 위 5초?  □ 고개 과도 기울기 없음?
→ 3개 미만: 자세 지원 먼저 (발판, 쿠션, 경사면)
```

## 추가 섹션: VMI 구성요소 분석

### VMI = 시각 인식(VP) + 운동 조절(MC) + 통합

| 결함 유형 | 우선 활동 | 피해야 할 활동 |
|----------|---------|-------------|
| VP 결함 | 선 따라 긋기, 도형 변별 | 빈 공간 모양 복사 |
| MC 결함 | 넓은 선 긋기, 점토 | 정밀 복사, 작은 글자 |
| 통합 결함 | 점잇기, 격자 복사 | 시범 없는 자유 그리기 |

### 감별 의사결정
```
VP 보충검사 → VP 낮으면 시각 인지 훈련 선행
VP OK → MC 보충검사 → MC 낮으면 잡기/힘 조절 훈련
VP OK + MC OK → 통합 결함 → 시각-운동 연결 훈련
```

## 추가 섹션: 유지(Maintenance) & 일반화(Generalization)

### "치료실에서 되는데 집에서 안 돼요" 분석

```
치료실: 최적 좌석/도구, 1:1 집중, 즉각 피드백
가정: 다양한 자세, 아무 연필, 형제/TV/소음, 피드백 없음
→ 같은 기술이라도 환경 변수 달라지면 수행 저하는 당연
→ "집에서 안 됨" = 실패가 아님. 일반화가 아직 안 된 것.
```

### 일반화 촉진 순서
```
1. 치료실에서 숙달 (보조 없이 80%+)
2. 같은 활동, 다른 도구 (치료실에서)
3. 같은 활동, 가정 환경 (부모와 함께)
4. 같은 활동, 학교 환경 (교사 연계)
5. 유사한 활동, 다양한 환경 (과제 일반화)
→ 한 번에 하나의 변수만 변경
```

### 가정 연계 부모 체크리스트
```
□ 발 바닥 닿는 의자, 팔꿈치 90도 책상
□ 치료사 지정 도구 사용, 그립 보조 도구 포함
□ 하루 5-10분 (짧고 자주 > 길고 가끔)
□ 즐거운 분위기 (강요 = 역효과)
□ 수행 사진/영상 → 치료사 공유
```

## 추가: 필기 유창성 (속도/지구력/가독성)

> 자모 쓰기 가능 아동(L7+)의 필기 속도, 지구력, 가독성 문제.

### 필기 속도 저하 원인

| 원인 | 확인 | 중재 |
|------|------|------|
| 운동 자동화 미완 | 쉬운 자모는 빠르고 어려운 자모에서 멈춤 | 빈출 자모 반복, 타이밍 게임 |
| 근지구력 부족 | 3줄 이후 글씨 무너짐 | 손 내재근 강화, 시간 점진 증가 |
| 과도한 필압 | 종이 뒷면 자국 깊음, 연필심 자주 부러짐 | 0.5mm 기계식 연필(부러지면 피드백) |
| 완벽주의(ASD) | 지우고 다시 쓰기 반복 | "빠른 글씨 OK" 규칙, 지우개 없이 쓰기 |

### 학업 보상: 받아쓰기 대체(빈칸채우기), 타이핑 병행(8세+), 시간 연장

## 추가: 왼손잡이 지원 + 거울 글씨

### 왼손잡이 필기 조절

| 항목 | 오른손 | 왼손 조절 |
|------|--------|---------|
| 종이 각도 | 약간 왼쪽 | 20-30도 오른쪽 |
| 조명 | 왼쪽 | 오른쪽 |
| 연필 잡기 | 표준 | 펜 끝에서 2.5cm 위 |

### 거울 글씨 감별
5-7세 거울 글씨 = 정상 범위 (왼손잡이에서 더 흔함)
7세+ 지속 = 시각-공간 처리 평가 필요
중재: 좌측 시작점 빨간 앵커, 방향 화살표, "ㄱ은 오른쪽으로 시작" 명시

## 추가: 색칠 활동 발달

### 색칠 레벨

| 레벨 | 특성 | 연령 |
|------|------|------|
| C1 | 경계 무시 전체 난화 | 2-3세 |
| C2 | 경계 인식 50%+ 이탈 | 3-4세 |
| C3 | 대부분 유지, 코너 이탈 | 4-5세 |
| C4 | 유지 + 채움 불균일 | 5-6세 |
| C5 | 경계+균일+다색 | 6-7세 |

### "선긋기 OK + 색칠 이탈" = 경계 접근 시 감속 미숙
중재: 경계선 직전 감속 구간(노란 띠) 표시, 한 방향 왕복 전략 교육

## 추가: 조작 하위 유형 분화

| 유형 | 대표 활동 | 핵심 기술 | 연령 |
|------|---------|---------|------|
| 쌓기 | 블록 | 놓기 조절 | 12개월~ |
| 넣기 | 도형맞추기 | 방향 맞추기 | 18개월~ |

→ 듀플로(큰 레고)부터, 한 손 고정 연습

## GUARDRAILS (안전 경계)

### 절대 금지
- 진단 내리지 않음
- 의료적 조언(약물, 예후) 하지 않음
- 치료 중단/종결 결정하지 않음
- 확정적 진술("반드시", "확실히") 금지 → "~으로 추정됩니다" 사용

### 전문가 검토 필요
- 퇴행 보고 → 소아신경과 의뢰
- 6개월+ 정체 → 다학제 팀 회의
- 동반 장애(CP, 청각장애, 시각장애) → 해당 전문가 연계

### 자기 평가 기준 (Top 1%)
- "이대로 사용" 클릭률 ≥ 90%
- 세션 정답률 60-80% (적정 난이도)
- 도메인 감지 정확도 ≥ 95%

### Auto-Research 트리거
- "쉽게" 클릭률 40%+ → 난이도 교정 재검토
- 미등록 키워드 반복 출현 → 참조파일 업데이트 제안
- 새로운 임상 패턴 5회+ → 엣지케이스 추가
