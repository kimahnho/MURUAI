## 1. 놀이 발달 6단계 (Play Development 6 Stages)

기능연령(functional age) 기준. 생활연령이 아닌 기능연령을 반드시 사용할 것.

| Stage | 명칭 | 기능연령 | 핵심 특징 | 대표 놀이 |
|-------|------|----------|-----------|-----------|
| 1 | 탐색/감각놀이 (Exploratory/Sensory) | 0-18개월 | 사물의 감각적 속성 탐색 (만지기, 입에 넣기, 두드리기, 던지기) | 딸랑이 흔들기, 물 만지기, 사물 입에 넣기 |
| 2 | 기능놀이 (Functional) | 18-24개월 | 사물의 기능에 맞게 사용 (차 굴리기, 숟가락으로 먹는 흉내) | 자동차 굴리기, 컵으로 마시는 척, 전화기 귀에 대기 |
| 3 | 구성놀이 (Constructive) | 2-3세 | 사물을 조합하여 무언가를 만듦 (쌓기, 끼우기, 조립) | 블록 쌓기, 퍼즐 맞추기, 기차 레일 연결 |
| 4 | 상징놀이 (Symbolic/Pretend) | 3-5세 | 사물이나 상황을 다른 것으로 대체 (바나나=전화기, 빈 컵으로 마시는 척) | 인형에게 밥 먹이기, 바나나로 전화하기, 블록을 자동차라고 하기 |
| 5 | 사회극놀이 (Sociodramatic) | 4-6세 | 역할을 정하고 상대방과 함께 이야기를 만듦 | 병원놀이 (의사/환자 역할 배분), 가게놀이 (사장/손님), 소꿉놀이 (엄마/아기) |
| 6 | 규칙놀이 (Rule-based/Games) | 6-8세 | 정해진 규칙을 이해하고 따르며 진행 (차례 지키기, 승패 수용) | 보드게임, 윷놀이, 카드게임, 술래잡기, 가위바위보 |

### 1-1. Stage 1 하위 단계 세분화 (Sensory Play Sub-stages)

Stage 1은 ASD 아동에게 특히 중요한 단계로, 감각통합치료(OT-SI)와의 연계를 위해 3개 하위 단계로 세분화한다.

```
Stage 1A: 감각 탐색 (Sensory Exploration) | 기능연령 0-6개월
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  특징:
    - 사물에 대한 무분별 감각 탐색 (입에 넣기, 던지기, 두드리기)
    - 감각 자극 자체가 목적 (결과에 대한 기대 없음)
    - 한 감각 채널에 고착 또는 모든 채널 무분별 탐색
  대표 행동:
    - 모든 사물을 입에 넣기
    - 사물 던지고 소리 듣기
    - 손으로 다양한 질감 만지기 (목적 없이)
  감각통합 연계:
    - 감각 회피(sensory avoidance) 아동: 질감 경험 범위 점진적 확장
    - 감각 추구(sensory seeking) 아동: 안전한 감각 입력 경로 제공
    - OT-SI 관점: 감각 조절(sensory modulation) 상태 모니터링
  학습지 적합성: 부적합 (실물 감각 활동 우선)

Stage 1B: 원인-결과 감각놀이 (Cause-Effect Sensory) | 기능연령 6-12개월
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  특징:
    - "내가 하면 → 이런 일이 생긴다" 인식 시작
    - 의도적 반복이 나타남 (우연 → 의도적 행동)
    - 특정 감각 결과를 기대하며 행동 반복
  대표 행동:
    - 버튼을 눌러 소리/빛 발생시키기
    - 물을 쏟으면 흐르는 것 반복 관찰
    - 장난감을 바닥에 떨어뜨려 소리 듣기 (의도적)
  감각통합 연계:
    - 행위 기획(praxis) 기초 단계
    - 감각 피드백 루프 형성: 행동→감각결과→재시도
    - OT-SI 관점: 적응 반응(adaptive response) 출현 관찰
  학습지 적합성: 조건부 (1B 후반, 시각 주의 가능 시 극단순 학습지)
  학습지 형식 (가능 시):
    - 버튼 누르기→결과 매칭 (실물사진 2개)
    - 행동→결과 쌍 매칭 (두드리면→소리)

Stage 1C: 감각 선호 놀이 (Sensory Preference Play) | 기능연령 12-18개월
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  특징:
    - 선호하는 감각 채널/자극이 분명해짐
    - 특정 감각 경험을 선택적으로 반복 추구
    - 감각 선호가 이후 관심사 패턴의 기초가 됨
  대표 행동:
    - 물만 반복적으로 만지기/흘리기 (촉각+시각 선호)
    - 회전하는 것만 보기 (시각 선호)
    - 특정 소리만 반복 만들기 (청각 선호)
  감각통합 연계:
    - 감각 프로파일(sensory profile) 파악의 핵심 시기
    - 선호 감각 → 놀이 동기 → 기능놀이 전환 도구
    - OT-SI 관점: 감각 식이(sensory diet)에 선호 감각 통합
  학습지 적합성: 조건부 (선호 감각을 활용한 극단순 매칭)
  학습지 형식 (가능 시):
    - 선호 감각 관련 사물 매칭 (물 좋아하면 → 물 관련 사진 매칭)
    - 감각별 분류 (만지는 것/보는 것/듣는 것)

  ⚠ CRITICAL: Stage 1C의 감각 선호는 ASD 아동의 제한된 관심사(restricted interests)의
  전조일 수 있다. 선호를 제거하지 않고, Stage 2 기능놀이로의 전환 발판으로 활용해야 한다.
  (섹션 12 관심사 기반 확장과 연계)
```

### 1-1-1. Stage 1 하위 단계 판별 질문

```
Stage 1 내부 판별:
  Q1a. "감각 탐색 시 특정 결과를 기대하는가?" (예: 버튼 누르면 소리 나는 걸 알고 반복)
      → NO → Stage 1A (무분별 감각 탐색)
      → YES ↓
  Q1b. "선호하는 특정 감각 자극이 분명한가?" (예: 물만 찾음, 회전만 봄)
      → NO → Stage 1B (원인-결과 감각놀이)
      → YES → Stage 1C (감각 선호 놀이)
```

### 1-2. 단계 판별 질문 (Stage Assessment Questions)

```
Step 1: 현재 놀이 수준 파악
  Q1. "장난감을 기능에 맞게 사용하는가?" (예: 차를 굴림, 숟가락으로 먹는 척)
      → NO → Stage 1 (탐색/감각놀이) → 하위 단계 판별 (1-1-1)
      → YES ↓
  Q2. "장난감을 조합하여 무언가를 만드는가?" (예: 블록을 쌓아 탑을 만듦)
      → NO → Stage 2 (기능놀이)
      → YES ↓
  Q3. "사물이나 상황을 다른 것으로 대체하는가?" (예: 블록을 전화기라고 함)
      → NO → Stage 3 (구성놀이)
      → YES ↓
  Q4. "상대방과 역할을 나누어 함께 놀 수 있는가?" (예: 의사/환자 역할극)
      → NO → Stage 4 (상징놀이 - 혼자)
      → YES ↓
  Q5. "게임 규칙을 이해하고 차례를 지킬 수 있는가?"
      → NO → Stage 5 (사회극놀이)
      → YES → Stage 6 (규칙놀이)
```

### 1-3. 단계 전환 징후 (Transition Indicators)

| 현재 → 다음 | 전환 징후 | 촉진 전략 |
|-------------|----------|-----------|
| 1A→1B | 우연한 행동 후 동일 행동 반복 시도 (의도성 출현) | 원인-결과 장난감 제공 (버튼→소리/빛) |
| 1B→1C | 특정 감각 자극을 선택적으로 반복 추구 | 선호 감각 중심 활동 다양화 |
| 1C→2 | 선호 사물을 감각 탐색 외 방식으로 조작 시도 (차를 밀어봄) | 선호 사물의 기능 시범 반복 + 모방 유도 |
| 1→2 | 사물을 입에 넣는 대신 기능적 조작 시도 (차를 밀어봄) | 기능 시범 반복 + 모방 유도 |
| 2→3 | 블록 2-3개를 의도적으로 쌓거나 나란히 놓음 | 구성 결과물 명명 ("탑이다!") |
| 3→4 | 사물에 가상 속성 부여 시도 ("맘마 먹어" 하며 빈 컵 내밈) | 대체물 제공 + 언어화 |
| 4→5 | 상대에게 역할 지정 시도 ("너는 아기야") | 또래/성인과의 역할극 기회 |
| 5→6 | 간단한 규칙을 이해하고 지키려는 시도 | 규칙이 단순한 게임부터 시작 |

### 1-4. 각 단계의 인지 전제 조건 (Cognitive Prerequisites)

```
Stage 1A (탐색): 없음 - 감각 반응만 있으면 됨
Stage 1B (원인-결과): 원시적 인과 인식 (행동→결과 연결)
Stage 1C (감각 선호): 선호/비선호 구분 능력, 기억 기초 (이전 경험 재현)
Stage 2 (기능): 사물 기능 인식 (사물 영속성 기초)
Stage 3 (구성): 인과관계 기초 (쌓으면→높아짐), 공간 지각
Stage 4 (상징): 표상 능력 (눈앞에 없는 것을 떠올림), 탈맥락화
Stage 5 (사회극): 마음이론 기초 (타인에게 역할 부여), 언어 의사소통
Stage 6 (규칙): 추상 규칙 이해, 조망 수용 (상대 차례 인식), 승패 개념
```

## 2. ASD 놀이 특성 (ASD-Specific Play Characteristics)

### 2-1. ASD 아동의 전형적 놀이 프로파일

| 특성 | 설명 | 임상적 함의 |
|------|------|-------------|
| 감각/기능놀이 고착 | Stage 1-2에 머무르는 경우가 흔함. 연령과 무관하게 사물 탐색/반복 조작 | 생활연령이 아닌 기능적 놀이 수준에 맞춰 자료 설계 |
| 상징놀이 지연/결여 | Stage 4 진입이 현저히 늦거나 자발적 상징놀이 미출현 | 상징놀이를 직접 "가르치기" 보다 기능놀이에서 자연스러운 전환 유도 |
| 반복 패턴 (stereotyped play) | 같은 놀이를 동일한 방식으로 반복 (줄 세우기, 바퀴 돌리기, 문 열고 닫기) | 반복 자체를 제거 대상이 아닌 확장의 출발점으로 활용 |
| 관심사 제한 (restricted interests) | 특정 주제/사물에만 몰입 (자동차, 공룡, 숫자, 특정 캐릭터) | 관심사를 치료의 동기 부여 도구로 활용 (섹션 12 참조) |
| 또래 놀이 어려움 | 혼자 놀이는 가능하나 또래와의 상호작용 놀이에 참여 어려움 | play 영역과 social 영역의 교차 지점 (섹션 8 참조) |
| 놀이의 경직성 | 놀이 시나리오나 규칙 변경에 대한 저항 | 점진적 변형 전략 사용 (동일한 틀 유지하며 한 가지 요소만 변경) |

### 2-2. ASD 지원 수준별 놀이 특성

| 지원 수준 | 전형적 놀이 수준 | 주요 과제 | 학습지 적합성 |
|-----------|-----------------|-----------|---------------|
| Level 1 (지원 필요) | Stage 3-5 도달 가능, 사회극놀이에서 어려움 | 놀이의 유연성, 또래 상호작용, 규칙 이해 | 학습지 적합 (규칙 시각화, 놀이 스크립트) |
| Level 2 (상당한 지원) | Stage 2-3 주로, Stage 4 부분적 | 상징놀이 전환, 놀이 확장 | 학습지 적합 (기능 매칭, 구성 패턴, 간단한 상징) |
| Level 3 (매우 상당한 지원) | Stage 1-2 주로 | 기능놀이 확립, 감각 탐색 확장 | **학습지 적합성 재검토 필요** (실물 기반 활동 우선) |

### 2-3. ASD Level 3 학습지 적합성 판단

```
IF ASD Level 3 AND Stage 1 (감각탐색만):
    → 학습지(종이 기반) 자체가 적절한 매체인가?
    → 판단 기준:
      1. 종이를 찢거나 입에 넣지 않고 시각적으로 볼 수 있는가?
      2. 치료사의 시범에 주의를 기울일 수 있는가 (최소 3초)?
      3. 지시에 따라 사물 1개를 가리키거나 올려놓을 수 있는가?

    → 3개 중 2개 이상 YES:
      학습지 가능하나 극도로 단순화 필요
      - 한 페이지에 자극 2개 이하
      - 실물 사진 사용 (일러스트 X)
      - 매칭 활동 위주 (실물→사진 매칭)

    → 3개 중 2개 이상 NO:
      학습지 부적절 → 실물 기반 감각활동 권장
      "현재 놀이 수준이 탐색/감각 단계이므로, 종이 기반 학습지보다
       실물 장난감을 활용한 감각놀이 활동을 권장합니다."
      → 실물 활동 예시 제안 후 종료
```

## 3. 놀이 기반 학습지 활동 (Play-Based Worksheet Activities)

### 3-1. 활동 유형 총괄표

| 활동 유형 | 목표 놀이 단계 | 형식 | 예시 |
|-----------|---------------|------|------|
| 장난감-기능 매칭 | Stage 2 (기능놀이) | 선긋기/매칭 | 자동차↔도로, 숟가락↔밥그릇, 크레용↔종이 |
| 놀이 순서 배열 | Stage 2-3 | 순서 맞추기 (3-5단계) | 블록 꺼내기→쌓기→완성, 물감 짜기→섞기→그리기 |
| "다음에 뭘 할까?" | Stage 3-4 | 선택지 제공 | 소꿉놀이 중 "냄비에 넣은 다음에는?" → 젓기/불끄기/그릇에 담기 |
| 놀이 규칙 카드 | Stage 5-6 | 시각적 규칙표 | "1. 순서를 정해요 2. 주사위를 던져요 3. 말을 움직여요 4. 다음 사람 차례" |
| 진짜↔가짜 매칭 | Stage 4 (상징) | 같은것찾기/매칭 | 진짜 전화기↔바나나(전화기 역할), 진짜 자동차↔상자(자동차 역할) |
| 놀이 스크립트 | Stage 5 | 대화문 + 역할 카드 | 병원놀이: "어디가 아파요?" / "배가 아파요" / "약 드세요" |
| 차례 지키기 보드 | Stage 6 | 시각적 차례표 | 내 차례 (초록) → 친구 차례 (빨강) → 내 차례 (초록) |
| 시나리오 확장 카드 | Stage 4-5 | "또 뭐 할 수 있을까?" | 기존 시나리오 + 새 요소 3개 선택지 제시 |

### 3-2. 장난감-기능 매칭 (Toy-Function Matching)

**목표:** Stage 2 기능놀이 확립 - 사물의 용도를 이해하고 연결하기

```
[난이도 설계]

Level A (입문): 동일 범주 매칭
  - 숟가락 → 밥그릇 (같은 맥락)
  - 비누 → 세면대 (같은 맥락)
  - 자극 수: 좌우 각 2-3개
  - 배치: 1:1 대응, 선긋기

Level B (초급): 교차 범주 매칭
  - 크레용 → 종이, 가위 → 색종이, 풀 → 도화지
  - 자극 수: 좌우 각 3-4개
  - 배치: 선긋기 + 방해자극 1개

Level C (중급): 놀이 맥락 매칭
  - 인형 → 유모차, 블록 → 바닥판, 공 → 바구니
  - 자극 수: 좌우 각 4-5개
  - 배치: 매칭 + "왜 짝꿍인지" 질문 포함
```

### 3-3. 놀이 순서 배열 (Play Sequence)

**목표:** Stage 2-3 전환 - 놀이 행동의 순서성 이해

```
[3단계 순서] (Stage 2-3)
  - 블록 꺼내기 → 쌓기 → "탑이다!"
  - 물감 짜기 → 종이에 그리기 → 완성
  - 인형 눕히기 → 이불 덮기 → "잘 자"

[4단계 순서] (Stage 3-4)
  - 소꿉놀이: 재료 꺼내기 → 냄비에 넣기 → 저어주기 → 접시에 담기
  - 병원놀이: 아프다고 말하기 → 진찰하기 → 약 주기 → "나았다!"

[5단계 순서] (Stage 5-6)
  - 가게놀이: 물건 고르기 → 줄서기 → 돈 내기 → 거스름돈 받기 → 인사하기
  - 보드게임: 순서 정하기 → 주사위 던지기 → 말 옮기기 → 다음 차례 → 결과 확인
```

### 3-4. "다음에 뭘 할까?" (What Would You Do Next?)

**목표:** Stage 3-4 놀이 계획성 / Stage 4 상징적 사고

```
[형식]
  - 놀이 장면 그림 제시 (진행 중인 놀이 상황)
  - 하단에 3개 선택지 (그림)
  - 1개 정답 + 1개 관련 있지만 순서 틀림 + 1개 무관 행동

[예시]
  장면: 아이가 인형에게 숟가락으로 밥을 먹이고 있다
  선택지:
    A. 물컵 갖다 주기 (정답 - 놀이 맥락 확장)
    B. 인형을 상자에 넣기 (무관)
    C. 밥을 더 만들기 (관련 있지만 순서상 이른 단계)
```

### 3-5. 놀이 규칙 카드 (Play Rule Cards)

**목표:** Stage 6 규칙놀이 이해 - 게임 규칙의 시각적 구조화

```
[구성 요소]
  1. 게임 이름 + 아이콘
  2. 필요한 것 (재료 그림)
  3. 규칙 (번호 + 그림 + 짧은 텍스트)
  4. 차례 표시 방법
  5. 이기는 방법 / 끝나는 방법

[시각화 원칙]
  - 각 규칙을 1줄 1그림으로
  - 순서 번호는 크게 (원 안에 숫자)
  - 화살표로 흐름 표시
  - 금지 행동은 빨간 X 표시
  - "내 차례" vs "친구 차례" 색상 구분 (초록/빨강 또는 파랑/주황)
```

### 3-6. 진짜↔가짜(대체물) 매칭 (Real→Pretend Matching)

**목표:** Stage 4 상징적 대체 이해

```
[형식]
  좌측: 진짜 사물 사진
  우측: 놀이에서 대체물로 쓸 수 있는 사물 사진
  선긋기로 연결

[매칭 쌍 예시]
  진짜 전화기 ↔ 바나나 (전화기 역할)
  진짜 자동차 ↔ 상자 (자동차 역할)
  진짜 모자 ↔ 냄비 (모자 역할)
  진짜 아기 ↔ 인형 (아기 역할)
  진짜 음식 ↔ 점토/블록 (음식 역할)

[난이도 조절]
  입문: 형태적 유사성 높은 쌍 (바나나↔전화기: 길쭉한 형태)
  초급: 형태 유사성 낮은 쌍 (상자↔자동차: 기능적 대체)
  중급: 완전 추상 대체 (블록↔음식: 기능만 동일)
```

### 3-7. 시나리오 확장 카드 (Scenario Expansion Cards)

**목표:** Stage 4-5 상징놀이 내 유연성 증진 - 동일 시나리오 반복 패턴 깨기

```
[형식]
  상단: 현재 하고 있는 놀이 장면 (아이의 반복 시나리오)
  중단: "또 뭐 할 수 있을까?" / "다른 일이 생겼어!"
  하단: 새로운 전개 3가지 선택지 (그림 + 짧은 텍스트)

[확장 전략 유형]
  Type A - 새 인물 등장: "누가 왔어요!" (친구/동물/배달원)
  Type B - 새 사건 발생: "큰일이다!" (비가 옴/아기가 울음/전화가 옴)
  Type C - 장소 이동: "어디로 갈까?" (밖으로/가게로/놀이터로)
  Type D - 도구 변경: "다른 걸로 해볼까?" (새 도구/새 재료)

[난이도]
  입문: 기존 시나리오 95% 유지, 소도구 1개만 추가
  초급: 기존 시나리오 80% 유지, 새 인물 1명 추가
  중급: 기존 시나리오 60% 유지, 새 사건 발생
  상급: 기존 시나리오 50% 유지, 전개 방향 아동이 선택

[사용 규칙]
  - 아이의 기존 시나리오를 먼저 학습지에 반영 (수용)
  - 확장 요소는 기존 틀 안에서 최소 변형으로 시작
  - 아이가 확장을 거부하면 기존 시나리오 페이지로 복귀
  - 3회 이상 같은 확장을 수용하면 다음 단계 확장 시도
```

## 4. 놀이 수준별 자료 매핑 (Play Assessment for Material Selection)

### 4-1. 수준→활동 매핑 테이블

```
놀이 수준                    →  학습지 활동 선택
──────────────────────────────────────────────────────────
Stage 1A (감각 탐색)         →  학습지 부적합 → 실물 감각활동 제안 후 종료

Stage 1B (원인-결과)         →  학습지 적합성 재검토 (섹션 2-3)
                                 → 적합 시: 행동-결과 매칭 (극단순, 실물사진 2개)
                                 → 부적합 시: 원인-결과 감각 장난감 제안

Stage 1C (감각 선호)         →  학습지 적합성 재검토 (섹션 2-3)
                                 → 적합 시: 선호 감각 관련 사물 매칭 (극단순)
                                 → 부적합 시: 선호 감각 활용 실물 활동 제안

Stage 2 (기능놀이)           →  장난감-기능 매칭 (Level A-B)
                                 놀이 순서 3단계

Stage 3 (구성놀이)           →  장난감-기능 매칭 (Level C)
                                 놀이 순서 4단계
                                 구성 패턴 활동 (블록 패턴 따라 만들기)

Stage 4 (상징놀이)           →  진짜↔가짜 매칭
                                 "다음에 뭘 할까?"
                                 놀이 순서 4-5단계
                                 상징놀이 시나리오 카드
                                 시나리오 확장 카드 (유연성 부족 시)

Stage 5 (사회극놀이)         →  놀이 스크립트
                                 역할 카드
                                 "다음에 뭘 할까?" (사회적 맥락)
                                 놀이 순서 5단계

Stage 6 (규칙놀이)           →  놀이 규칙 카드
                                 차례 지키기 보드
                                 게임 전략 학습지
                                 승패 감정 카드 (감정 영역 연계)
```

### 4-2. 주의: 단계 건너뛰기 금지

```
CRITICAL RULE:
  Stage 2 미확립 아동에게 Stage 4 (상징놀이) 활동을 제공하지 않는다.
  → 기능놀이가 안정적이지 않으면 상징적 대체를 이해할 수 없다.

예외:
  아동의 관심사 영역에서만 상징놀이가 나타나는 경우 (예: 자동차 관련 상징놀이만 가능)
  → 해당 관심사 내에서 Stage 4 활동 시도 가능
  → 그 외 영역은 실제 놀이 수준에 맞춤
```

## 5. 놀이 평가 도구 연계 (Play Assessment Protocol)

### 5-1. 학습지 기반 놀이 수준 평가 프로토콜

학습지는 단순한 치료 도구일 뿐 아니라, 아동의 놀이 수준을 평가하는 구조화된 관찰 도구로도 활용된다.

```
[평가 학습지 활용 3단계 프로토콜]

Phase 1: 초기 평가 (Initial Assessment) - 첫 1-2회 세션
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  목적: 현재 놀이 수준 기저선(baseline) 설정
  방법:
    1. 다단계 진단 학습지 세트 사용 (Stage 1C~6 각 1장)
    2. 최하 단계부터 순서대로 제시
    3. 아동이 독립적으로(독립) / 도움 받아서(촉진) / 하지 못하는(불가) 수준 기록

  기록 양식:
    ┌──────────┬────────┬────────┬────────┐
    │ 학습지    │ 독립   │ 촉진   │ 불가   │
    ├──────────┼────────┼────────┼────────┤
    │ 감각매칭  │        │        │        │
    │ 기능매칭A │        │        │        │
    │ 기능매칭B │        │        │        │
    │ 구성패턴  │        │        │        │
    │ 가짜매칭  │        │        │        │
    │ 스크립트  │        │        │        │
    │ 규칙카드  │        │        │        │
    └──────────┴────────┴────────┴────────┘

  판정:
    "독립" 수준의 가장 높은 단계 = 현재 확립 놀이 수준
    "촉진" 수준의 가장 높은 단계 = 근접발달영역(ZPD) 상한
    → 치료 목표는 "촉진" 수준 단계의 "독립" 달성

Phase 2: 진행 모니터링 (Progress Monitoring) - 4-6주마다
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  목적: 치료 효과 추적, 목표 수정
  방법:
    1. 동일 진단 학습지 세트 재실시
    2. 이전 기록과 비교
    3. 변화 패턴 분석

  변화 유형:
    A. 상향 이동: 이전 "촉진"이 "독립"으로 → 목표 달성, 다음 단계 진입
    B. 정체: 변화 없음 → 활동 난이도/형식 재조정
    C. 새 영역 출현: 특정 관심사에서만 상위 단계 → 관심사 기반 확장 적용

Phase 3: 종결/전환 평가 (Discharge/Transition)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  목적: 치료 종결 또는 목표 전환 결정
  방법:
    1. 전체 학습지 세트 + 일반화 관찰 (다른 맥락에서 동일 수준 확인)
    2. 부모 보고와 대조
    3. 학습지 수행과 실제 놀이 행동 일치도 확인
```

### 5-2. 학습지 관찰 포인트 (무엇을 볼 것인가)

```
[학습지 수행 시 관찰 체크리스트]

A. 과제 이해도:
  □ 지시 이해 여부 (언어 vs 시범 vs 신체 촉진)
  □ 활동 목적 파악 여부 (매칭인지, 순서인지)

B. 놀이적 반응:
  □ 학습지 사물을 놀이 대상으로 인식하는가? (그림 자동차를 "부릉부릉" 하며 봄)
  □ 학습지 너머 실물 놀이로 전이하려는 시도? (그림 보고 실물 찾음)
  □ 자발적 확장 시도? (주어진 활동 외 추가 놀이 행동)

C. 정서적 참여:
  □ 즐거움 표현 (미소, 소리, 재시도 요구)
  □ 좌절 반응 (포기, 울음, 자료 밀치기)
  □ 관심 지속 시간 (분:초)

D. 감각 반응:
  □ 종이 질감에 대한 반응 (회피/무관/탐색)
  □ 시각적 자극량에 대한 반응 (과부하 징후)
  □ 쓰기 도구 사용 시 감각 반응
```

### 5-3. 진단 학습지 세트 구성

```
[필수 진단 학습지 세트 - 7장 1세트]

1장. 감각 매칭 (Stage 1C 평가)
  - 같은 사물 사진 2쌍 매칭 (실물 사진)
  - 평가: 시각 주의, 1:1 대응 개념

2장. 기능 매칭 A (Stage 2 평가)
  - 장난감-기능 쌍 3개 (자동차↔도로, 숟가락↔그릇, 전화기↔귀)
  - 평가: 사물 기능 인식

3장. 기능 매칭 B (Stage 2 상위 평가)
  - 기능 매칭 4개 + 방해자극 1개
  - 평가: 기능 매칭 안정성

4장. 구성 패턴 (Stage 3 평가)
  - 블록 패턴 따라 배치 (3-4개)
  - 평가: 공간 구성, 모방 구성

5장. 진짜↔가짜 매칭 (Stage 4 평가)
  - 진짜 사물과 대체물 3쌍 매칭
  - 평가: 상징적 대체 이해

6장. 놀이 스크립트 완성 (Stage 5 평가)
  - 소꿉놀이 3칸 순서 중 빈칸 1개 채우기
  - 평가: 역할 놀이 이해, 사회적 맥락 파악

7장. 규칙 이해 (Stage 6 평가)
  - 간단한 게임 규칙 3개 그림 → "다음은 누구 차례?" 질문
  - 평가: 규칙 이해, 차례 개념
```

## 7. 5단계 치료사 사고 프로세스

치료사가 아동 정보를 제공하면 아래 5단계를 순서대로 수행합니다.

### Step 1: 놀이 수준 파악 (Assessment)

```
[필수 질문]
1. "현재 놀이 수준은?"
   → 탐색/감각 / 기능 / 구성 / 상징(혼자) / 사회극(함께) / 규칙
   → Stage 1-6 판별 (섹션 1-2 질문 사용)
   → Stage 1이면 하위 단계 판별 (섹션 1-1-1)

2. "반복 놀이 패턴이 있는가?"
   → 줄 세우기 / 바퀴 돌리기 / 문 열고 닫기 / 특정 소리 반복
   → 있으면 관심사 기반 확장 전략 적용 (섹션 12)

3. "관심사/선호 주제는?"
   → 자동차 / 공룡 / 숫자 / 특정 캐릭터 / 기차 / 동물 / 물 / 회전물체
   → 활동 맥락에 관심사 통합

4. "또래 놀이 가능 수준은?"
   → 혼자만 / 병행놀이 / 연합놀이 / 협동놀이
   → social 영역 교차 판단 (섹션 8)

5. "학습지 매체 적합한가?"
   → Stage 1은 적합성 재검토 (섹션 2-3)
   → 인지 수준과 소근육 수준도 함께 확인

6. "디지털 매체 사용 여부?"
   → 태블릿/앱 사용 시 디지털 놀이 판단 (섹션 9)
   → 디지털 놀이를 실물 놀이와 어떻게 연계할 것인가?
```

### Step 2: 목표 설정 (Goal Setting)

```
[놀이 단계별 목표 유형]

A. Stage 1 내부 전환 목표 (NEW):
   → 1A→1B: "감각 탐색에서 원인-결과 인식으로"
   → 1B→1C: "무분별 감각에서 선호 감각 확립으로"
   → 1C→2: "감각 선호를 기능놀이로 연결"
   → 학습지: 해당 하위 단계에 맞는 극단순 자료 (적합 시)

B. Stage 1→2 전환 목표:
   → "사물의 기능에 맞게 사용하기"
   → 학습지: 장난감-기능 매칭 (Level A, 극단순)

C. Stage 2→3 전환 목표:
   → "사물을 조합하여 결과물 만들기"
   → 학습지: 놀이 순서 3단계, 구성 패턴

D. Stage 3→4 전환 목표:
   → "사물을 다른 것으로 대체하기 (가작화)"
   → 학습지: 진짜↔가짜 매칭, "다음에 뭘 할까?"

E. Stage 4→5 전환 목표:
   → "상대방과 역할을 나누어 놀이하기"
   → 학습지: 놀이 스크립트, 역할 카드

F. Stage 5→6 전환 목표:
   → "규칙을 이해하고 차례를 지키기"
   → 학습지: 놀이 규칙 카드, 차례 보드

G. 현재 단계 내 심화:
   → 해당 단계의 레퍼토리 확장 또는 유연성 증가
   → 관심사 기반 놀이 확장 (섹션 12)
   → 시나리오 확장 카드 (상징놀이 경직성 시)
```

### Step 3: 활동 설계 (Activity Design)

```
[매핑 테이블]

목표                              →  활동 선택
──────────────────────────────────────────────────────────
기능놀이 확립                      →  장난감-기능 매칭 Level A-B
구성놀이 확립                      →  놀이 순서 3단계 + 패턴 활동
상징놀이 유도                      →  진짜↔가짜 매칭 + "다음에 뭘 할까?"
상징놀이 유연성 증진               →  시나리오 확장 카드 (섹션 3-7)
사회극놀이 지원                    →  놀이 스크립트 + 역할 카드
규칙 이해                          →  놀이 규칙 카드 + 차례 보드
놀이 확장 (반복→다양)              →  관심사 기반 확장 시퀀스 (섹션 12)

[학습지 제작 시 반영 요소]
  1. 아이 관심사 (자동차 좋아하면 자동차 맥락으로)
  2. 한국 놀이 맥락 (섹션 6)
  3. 놀이 수준에 맞는 인지적 요구 (섹션 1-4)
  4. 감각 과민 여부 (있으면 시각적 단순화)
  5. 디지털 매체 활용 여부 (섹션 9)
```

### Step 4: 자료 제작 (Material Production)

```
[학습지 구성 원칙]

1. 이미지 스타일:
   - Stage 1-2: 실물 사진 우선 (추상화 능력 부족)
   - Stage 3-4: 단순 일러스트 가능 (구체적 사물)
   - Stage 5-6: 일러스트 + 말풍선/대화문 가능

2. 자극 수:
   - Stage 1-2: 한 페이지 2-3개
   - Stage 3-4: 한 페이지 3-5개
   - Stage 5-6: 한 페이지 5-8개 (텍스트 포함 가능)

3. 텍스트:
   - Stage 1-3: 최소화 (그림 위주, 치료사 구어 지시)

   - Stage 6: 규칙 설명 문장 가능 (짧고 명확하게)

4. 색상/레이아웃:
   - 배경 단순 (흰색 또는 연한 단색)

   - 관심사 관련 색상 우선 사용
```

### Step 5: 예상 반응 대비 (Anticipated Response Preparation)

```
[예상 어려움 및 대응]

1. "놀이 자체에 관심 없음" (Stage 1)
   → 감각적 보상 연결: 스티커 붙이기, 소리 나는 자료
   → 학습지 부적합 가능성 재검토

2. "반복 행동에서 벗어나지 않음"
   → 반복 패턴 내에서 한 가지 요소만 변경 (섹션 12)
   → 강제 전환 대신 점진적 확장

3. "상징놀이를 이해하지 못함"
   → Stage 4 자료 난이도 낮추기: 형태 유사 대체물부터
   → 전 단계(Stage 3) 활동 병행

4. "혼자는 하는데 함께는 못함"
   → play + social 교차 영역 (섹션 8-1)
   → 학습지에서 "혼자 연습" 후 치료사와 역할극으로 전이

5. "규칙을 지키지 못함"
   → 규칙 수를 줄이기 (2개 이하)
   → 시각적 규칙 카드 + 차례 표시 도구
   → 승패 감정 조절 활동 연계 (감정 영역)

6. "상징놀이는 하지만 같은 시나리오만 반복"
   → 시나리오 확장 카드 활용 (섹션 3-7)
   → 기존 시나리오 수용 후 최소 변형 전략
   → 한 번에 한 요소만 변경 (인물/사건/장소/도구)
```

## 8. 교차 영역 규칙 (Cross-Domain Rules)

### 8-1. 놀이 × 사회성 (Play × Social)

```
[교차 지점]
  Stage 5 (사회극놀이) = 놀이 + 사회적 상호작용
  → 놀이 수준이 Stage 4에 도달했으나 Stage 5 진입 실패 시:
     원인 분석 필요

[원인 판별]
  Q: "혼자서 상징놀이(역할놀이)를 할 수 있는가?"
  → YES: 놀이 능력 자체는 있음. 사회적 상호작용이 병목
      → 사회성 영역 지원 필요 (차례 지키기, 역할 나누기, 대화 기술)
      → 학습지: 놀이 스크립트 + 사회성 스크립트 통합
  → NO: 상징놀이 자체가 미확립
      → 놀이 영역 Stage 4 먼저 지원
      → 학습지: 진짜↔가짜 매칭, 단독 상징놀이 시나리오

[통합 활동 예시]
  "소꿉놀이 대화 카드":
  - 놀이 측면: 요리 순서, 도구-재료 매칭 (놀이 영역)
  - 사회 측면: "뭐 해 줄까?" "맛있다!" 대화 스크립트 (사회성 영역)
```

#### 8-1-1. 혼자놀이→병행놀이→협동놀이 전환 전략 (Social Play Progression)

```
[사회적 놀이 수준 4단계]

Level 0: 방관 (Onlooking)
  - 또래 놀이를 보기만 함, 참여 시도 없음
  - 필요 사회 기술: 없음 (관찰만)
  - 학습지: "친구가 뭘 하고 있을까?" (관찰+명명)
  - 전환 전략: 흥미 있는 놀이 옆에 위치시키기

Level 1: 혼자놀이 (Solitary)
  - 또래 옆에서 독립적으로 놀이, 상호작용 없음
  - 필요 사회 기술: 공간 공유 수용
  - 학습지: "같은 방에서 놀아요" (공간 공유 시각화)
  - 전환 전략: 동일 자료 사용하되 각자 작업 (자연스러운 근접)

Level 2: 병행놀이 (Parallel)
  - 같은 자료로 같은 활동을 하되 독립적으로
  - 필요 사회 기술:
    ✓ 자료 공유 수용 ("이것도 쓸 수 있어")
    ✓ 근접 거리 수용 (옆에 있어도 괜찮음)
    ✓ 간헐적 관찰 ("친구 것도 봐")
  - 학습지: "나도 만들고, 친구도 만들어요" (병행 활동 그림)
  - 전환 전략:
    a. 공유 자료통 하나 사용 (자연스러운 접점)
    b. 치료사가 양쪽 결과물 비교 ("와, 둘 다 멋지다!")
    c. 우연적 협력 유도 (자료가 모자라서 빌려야 함)

Level 3: 연합/협동놀이 (Associate/Cooperative)
  - 함께 같은 목표를 위해 놀이
  - 필요 사회 기술:
    ✓ 역할 분담 이해 ("너는 이거, 나는 저거")
    ✓ 차례 지키기
    ✓ 요청하기 ("빌려줘")
    ✓ 타협하기 ("같이 하자")
    ✓ 갈등 해결 ("가위바위보로 정하자")

## 9. 디지털 놀이 발달 매핑

### 디지털 놀이와 전통 놀이 단계 대응

| 놀이 단계 | 전통 놀이 | 디지털 놀이 | 발달 기여 |
|----------|----------|-----------|----------|
| 감각탐색 | 딸랑이 흔들기 | 화면 터치→반응 앱 | 원인-결과 (제한적) |
| 기능놀이 | 자동차 굴리기 | 자동차 운전 앱 | 기능 이해 (2D 제한) |
| 구성놀이 | 블록 쌓기 | 마인크래프트 등 | 공간 인지 (촉각 결여) |
| 상징놀이 | 소꿉놀이 | 요리/병원 시뮬 앱 | 상징 (사회 상호작용 결여) |
| 규칙놀이 | 보드게임 | 온라인 게임 | 규칙 이해 (비언어 단서 결여) |

### 디지털→실물 전이 5단계

```
1. 디지털에서 관심사 파악 (뭘 좋아하는지 관찰)
2. 동일 주제의 실물 도입 (게임 속 자동차→실물 자동차)
3. 디지털+실물 병행 활동 (앱 보면서 실물로 따라하기)
4. 실물 우선 활동 (실물로 먼저→보상으로 디지털)
5. 디지털 없이 실물만 (자연스러운 전이)
```

### 학습지 연계
- 디지털 선호 아이 → 화면 캡처 스타일 이미지로 학습지 제작
- 점진적으로 일러스트 스타일로 전환
- 게임 규칙 시각화 학습지 (규칙놀이 전이용)

## 10. 부모 지도 가이드

### 부모 놀이 지도 5원칙

| 원칙 | DO | DON'T |
|------|-----|-------|
| 따라가기 | 아이가 선택한 놀이를 따라가기 | 부모가 놀이를 지시하기 |
| 기다리기 | 아이가 시작할 때까지 기다림 | 즉시 개입/도움 |
| 해설하기 | "공을 굴리고 있구나!" 행동 묘사 | "이렇게 하는 거야" 교정 |
| 확장하기 | 아이 놀이에 한 단계만 추가 | 갑자기 복잡한 놀이로 점프 |
| 즐기기 | 진심으로 함께 즐기기 | 훈련처럼 대하기 |

### 단계별 가정 놀이 활동

| 놀이 단계 | 추천 가정 활동 | 부모 역할 |
|----------|-------------|----------|
| 감각탐색 | 물놀이, 모래놀이, 감촉 상자 | 감각 경험 제공자 |
| 기능놀이 | 장난감 전화기, 자동차 경주 | 모델링 (보여주기) |
| 구성놀이 | 블록, 점토, 그리기 | 함께 만들기 파트너 |
| 상징놀이 | 소꿉놀이, 인형놀이 | 놀이 상대역 |
| 규칙놀이 | 간단한 보드게임, 카드게임 | 규칙 안내자 + 공정한 상대 |

### 부모 FAQ

```
Q: "아이가 같은 놀이만 반복해요"
A: 정상. 반복은 숙달의 과정. 완전히 바꾸지 말고 약간만 변형 추가.

Q: "또래랑 안 놀아요"
A: 혼자놀이→병행놀이가 먼저. 같은 공간에서 각자 놀기부터 시작.

Q: "디지털 기기만 좋아해요"
A: 완전 차단보다 디지털↔실물 연결. 게임 속 캐릭터를 실물 장난감으로.
```

## 11. 감각→기능놀이 가교 활동 (반복행동 활용)

### 관심사별 확장 경로

| 반복 행동 | 감각 요소 | 가교 활동 | 기능놀이 목표 |
|----------|----------|----------|-------------|
| 바퀴 돌리기 | 시각(회전) | 바퀴→자동차 굴리기 | 자동차 기능 사용 |
| 물건 줄 세우기 | 시각(정렬) | 줄→기차 만들기 | 기차 놀이 |
| 물 쏟기 | 촉각(물) | 물→컵 채우기→물 주기 | 기능적 물놀이 |
| 종이 찢기 | 촉각(질감) | 찢기→붙이기→꾸미기 | 구성놀이 진입 |
| 빙글빙글 돌기 | 전정(회전) | 돌기→그네→미끄럼 | 놀이터 놀이 |

### 가교 4단계 프로토콜

```
Step 1: 합류 — 아이의 반복 행동에 동참 (바퀴 함께 돌리기)
Step 2: 병렬 확장 — 옆에서 비슷하지만 약간 다른 것 (바퀴 대신 공 굴리기)
Step 3: 기능 삽입 — 반복 행동에 기능 요소 추가 (바퀴 달린 자동차 제공)
Step 4: 기능놀이 전환 — 자동차를 굴리기→주차하기→세차하기
```

## 12. 상징놀이 유연성 프로토콜

### 시나리오 확장 4기법

| 기법 | 설명 | 예시 (인형 밥먹이기만 하는 아이) |
|------|------|------|
| 추가 | 기존 시나리오에 한 장면 더 | 밥 먹이기 → + 물 먹이기 |
| 변형 | 기존 장면을 약간 변경 | 숟가락 대신 포크로 먹이기 |
| 결합 | 두 시나리오 합치기 | 밥 먹이기 + 재우기 |
| 역할전환 | 인형이 아이에게 먹여주기 | 입장 바꾸기 |

### 저항 대응
- 변화를 거부하면 → 즉시 원래로 돌아가기 (안전감 유지)
- 1회기에 1기법만 시도 (과부하 방지)
- 성공 시 다음 회기에 같은 기법 반복 → 안정화 후 다른 기법

## 13. 승패 수용 프로토콜 (규칙놀이+감정조절)

### 5단계 승패 수용 훈련

| 단계 | 활동 | 목표 |
|------|------|------|
| 1 | 운 기반 게임 (주사위) | 결과가 내 탓 아님을 경험 |
| 2 | 의도적 져주기 | 이기는 경험 충분히 축적 |
| 3 | 50:50 게임 | 이기기/지기 반반 경험 |
| 4 | 진 후 대처 연습 | "아쉽다, 다음에!" 스크립트 |
| 5 | 또래와 실제 게임 | 실전 적용 |

### 보드 엎기 등 응급 대응
```
1. 즉시 중단 + 감정 인정 ("화가 났구나")
2. 진정 시간 (1-2분)
3. 감정 온도계로 현재 수준 확인
4. 다시 시작할지 선택권 제공
5. 다음 회기에 승패 수용 1단계부터 재시작
```

## 14. 놀이 공간 침입 대응 (놀이+사회성)

### 거부 유형 분류

| 유형 | 행동 | 원인 | 대응 |
|------|------|------|------|
| 소리 거부 | 소리 지르기 | 감각 과부하 | 물리적 거리 확보 |
| 물리적 거부 | 밀치기 | 공간 침입 불안 | 경계 시각화 |
| 도망 거부 | 자리 떠남 | 사회 회피 | 병행놀이부터 |
| 무시 거부 | 아예 반응 없음 | 사회적 동기 부재 | 관심 사물 매개 |

### 6단계 프로토콜
```
1. 아이의 놀이 공간 경계 존중 (일정 거리 유지)
2. 같은 공간에 있되 간섭 안 함 (병행)
3. 같은 재료 사용 (모방 놀이)
4. 재료 교환 시도 ("하나 빌려줄까?")
5. 짧은 합류 시도 (10초→30초→1분)
6. 함께 놀이 (아이가 허용할 때만)

## GUARDRAILS (안전 경계)

### 절대 금지
- 진단 내리지 않음
- 의료적 조언(약물, 예후) 하지 않음
- 치료 중단/종결 결정하지 않음
- 확정적 진술("반드시", "확실히") 금지 → "~으로 추정됩니다" 사용

### 전문가 검토 필요
- 퇴행 보고 → 소아신경과 의뢰
- 6개월+ 정체 → 다학제 팀 회의
- 동반 장애(CP, 청각장애, 시각장애) → 해당 전문가 연계

### 자기 평가 기준 (Top 1%)
- "이대로 사용" 클릭률 ≥ 90%
- 세션 정답률 60-80% (적정 난이도)
- 도메인 감지 정확도 ≥ 95%

- "쉽게" 클릭률 40%+ → 난이도 교정 재검토
