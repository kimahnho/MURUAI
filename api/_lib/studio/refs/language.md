# Language Domain Specialist (언어치료 전문가)

> 이 문서를 읽은 AI는 한국 1급 언어재활사(15년+ 경력)처럼 사고한다.
> 대상: ASD, 지적장애, 언어발달지연 등 발달장애 아동 (만 1-12세)
> 근거: 한국어 발달 규준(김영태, 2014; 김민정 & 배소영, 2005; 김수진 & 신지영, 2015)

---

## 1. 한국어 조음 발달

### 1.1 자음 습득 연령표 (75% 기준)

| 연령 | 습득 자음 | 범주 |
|------|----------|------|
| 2세 | ㅂ*(ㅃ), ㄱ*(ㄲ), ㅍ, ㅎ, ㅌ, ㅋ | 경음, 격음 일부 |
| 3세 | ㅂ, ㄷ, ㅁ, ㄴ, ㅈ, ㅊ, ㅉ, ㄱ, ㅇ(종성) | 평음, 비음, 파찰음 |
| 4세 | **ㅅ** | 치조마찰음 (늦은 습득) |
| 5세 | **ㄹ** | 유음 (가장 늦은 습득) |
| 6-7세 | 전체 완성 | 성인 수준 조음 |

### 1.2 연령 경계 판단 규칙

```
Q: 목표 음소의 습득 연령이 지났는가?
├─ YES (예: 5세인데 ㅅ 미산출) → 치료 시작
├─ 경계선 (예: 3세11개월 + ㅅ 미산출)
│   체크리스트:
│   □ 자극반응도: 모방 시 산출 가능?
│   □ 오류 패턴: 대치/생략/왜곡?
│   □ 또래 비교: 현저히 느린가?
│   □ 의사소통 영향: 발음이 의사소통 방해?
│   → 2개+ YES → 치료 권장
│   → 1개 이하 → 3개월 모니터링
└─ 아직 멀리 (예: 3세 + ㄹ 미산출) → 정상 발달
```

### 1.3 조음 오류 패턴

| 오류 | 예시 | 의미 | 중재 |
|------|------|------|------|
| 대치 | "사과"→"타과" | 위치/방법 혼동 | 변별→산출 |
| 생략 | "사과"→"아과" | 음소 표상 미형성 | 인식→산출 |
| 왜곡 | ㅅ가 부정확 | 운동 조절 | 조음점 교정 |
| 첨가 | "사과"→"스사과" | 드묾, 유창성 관련 | 유창성 평가 의뢰 |

### 1.4 다중 음소 우선순위

여러 음소 동시 미산출 시:
```
1순위: 발달 순서 (일찍 습득 음소부터) ㅅ(4세) > ㄹ(5세)
2순위: 자극반응도 (모방 가능한 음소부터)
3순위: 기능적 영향 (고빈도 음소 = 의사소통 영향↑)
4순위: 가시성 (양순음 ㅂ > 연구개음 ㄱ)
```

| 갭 | 해석 | 대응 |

| 12-24개월 | 유의미 지연 | 직접 중재 |

## 3. 언어 이해 (수용 언어)

### 3.1 지시 따르기

| 단계 | 예시 | 연령 |
|------|------|------|
| 1단계 | "공 줘" | 2-3세 |
| 2단계 | "공 갖고 와서 상자에 넣어" | 3-4세 |
| 3단계 | "빨간 공을 큰 상자에 넣고 문 닫아" | 5-6세 |
| 조건부 | "파란 거면 왼쪽, 빨간 거면 오른쪽" | 7세+ |

### 3.2 질문 이해 계층

```
뭐?(2세) → 누가?(2-3) → 어디?(3-4) → 언제?(4-5) → 왜?(5-6) → 어떻게?(6-7)
```

### 3.3 이야기 이해

| 수준 | 질문 | 연령 |
|------|------|------|
| 사실 | "누가 나왔어?" | 3-4세 |
| 순서 | "먼저 뭘 했어?" | 4-5세 |
| 원인 | "왜 울었어?" | 5-6세 |
| 감정추론 | "기분이 어때?" | 6-7세 |
| 예측 | "다음에 어떻게 될까?" | 7세+ |

## 4. 언어 표현 (표현 언어)

### 4.1 한국어 MLU (어절)

| 연령 | MLU | 예시 |
|------|-----|------|
| 18-24개월 | 1.0-1.5 | "엄마", "물 줘" |
| 24-30개월 | 1.5-2.0 | "엄마 물 줘" |
| 30-36개월 | 2.0-3.0 | "엄마 찬 물 줘" |
| 36-48개월 | 3.0-4.0 | 복문 시작 |
| 48개월+ | 4.0+ | 접속사 연결 |

### 4.2 문법형태소 발달

| 연령 | 문법 | 예시 |
|------|------|------|
| 2세 | 주격 -이/가 | "엄마가" |
| 2-3세 | 목적격 -을/를 | "물을 줘" |
| 3-4세 | 과거 -았/었, 연결 -고 | "먹었어", "먹고 놀자" |
| 4-5세 | 이유 -니까, 추측 | "아프니까" |
| 5-6세 | 존대 -요, 피동/사동 | "주셨어요" |

## 5. 텍스트 라벨 정책

```
조음치료 → 라벨 필수 (목표 음소 단어)
어휘치료 → 라벨 필수 (이미지+한글)
이해치료 → 라벨 없음 (상황 그림만)
AAC → 라벨 필수 (심볼+한글)
폰트: 둥근 고딕, 큰 크기 (A4 최소 20pt)
```

## 6. 5단계 치료사 사고 과정

### Step 1: 수준 파악
수용언어 연령, 표현언어 연령, 조음 미산출 음소 확인, 갭 해석

### Step 2: 목표 설정
- 조음: 음소 우선순위(1.4) → 위치(1.5) → 단어 난이도
- 어휘: 현재 수 → 목표 범주(2.3) → 수용→표현 순서
- 이해: 지시 단계(3.1) → 질문 계층(3.2) → 다음 단계
- 표현: 현재 MLU → 다음 MLU, 문법형태소 확인

### Step 3: 활동 선택
조음→짝맞추기(단어+그림, 라벨 필수), 어휘→카테고리분류/짝맞추기(라벨 필수), 이해→순서맞추기/상황그림(라벨 없음), 표현→상황그림(자유발화 유도)

### Step 4: 자료 제작
단어뱅크(1.5)에서 선택, 범주어휘(2.3)에서 선택, 라벨 정책(5) 적용

### Step 5: 예상 & 대비
- 조음: 초성 80%→종성 이동, 어려우면 음절 수준으로
- 어휘: 10개 중 8개→다음 범주, 5개 미만→같은 범주 반복
- 이해: 80%→다음 단계, 50% 미만→이전 단계

## 7. 엣지케이스

### 7.1 이중언어/다문화
```
베트남: 성조 영향, /f/ 대치
중국: ㅂ/ㅃ 혼동 (경음 없음), 종성 약함
일본: ㄹ/ㄴ 혼동
필리핀: ㅂ/ㅍ 혼동, 종성 탈락

판별: L1에서도 비정상 → 장애 / L1 정상 → 간섭
→ 불확실시 이중언어 전문가 의뢰
```

### 7.2 극단적 수용-표현 갭 (24개월+)
수용 수준 내용 + 표현 수준 반응 방식 (선택하기, 가리키기)

### 7.3 반향어 ASD
즉각/지연 반향어 → 의사소통 의도 있을 수 있음
→ 선택형 활동 적합, 자유 명명 피하기

### 7.4 유창성 장애 (말더듬)
범위 밖. 유창성 전문 치료사 의뢰. 조음 동반 시 조음만 자료 생성.

### 7.5 AAC 아동
핵심어휘 중심, 이미지+라벨 필수, 배경 없는 단일 객체, AAC 시스템과 스타일 통일

## 8. 활동-도메인 매핑

| 목표 | 활동 | 라벨 |
|------|------|------|
| 조음 | 짝맞추기 (단어+그림) | 필수 |
| 어휘 | 카테고리분류, 짝맞추기 | 필수 |
| 이해 | 순서맞추기, 상황그림 | 없음 |
| 표현 | 상황그림 (자유발화) | 없음 |
| 문장 | 순서맞추기 (문장순서) | 없음 |

---
> 언어+인지: "분류하며 이름 말하기" → language primary
> 언어+감정: "감정 단어 표현" → 감정어휘는 language, 감정추론은 emotion
> 언어+사회성: "인사말 연습" → social primary, language secondary

## 9. 언어 이전(Prelinguistic) 의사소통 발달

> 첫 단어 출현 전 단계. 2세까지 첫 단어가 미출현 시 이 단계 점검 필수.

### 발달 이정표

| 연령 | 단계 | 행동 | 의사소통 기능 |
|------|------|------|-------------|
| 0-2개월 | 반사적 발성 | 울음, 생리적 소리 | 불쾌 표현 (비의도적) |
| 2-4개월 | 쿠잉(cooing) | "아~", "우~" 모음 발성 | 사회적 미소 동반 |
| 4-6개월 | 변계적 옹알이 | 자음+모음 시도, 웃음 | 발성 놀이 |
| 6-10개월 | 반복적 옹알이 | "바바바", "마마마" | 의도적 발성 시작 |
| 9-12개월 | 변형 옹알이 | "바다가", 억양 모방 | 자곤, 억양 패턴 |
| 9-12개월 | 의사소통 제스처 | 가리키기, 보여주기, 주기 | 요구/공유 기능 분화 |
| 12-18개월 | 첫 단어 + 제스처 | "맘마" + 가리키기 | 단어-제스처 조합 |

### 의사소통 의도 발달

```
비의도적 (0-8개월): 울음, 몸 움직임, 미소, 눈맞춤

의도적 전언어적 (8-12개월):
  요구: 손 뻗기, 잡아당기기
  사회적: 까꿍 놀이, 주고받기
  공동주의:
    반응적(RJA): 성인 시선 따라가기 (9-10개월)
    주도적(IJA): 가리키며 성인 쳐다보기 (11-12개월)
    ※ IJA 결핍 = ASD 조기 선별 핵심 지표

의도적 언어적 (12개월+): 단어로 요구/명명/거부/인사
```

### 2세 반 첫 단어 미출현 시 점검

```
□ 옹알이 질 (canonical babbling 출현?)
□ 의사소통 제스처 빈도 (요구적/서술적)
□ 공동주의 (RJA/IJA) 확인
□ 청력 선별 결과
□ 구강운동 기능 (빨기, 씹기, 혀 움직임)

→ 공동주의+의사소통 의도 결핍 → ASD 선별 의뢰
→ 옹알이 빈약+구강운동 문제 → 운동말장애 고려
→ 수용 정상+표현만 지연 → 표현언어지연(late talker) 가능
```

## 10. 문법형태소 세부 습득 + 조사 중재

### 조사 습득 순서

| 순서 | 조사 | 습득 연령 | 예시 |
|------|------|----------|------|
| 1 | -이/가 (주격) | 24-30개월 | "엄마**가**" |
| 2 | -을/를 (목적격) | 27-33개월 | "밥**을**" |
| 3 | -에 (처소) | 30-36개월 | "집**에**" |
| 4 | -에서 (동작장소) | 33-39개월 | "밖**에서**" |
| 5 | -도 (보조사) | 30-36개월 | "나**도**" |
| 6 | -(으)로 (도구) | 36-42개월 | "손**으로**" |
| 7 | -한테 (여격) | 36-42개월 | "엄마**한테**" |
| 8 | -의 (관형격) | 42-48개월 | 구어에서 생략 흔함 |
| 9 | -보다 (비교) | 48-54개월 | "나**보다**" |

### ASD 조사 탈락 패턴

```
"엄마 물" (주격+목적격 모두 탈락)
반향어에서는 조사 포함 → 자발화에서 탈락 = 처리 부하 문제
특정 조사 과잉("~도"만 반복), 격식체 과잉("~에게" 미디어 학습)
```

### 조사 중재 전략

| 전략 | 방법 | 예시 |
|------|------|------|
| 대비 모델링 | 조사 유무 대비 | "공 줘" vs "공**을** 줘" |
| 빈칸 채우기 | 캐리어 문구 | "엄마___ 줘" → "엄마**가**" |
| 선택형 | 2개 중 선택 | "엄마가? 엄마를?" |
| 시각 단서 | 색깔 코딩 카드 | 주격=빨강, 목적격=파랑 |

## 11. 이야기 산출(Narrative Production)

### 발달 단계

| 단계 | 연령 | 구조 | 예시 |
|------|------|------|------|
| 나열 | 2-3세 | 무관한 사건 | "강아지. 공. 엄마." |
| 연쇄 | 3-4세 | 시간순, 중심 없음 | "놀았어. 밥 먹었어. 잤어." |
| 원시적 이야기 | 4-5세 | 중심 인물+시간순 | "아이가 공원 갔어. 넘어졌어." |
| 집중 연쇄 | 5-6세 | 인과관계 출현 | "비가 와서 우산 썼어." |
| 완전 이야기 | 6-7세+ | 배경-문제-시도-결과-반응 | 이야기 문법 요소 포함 |

### 이야기 문법(Story Grammar)

```
1. 배경(setting): 시간, 장소, 인물
2. 계기사건(initiating event): 문제 발생
3. 내적 반응(internal response): 감정/생각
4. 시도(attempt): 해결 노력
5. 결과(consequence): 시도의 결과
6. 반응(reaction): 최종 감정/교훈

채점: 6개 중 0-1개=이야기 이전, 2-3=미완성, 4-5=불완전, 6=완전
```

### Story Retelling 중재

| 단계 | 지원 수준 | 활동 |
|------|----------|------|
| 1 | 최대 | 4컷 그림 순서 배열 |
| 2 | 시각+질문 | 그림 보며 "먼저?" "그 다음?" |
| 3 | 도식 | 이야기 지도(배경/문제/해결 칸) |
| 4 | 최소 | 핵심 단어 3-4개만 제시 |
| 5 | 독립 | 듣고 바로 다시 말하기 |

## 12. 화용언어(Pragmatics)

### 발달 이정표

| 연령 | 대화 기술 | 주제 관리 | 청자 고려 |
|------|----------|----------|----------|
| 2-3세 | 1-2회 차례 교대 | 주제 유지 1-2회 | 없음 |
| 3-4세 | 3-4회 차례 교대 | 관련 발화 | 친숙/낯선 구분 |

| 5-6세 | 명료화 요구 | 5회+ 유지 | 존대/반말 구분 |
| 6-7세 | 간접 요구 이해 | 정교화 | 청자 지식 고려 |
| 8세+ | 비유/농담 | 복수 주제 전환 | 공손 전략 |

### 의사소통 기능 발달 (ASD 취약 기능 표시)

| 기능 | 출현 | ASD |
|------|------|-----|
| 요구 | 9-12개월 | 양호 |
| 거부 | 9-12개월 | 양호 |
| 명명 | 12-18개월 | 양호 |
| 공유/보여주기 | 12-18개월 | **취약** |

### ASD 화용 중재

주제 유지: 주제 카드 → 토큰 보드 → 대화 스크립트 → 또래 대화

시각 지원: "내 차례/네 차례" 카드 필수

## 13. 비유적 언어(Figurative Language)

### 유형 및 발달

| 유형 | 이해 연령 | 예시 |
|------|----------|------|
| 직유 | 6-7세 | "토끼처럼 빠르다" |
| 관용어 | 7-9세 | "발이 넓다" = 아는 사람 많다 |
| 은유 | 8-10세 | "시간이 금이다" |
| 과장 | 7-8세 | "배가 터지겠다" |
| 반어 | 9-12세 | 비 올 때 "날씨 좋다" |

### ASD 비유 중재

```
DO: 시각 지원(문자적 vs 비유적 그림 대비), 명시적 규칙, 개인 관용어 사전
DON'T: "느껴봐", 한 번에 많이, 맥락 없이 목록 암기
```

### 한국어 관용어 교수 목록 (8-10세)

| 관용어 | 의미 | 시각 힌트 |
|--------|------|----------|
| 발이 넓다 | 아는 사람 많다 | 여러 사람 연결 |
| 눈이 높다 | 기준 까다롭다 | 고르는 그림 |
| 손이 크다 | 넉넉하다 | 큰 그릇 |
| 식은 죽 먹기 | 매우 쉽다 | 쉬운 과제 |

## 14. 의성어→실제 단어 전환

### 전환 중재 (브릿지 기법)

| 단계 | 전략 | 예시 |
|------|------|------|
| 1 | 병렬 명명 | "멍멍! 강아지!" (동시 제시) |
| 2 | 확장 | "멍멍 강아지 왔네" (명사 삽입) |
| 3 | 선택형 | "멍멍이야? 강아지야?" |
| 4 | 지연 반응 | 의성어에 즉각 반응 안 함 |
| 5 | 독립 명명 | "이게 뭐야?" (의성어 없이) |

핵심: 의성어를 부정하지 않음. "멍멍도 맞지만, 이름은 강아지야" 접근.

## 15. 말 속도/운율(Prosody)

### 속화(Cluttering) vs 말더듬 감별

| 특성 | 속화 | 말더듬 |
|------|------|--------|
| 인식 | 비인식 | 본인 인식 |
| 비유창 유형 | 생략, 축약 | 반복, 연장, 막힘 |
| 주의 시 | 개선 | 악화 가능 |

### ASD 운율 특성

단조로운 억양, 과장된 억양, 부적절한 강세, 속도/볼륨 조절 실패
→ 화용언어(12절)와 연계 중재

## 16. 청각장애 동반 조절

### 핵심 규칙

| 항목 | 일반 | 청각장애 조절 |
|------|------|-------------|
| 발달 기준 | 생활 연령 | **청각 연령** (보장구 착용부터) |
| 말 속도 | 자연스러운 | 약간 느리게 (과장 금지) |
| 시각 단서 | 선택적 | **필수** |
| 배경 소음 | 최소화 | **반드시 제거** |

### "어휘 있지만 문장 이해 약함" 패턴

원인: 조사/어미는 음향 에너지 낮음 → 보청기로 잘 안 들림
→ "엄마가 아빠를 밀었어" → "엄마" "아빠" "밀었어"만 인지
→ 조사 강조(acoustic highlighting) + 시각적 문법 표지 필요

## 17. 구어-문해력(Literacy) 연계

### Simple View of Reading

```
읽기 이해 = 해독(Decoding) × 언어 이해
"말은 잘하는데 읽기 이해 안 됨" = 해독○ + 숨겨진 언어 약점
→ 학업 어휘, 복문, 추론이 대화에서는 보상되지만 글에서 노출
```

### 언어재활사 담당 범위

```
담당: 어휘 확장, 복문/문법 이해, 이야기/텍스트 구조, 추론, 음운인식
의뢰: 해독 자체 → 특수교육, 시각 처리 → 안과, 주의력 → ADHD 평가
```

## 18. 러시아어-한국어 이중언어 간섭

### 주요 간섭 패턴

| 특성 | 러시아어 | 한국어 | 간섭 |
|------|---------|--------|------|
| 어순 | SVO | SOV | "나 먹어 밥" |
| 조사 | 없음(격변화) | 조사 | 전체 탈락 |
| 경어 | вы/ты | 복잡한 존대 | 혼란 심함 |
| ㄹ 발음 | trill /r/ | 탄설음/설측음 | 러시아식 /r/ |

### 어순 교정

색깔 블록 어순 (주어=빨강, 목적어=파랑, 서술어=초록)
"한국어는 항상 동사가 마지막!" 규칙 명시적 교수

## 19. 극소 표현어 프로필 (표현어 1-5개, 수용 정상)

> 2세+ 표현어 5개 미만 + 수용 정상 = Late Talker 패턴
> 감별: 수용도 지연 → 전반적 언어지연/지적장애 고려

중재: 현재 보유 어휘 안정화 → 의미적으로 가까운 단어 1-2개 도입
→ "물" 보유 시 "우유, 주스" 도입 (같은 '마시는 것' 범주)
학습지 적합성: 그림 명명 가능? YES→짝맞추기(라벨), NO→실물 명명 우선

## 20. 한국어 경어법/존댓말 중재

> 6세 ASD L1이 선생님에게 반말 → 화용언어 12절 + 존댓말 특화

### 존댓말 오류 유형

| 유형 | 예시 | 중재 |
|------|------|------|
| 전면 반말 | 어른에게 "줘" | 기본 존대 "-요" 붙이기 규칙 |
| 과잉 존대 | 또래에게 "주세요" | 반말/존대 전환 규칙 (대상별) |
| 혼합 | 문장 내 반말+존대 혼재 | 문장 완성 연습 |

중재: "이 사람은 선생님 → -요 붙이기 / 이 사람은 친구 → 반말 OK"
시각 카드: 어른 사진 = 빨간 테두리(존대), 또래 사진 = 파란 테두리(반말)

## 21. 읽기 이해 추론 중재

> 해독 OK + 추론 NO 패턴. Section 17 확장.

### 추론 유형 계층

| Level | 유형 | 예시 | 연령 |
|-------|------|------|------|
| RI-1 | 텍스트 내 사실 | "누가 나왔어?" | 6-7세 |
| RI-2 | 텍스트 내 연결 | "왜 울었어?" (글에 원인 있음) | 7-8세 |
| RI-3 | 텍스트 외 추론 | "다음에 뭘 할까?" (글에 없는 추론) | 8-9세 |
| RI-4 | 감정/동기 추론 | "이 친구 기분이 어떨까?" | 9-10세 |
| RI-5 | 비판적 추론 | "이 이야기가 맞다고 생각해?" | 10세+ |

## 22. 쌍둥이/형제 대리 의사소통

> 형제(특히 쌍둥이)가 대신 말해주는 환경에서 표현 동기 저하

감별: 형제 없이 1:1 상황에서 표현 수준 재측정
→ 1:1에서 더 나음 = 환경적 요인 → 부모 교육(대리 말하기 줄이기)
→ 1:1에서도 동일 = 내재적 언어 지연 → 직접 중재

## 23. 과잉 접속사/문장 분절

> "그리고그리고그리고" 연쇄 = 서술 구조화 부족

중재: "한 번에 한 문장" 규칙 + 마침표 시각 단서
→ 빨간 점(마침표 카드)을 문장 끝에 놓기 연습
→ "그리고" 대신 다양한 연결어 ("그래서", "그런데", "그 다음에")
→ Section 11 이야기 문법과 연계: 배경-문제-해결 구조화

## GUARDRAILS (안전 경계)

### 절대 금지
- 진단 내리지 않음
- 의료적 조언(약물, 예후) 하지 않음
- 치료 중단/종결 결정하지 않음
- 확정적 진술("반드시", "확실히") 금지 → "~으로 추정됩니다" 사용

### 전문가 검토 필요
- 퇴행 보고 → 소아신경과 의뢰
- 6개월+ 정체 → 다학제 팀 회의
- 동반 장애(CP, 청각장애, 시각장애) → 해당 전문가 연계

### 자기 평가 기준 (Top 1%)
- "이대로 사용" 클릭률 ≥ 90%
- 세션 정답률 60-80% (적정 난이도)
- 도메인 감지 정확도 ≥ 95%

### Auto-Research 트리거
- "쉽게" 클릭률 40%+ → 난이도 교정 재검토
- 미등록 키워드 반복 출현 → 참조파일 업데이트 제안
- 새로운 임상 패턴 5회+ → 엣지케이스 추가

## 부록: 언어재활사 자격시험 핵심 임상 지식 (에이전트 추론 참조용)

### 자음정확도(PCC) 연령별 기준
| 연령 | PCC | 해석 |
|------|-----|------|
| 2;6 | 80~85% | 정상 범위 |
| 3세 | 85~90% | 정상 범위 |
| 4세 | 90~95% | 정상 범위 |
| 5세 | 95~98% | 거의 성인 수준 |

### 음운변동(음운과정) 소거 연령
| 음운변동 | 설명 | 소거 연령 |
|---------|------|----------|
| 종성생략 | 받침 탈락: '밥'→'바' | 3세 이전 |
| 음절탈락 | 다음절 축소: '바나나'→'나나' | 3세 이전 |
| 자음조화 | '강아지'→'강아기' | 3세 이전 |
| 파열음화 | 마찰음→파열음: '사과'→'다과' | 3~4세 |
| 전방화 | 연구개→치조: '가방'→'다방' | 3;6~4세 |
| 파찰음화 | 마찰음→파찰음: '사과'→'자과' | 4~5세 |
| 유음단순화 | /ㄹ/→/ㄴ/ 또는 생략 | 5세 |
| 긴장음화 | 이완→경: '바지'→'빠지' | 4~5세 |

음운변동이 소거 연령을 지나도 지속되면 치료 대상.
비전형적(어떤 연령에서도 10% 미만) 음운변동은 즉시 치료 대상.

### 오류 심각도 순서
**생략 > 대치 > 왜곡 > 첨가** (말명료도 영향 순)
- 생략: 가장 심각, 말명료도 크게 저하
- 대치: 중간, 의미 혼동 가능
- 왜곡: 경미, 음성적으로 유사
- 첨가: 드묾

### 조음 치료 접근법 선택 가이드
| 아동 프로필 | 추천 접근법 | 치료 자료 형태 |
|------------|-----------|--------------|
| 소수 오류음, 왜곡 중심 | Van Riper (전통적) | 음소 위치별(어두초성/어중/어말종성) 단어 카드 |
| 다수 음운변동, 패턴 오류 | 음운변동 접근법 (Hodson) | 목표 변동을 유발하는 단어 세트 |
| 의미 변별 어려움 | 최소대립쌍 (Minimal Pairs) | 하나의 음소만 다른 단어쌍 (예: 달 vs 발) |
| 실생활 일반화 필요 | 의사소통중심 접근 | 대화/역할놀이 장면 카드 |

### 언어발달 지연 경고 신호 (Red Flags)
| 연령 | 경고 신호 |
|------|----------|
| 12개월 | 옹알이 없음, 제스처 없음 |
| 18개월 | 한 낱말도 표현하지 못함 |
| 24개월 | 두 낱말 조합 없음, 표현어휘 50개 미만 |
| 36개월 | 3어 조합 없음, 낯선 사람이 50% 이상 이해 못 함 |

### MLU(평균발화길이) 보충
| 연령 | MLU-w(낱말) | MLU-m(형태소) |
|------|-----------|-------------|
| 2세 | 1.5~2.0 | 2.0~3.0 |
| 3세 | 2.5~3.5 | 3.5~4.5 |
| 4세 | 3.0~4.0 | 4.0~5.0 |

MLU가 해당 연령 기대치보다 1.0 이상 낮으면 언어발달 지연 시사.

### 음운인식 발달 순서
| 순서 | 수준 | 연령 | 예시 |
|------|------|------|------|
| 1 | 단어 수준 | 3~4세 | 문장에서 단어 수 세기 |
| 2 | 음절 수준 | 4~5세 | "사과"가 몇 음절? (2개) |
| 3 | 각운(onset-rime) | 5~6세 | "달"과 "발"이 비슷하게 들려 |
| 4 | 음소 수준 | 6~7세 | "사과"의 첫 소리? (ㅅ) |

음운인식은 문해력(읽기/쓰기) 발달의 핵심 선행 기술.
학습지 설계 시 아동의 음운인식 수준에 맞는 과제 선택 필요.

### 상징놀이 발달 순서
| 단계 | 연령 | 설명 | 예시 |
|------|------|------|------|
| 1 | 12~18개월 | 자기 대상 가작화 | 빈 컵으로 마시는 시늉 |
| 2 | 18~24개월 | 타인 대상 가작화 | 인형에게 먹이기 |
| 3 | 24~30개월 | 사물 대치 | 바나나를 전화기처럼 사용 |
| 4 | 30~36개월 | 탈맥락 연속 가작화 | 없는 음식으로 요리놀이 |
| 5 | 36개월+ | 역할놀이/사회극놀이 | 엄마/아빠 역할놀이 |

놀이 도메인 학습지 설계 시 아동의 놀이 발달 단계에 맞는 활동 선택 필수.

### 언어학습장애(LLD) vs SLI vs 지적장애 감별
| 진단 | 비언어 IQ | 언어 | 읽기 |
|------|----------|------|------|
| SLI (단순언어장애) | 정상 | 저하 | 정상~경도 저하 |
| LLD (언어학습장애) | 정상 | 저하 | 저하 |
| 지적장애 | 저하 | 저하 | 저하 |

감별 포인트: LLD는 비언어 IQ가 정상이지만 언어+읽기 모두 저하.

### 집중적 자극(Focused Stimulation) 기법
- 목표 언어형식을 집중적으로 모델링하되, 아동에게 산출을 요구하지 않음
- 자연스러운 놀이/대화 맥락에서 목표형식을 과장하여 반복 제시
- 적용: 표현언어 초기 단계, 모방이 어려운 아동
- 학습지: 목표형식이 반복 등장하는 그림 이야기 카드

### 종성 평가 문맥 설계 원칙
- 종성 음소를 평가할 때 **연음화 환경을 회피**해야 함
- 다음 음절이 모음으로 시작하면 종성이 초성화(연음)되어 정확한 평가 불가
- 올바른 문맥: "호박도" (자음 앞) — 종성 /ㄱ/이 유지됨
- 잘못된 문맥: "호박에" (모음 앞) — /ㄱ/이 다음 음절 초성으로 연음
- 학습지 단어 선정 시에도 같은 원칙 적용

### 독립분석 vs 관계분석
| 분석 유형 | 기준 | 결과 |
|----------|------|------|
| 독립분석 | 아동 자체 산출 | 아동이 산출 가능한 음소/음절구조 목록 |
| 관계분석 | 성인 목표 대비 | 정확도(PCC), 오류 유형, 음운변동 패턴 |

독립분석은 아동의 음운 체계를 이해하는 데 필수.
관계분석은 치료 목표 설정과 진전도 측정에 필수.

### PCC(자음정확도) 계산법
```
PCC = (정조음 자음 수 / 전체 목표 자음 수) × 100

예: "사과 먹어" → 목표 자음: ㅅ,ㄱ,ㅇ,ㅁ,ㄱ,ㅇ (6개)
    정조음: ㄱ,ㅇ,ㅁ,ㄱ,ㅇ (5개), ㅅ→ㄷ 대치 (1개 오류)
    PCC = 5/6 × 100 = 83.3%
```
- PCC-R(수정): 왜곡을 정반응으로 처리 (PCC보다 높게 나옴)
- 종성 자음정확도: 종성 위치만 따로 계산 가능

### 촉진적 음성 문맥(Facilitating Context)
특정 음소가 산출되기 쉬운 음성 환경:
| 목표 음소 | 촉진 모음 | 이유 |
|----------|----------|------|
| ㅅ, ㅆ | /ㅜ/, /ㅡ/ | 입술 둥글림이 혀끝 접근 보조 |
| ㅈ, ㅉ, ㅊ | **/ㅣ/** | 고전설모음이 경구개 접촉 촉진 |
| ㄹ (설측음) | /ㅏ/, /ㅗ/ | 넓은 구강 개방이 설측 기류 촉진 |
| ㄹ (탄설음) | 어중 위치 (VCV) | 연속 조음에서 탄설 쉬움 |

치료 단어 선정 시 촉진적 문맥을 우선 사용하면 초기 성공 경험 가능.

### 주기법(Cycles Approach) — Hodson & Paden
- 대상: 고도 음운장애 (PCC < 50%), 다수 음운변동 보이는 아동
- 핵심: 한 음운변동을 집중 치료 후, 다음 변동으로 전환. 순환 반복.
- 절차:
  1. 자극반응도 평가 → 목표 패턴 선정
  2. 듣기 활동 (1분): 목표 패턴 단어 듣기
  3. 산출 연습 (놀이 맥락): 목표 단어 20~30회 산출
  4. 듣기 활동 (1분): 다음 목표 단어 미리 듣기
  5. 가정 연습: 부모에게 듣기 목록 제공
- 주기: 1변동당 2~4주 → 다음 변동 → 전체 순환 후 재평가

### 비음 동화/첨가 오류 패턴
| 오류 | 예시 | 분석 |
|------|------|------|
| 비음 동화 | "물고기"→"뭉고기" | ㅁ이 후행 자음에 비음 영향 |
| 비음 첨가 | "우산"→"무산" | 어두 초성에 비음 첨가 |
| 비음 대치 | "오리"→"노리" | ㅇ(영성문) → ㄴ(비음) |

비음 관련 오류는 음운변동 분석에서 동화 변동으로 분류.

### 읽기 발달 5단계 (Chall, 1983)
| 단계 | 명칭 | 연령 | 핵심 |
|------|------|------|------|
| 0 | 문해 사회화 | 0~6세 | 글자에 노출, 읽는 흉내 |
| 1 | 해독(Decoding) | 6~7세 | 글자-소리 대응 학습 |
| 2 | 유창성(Fluency) | 7~8세 | 자동적 읽기 |
| 3 | 학습을 위한 읽기 | 9~13세 | 내용 이해 중심 |
| 4 | 비평적 읽기 | 14세+ | 다관점, 비판적 사고 |

문해력 학습지 설계 시 아동의 읽기 발달 단계에 맞는 과제 선택.

### 발달적 vs 비발달적 오류패턴 구분
| 구분 | 정의 | 예시 | 치료 시급성 |
|------|------|------|-----------|
| 발달적 | 해당 연령 아동의 10%+ 사용 | 3세 파열음화 | 관찰/대기 가능 |
| 지체된(delayed) | 10% 미만이나 더 어린 연령에선 10%+ | 5세 종성생략 | 치료 대상 |
| 비발달적(unusual) | 어떤 연령에서도 10% 미만 | 초성 완전 생략, 후방화 | 즉시 치료 |

비발달적 오류는 진단적으로 중요: CAS, 청각장애 등 감별 필요.

### 유음 위치별 난이도
| 유음 유형 | 위치 | 난이도 | 습득 |
|----------|------|--------|------|
| 설측음(lateral) | 종성 | 쉬움 | 3~4세 |
| 탄설음(flap) | 어두초성 | 어려움 | 5세 |
| 탄설음(flap) | 어중초성 | 가장 어려움 | 5~6세 |

치료 시 설측음(종성) → 탄설음(어두) → 탄설음(어중) 순서로 진행.

### 이어진문장 → 내포문 발달 순서
| 유형 | 예시 | 습득 연령 |
|------|------|----------|
| 이어진문장 (등위접속) | "밥 먹고 놀았어" | 3세~ |
| 이어진문장 (대등접속) | "비가 오는데 우산이 없어" | 3;6~ |
| 내포문 (관형절) | "내가 좋아하는 과자" | 4세~ |
| 내포문 (명사절) | "엄마가 왔다는 것을 알아" | 5세~ |

복문 목표 설정 시 이어진문장을 먼저, 내포문은 이후에 목표로 설정.

### 자질 기반 치료 목표 설정
| 자질 | 음소 | 설명 |
|------|------|------|
| 조음위치 | ㅂ(양순) vs ㄷ(치조) vs ㄱ(연구개) | 위치 대립 |
| 조음방법 | ㅂ(파열) vs ㅁ(비음) vs ㅅ(마찰) | 방법 대립 |
| 발성유형 | ㅂ(평음) vs ㅃ(경음) vs ㅍ(격음) | 기식/긴장 대립 |

자질 기반 접근: 하나의 자질(예: 조음위치)을 변경하면 관련 음소 전체에 일반화 기대.

## 부록 2: 1급 기출 기반 보강 지식

### 단서 유형 구분 (이름대기 촉진)
| 단서 유형 | 설명 | 예시 |
|----------|------|------|
| 음향음소적 | 목표 단어의 첫 음소/음절 제공 | "ㅅ..." → "사과" |
| 의미적 | 목표 단어의 범주/기능 힌트 | "과일이야, 빨갛고..." |
| 시각적 | 그림/제스처 제공 | 사과 그림 보여주기 |

치료 활동에서 단서 계층 사용: 시각 → 의미 → 음향음소 순으로 독립성 높임.

### 설명담화(Expository Discourse) 하위 구조
| 구조 | 설명 | 예시 |
|------|------|------|
| 나열 | 항목 열거 | "동물에는 강아지, 고양이, 토끼가 있어요" |
| 비교대조 | 공통점/차이점 | "사과는 빨갛고 바나나는 노랗지만 둘 다 과일이에요" |
| 순서/절차 | 단계별 설명 | "먼저 손을 씻고, 그 다음 밥을 먹어요" |
| 원인결과 | 인과관계 | "비가 오니까 우산을 써요" |
| 목적수단 | 이유와 방법 | "감기에 안 걸리려고 마스크를 써요" |

학습지에서 설명담화 연습: 상황 그림 카드 + 구조별 프레임 제공.

#### 설명담화 중재 난이도 위계
| 순서 | 구조 | 난이도 | 권장 시작 연령 | 이유 |
|------|------|--------|-------------|------|
| 1 | 나열 | 가장 쉬움 | 4~5세 | 항목 열거만 → 인과/시간 불필요 |
| 2 | 순서/절차 | 쉬움 | 5~6세 | 시간 순서만 따름 |
| 3 | 비교대조 | 중간 | 5~6세 | 두 대상 동시 분석 필요 |
| 4 | 원인결과 | 어려움 | 6~7세 | 인과추론 필요 |
| 5 | 목적수단 | 가장 어려움 | 7세+ | 의도+방법 동시 표현 |

이야기 구성이 어려운 아동 → 나열부터 시작 (인지적 부담 최소).

### 상위언어능력(Metalinguistic Ability)
- 정의: 언어 자체를 사고 대상으로 인식하고 조작하는 능력
- 음운인식보다 상위: 문법 판단, 중의성 인식, 비유 이해 포함
- 발달: 6~7세부터 급격히 발달 (문해력과 병행)
- 평가: "이 문장이 이상해?" 판단 과제, 동음이의어 변별
- 학습지 적용: "맞는 문장 찾기", "이상한 부분 고치기" 활동

### AAC (보완대체의사소통)

#### 기기 유형 선택 기준
| 아동 프로필 | 추천 AAC | 이유 |
|------------|---------|------|
| 무발화 + 미세운동 OK | 의사소통 그림판/VOCA | 손가락 지적 가능 |
| 무발화 + 미세운동 어려움 | 시선추적 AAC | 눈으로 선택 |
| 발화 시도 있으나 명료도 낮음 | VOCA + 구어 병행 | 보완적 사용 |
| 기능연령 매우 낮음 (<18개월) | 실물/사진 기반 교환 | PECS Phase I |

#### 핵심어휘 vs 상황어휘
| 유형 | 정의 | 예시 | 비율 |
|------|------|------|------|
| 핵심어휘(Core) | 모든 상황에서 빈번히 사용 | 더, 싫어, 줘, 가, 이거 | 전체 발화의 ~80% |
| 상황어휘(Fringe) | 특정 상황에서만 사용 | 공룡, 미끄럼틀, 김밥 | 전체 발화의 ~20% |

AAC 보드 설계: 핵심어휘를 중앙에, 상황어휘를 주변에 배치.

#### PECS 6단계
| 단계 | 목표 | 활동 |
|------|------|------|
| I | 물리적 교환 | 원하는 물건 그림을 건네기 |
| II | 거리/지속성 | 먼 곳에서도 그림 가져와 교환 |
| III | 그림 변별 | 여러 그림 중 원하는 것 선택 |
| IV | 문장 카드 | "나는 ○○ 원해요" 문장 구성 |
| V | 질문 응답 | "뭐 원해?" 질문에 카드로 응답 |
| VI | 댓글/보고 | 자발적으로 관찰한 것 보고 |

### 비유언어 투명도(Transparency)
| 투명도 | 정의 | 예시 | 중재 순서 |
|--------|------|------|----------|
| 투명 | 문자적 유추 가능 | "시간이 금이다" (시간=귀중) | 먼저 |
| 반투명 | 맥락 힌트로 유추 | "발이 넓다" (아는 사람 많음) | 중간 |
| 불투명 | 유추 어려움 | "식은 죽 먹기" (쉬운 일) | 나중에 |

비유 학습지: 투명→반투명→불투명 순서로 난이도 조절.

### 참조적 의사소통(Referential Communication)
- 정의: 청자의 관점을 고려하여 정보를 전달하는 능력
- 장벽 과제(Barrier Task): 화자와 청자 사이에 장벽을 두고 그림/블록 설명
- 발달: 4~5세 시작, 7~8세 성숙
- 학습지: "친구에게 설명하기" 활동 — 그림을 보고 상대방이 같은 그림을 그릴 수 있게 설명

### 수직적 구조화(Vertical Structuring)
- 정의: 치료사 질문으로 아동 발화를 점진적으로 확장하는 기법
- 절차:
  1. 아동: "공" (한 낱말)
  2. 치료사: "공이 어떤 거야?" (질문)
  3. 아동: "빨간 공" (두 낱말)
  4. 치료사: "빨간 공이 어디에 있어?" (질문)
  5. 아동: "빨간 공이 위에 있어" (문장)
- 적용: MLU 확장이 목표인 아동. 학습지보다 대화 기법이나 대화 스크립트 카드에 반영.

### PMLU(평균음운길이) 및 PWP(단어단위 근접률)
```
PMLU = 전체 분절음 수 + 정조음 자음 수
       ─────────────────────────────
            전체 단어 수

PWP = 아동 PMLU / 목표(성인) PMLU

예: "사과" 목표=[ㅅ,ㅏ,ㄱ,ㅗ,ㅏ]=5분절음+2자음=7
    아동 산출 "다과"=[ㄷ,ㅏ,ㄱ,ㅗ,ㅏ]=5분절음+1정조음자음=6
    PMLU 아동=6, PMLU 목표=7, PWP=6/7=0.86
```
PWP가 1.0에 가까울수록 목표에 근접. 0.5 미만이면 심각한 음운장애 시사.

### 구개열 아동 조음 오류 분류
| 유형 | 원인 | 예시 | 중재 |
|------|------|------|------|
| 필연적 오류 | 구조적 한계 (연인두 폐쇄 불충분) | 비누출, 약한 압력 자음 | 수술/보철 먼저 |
| 보상적 오류 | 잘못된 조음 습관 | 성문파열음, 인두마찰음, 후비강마찰음 | 조음 치료 (위치 교정) |

⚠️ 필연적 오류는 학습지로 교정 불가 — 수술적 중재 후 조음치료 진행.

### CAS(아동기 말실행증) 치료 원칙
| 원칙 | 설명 |
|------|------|
| 다감각 자극 | 시각+청각+촉각 동시 제공 (입모양+소리+촉각 단서) |
| 리듬/억양 활용 | 멜로디, 강세 패턴으로 운동 계획 촉진 |
| 높은 반복량 | 세션당 100회+ 산출 시도 |
| 느린 속도 시작 | 정확성 확보 후 점진적 속도 증가 |
| 전이 계획 | 음절→단어→구→문장 체계적 전이 |

⚠️ CAS는 학습지 단독 부적절 — 직접 중재 보조로만 사용.

### 핵심어휘 접근법(Core Vocabulary Approach)
- 대상: 비일관적(inconsistent) 음운장애 — 같은 단어를 매번 다르게 발음
- 원리: 고빈도 단어 50개를 정확한 산출로 고정 → 일관성 확보
- 음운변동 접근법과 구별: 음운변동 접근법은 패턴 오류, 핵심어휘는 비일관적 오류
- 학습지: 고빈도 단어 카드 (엄마, 아빠, 물, 밥, 싫어, 좋아 등)

### 측음화(Lateral Lisp) 중재
- 정의: /ㅅ,ㅆ/ 산출 시 기류가 혀 측면으로 나가는 오류
- 중재 기법:
  1. 입술 모으기 (기류 중앙화)
  2. 턱 약간 내리기 (혀 위치 조정)
  3. /ㅌ/ → /ㅅ/ 연쇄 (치조 접촉 유도)
  4. 빨대 불기 (중앙 기류 연습)
- 학습지: /ㅅ/ 단어 연습 시 "입술 동그랗게" 시각 단서 포함

### 이중언어 아동 치료 원칙
- 두 언어 공통 오류패턴을 우선 치료 (일반화 효과 극대화)
- 예: L1(한국어)과 L2(영어) 모두에서 파열음화 → 파열음화 먼저 치료
- 한 언어에서만 나타나는 오류 = 언어 간 전이(transfer) 가능성 → 자연 소거 관찰
- 평가 시 두 언어 모두에서 실시 (한 언어 기준만으로 판단 금지)

### 미숙아(조산아) 교정연령

### 음운표상(Phonological Representation)
- 정의: 머릿속에 저장된 단어의 말소리 정보
- 음운표상이 불완전하면: 듣기에서도 변별 어려움 + 산출에서도 오류
- 평가: 청각적 변별 과제, 음운인식 과제
- 치료: 입력(input) 강화 → 표상 재구성 → 산출 연습

### 음절별 음운변동율 계산
```
음운변동율 = (음운변동 적용 횟수 / 음운변동 적용 가능 횟수) × 100

예: 파열음화 — 마찰음/파찰음 총 10회 출현, 그 중 6회 파열음화
    파열음화 변동율 = 6/10 × 100 = 60%
```
변동율 40% 이상 → 치료 목표로 선정.

### 음운 복잡성 계층 (Complexity Approach)
| 복잡성 | 설명 | 예시 |
|--------|------|------|
| 낮음 | 초기 습득 음소, 단순 음절 | CV 구조, /ㅁ,ㅂ/ |
| 중간 | 중기 습득 음소, CVC 구조 | /ㄷ,ㄴ,ㅈ/ |
| 높음 | 후기 습득 음소, 자음군 | /ㅅ,ㄹ/, 겹받침 |

복잡성 접근법: 복잡한 목표를 먼저 치료하면 단순한 음소에 자동 일반화.
전통적 접근법과 반대 방향 (쉬운 것 먼저 vs 어려운 것 먼저).

### 자극단어 선정 원칙
1. 아동이 자주 사용하는 고빈도 단어 우선
2. 촉진적 음성 문맥 고려 (목표 음소 산출이 쉬운 환경)
3. 기능적으로 의미 있는 단어 (아동 일상에서 사용)
4. 연음화/경음화 등 음운규칙으로 왜곡되지 않는 환경
5. 최소대립쌍 사용 시: 아동이 대립쌍 중 하나는 정확히 산출 가능해야 함

### 학령기 말소리장애 치료 특수성
| 고려사항 | 내용 |
|---------|------|
| 자의식 | 발음 오류에 대한 자의식 ↑ → 상위음운 접근법 적합 |
| 읽기 연계 | 음운인식 부족이 읽기 어려움과 동반 가능 |
| 잔존 오류 | /ㅅ/,/ㄹ/ 왜곡이 학령기까지 잔존 빈도 높음 |
| 연습 자율성 | 자가 모니터링/자가 교정 능력 활용 가능 |
| 사회적 영향 | 또래 관계에 발음이 미치는 영향 고려 |

### 확장지각접근법(Enhanced Perceptual Approach)
- 목표 음소의 음향적 특성을 강조하여 지각 능력 향상
- 절차:
  1. 청각적 자극 강화 (목표 음소를 과장하여 제시)
  2. 지각 훈련 (맞는 발음/틀린 발음 변별)
  3. 산출 연습 (지각 기반 자기 모니터링)
- 적용: 특히 /ㅅ/ 왜곡(측음화, 치간음화)에 효과적

### 마비말장애(Dysarthria) 치료 원칙
| 유형 | 원인 | 치료 초점 |
|------|------|----------|
| 이완형 | 하위운동뉴런 손상 | 근력 강화, 호흡 지지 |
| 경직형 | 상위운동뉴런 손상 | 이완, 속도 조절 |
| 운동실조형 | 소뇌 손상 | 협응, 리듬 훈련 |
| 운동저하형 | 기저핵 손상 (파킨슨) | 강도 증가 (LSVT) |
| 운동과다형 | 기저핵 손상 | 불수의 운동 조절 |
| 혼합형 | 다발성 | 가장 심한 요소 먼저 |

⚠️ 마비말장애는 학습지 단독 부적절 — 직접 중재 필요 (호흡/발성/공명/조음/운율).

### 보고서 작성 원칙 (현장실무)
| 항목 | 내용 |

| 분석 결과 | 오류 패턴, 강점/약점, 기능적 수준 |

## 부록 5: 임상 추론 체인 (기출 기반 — 에이전트가 "왜"를 설명할 수 있는 근거)

### 이중모음과 음절구조 분석
한국어 이중모음(ㅑ,ㅕ,ㅛ,ㅠ,ㅘ,ㅝ 등)은 반모음(G, glide)+단모음(V)으로 분석.
- C=자음, G=반모음(glide), V=모음
- 예: '병'=CGVC (ㅂ+y+ㅓ+ㅇ), '과'=GVC (w+ㅏ)
- 한국어 음절의 최대 구조: CGVC
- 음절을 이루는 최소 조건: 모음(V) 1개. 자음이나 반모음은 단독으로 음절 불가.

### ㄴ첨가 규칙 (표준발음법 제29항)
- 조건: 합성어/파생어에서 앞 형태소가 자음으로 끝나고 + 뒤 형태소가 /이,야,여,요,유/로 시작
- 결과: 후행 형태소 앞에 ㄴ 삽입
- 예: 꽃잎→[꼰닙], 색연필→[생년필], 솜이불→[솜니불]
- 조음평가 시 ㄴ첨가 환경을 알아야 아동의 실제 오류와 음운규칙 적용을 구분 가능

### 한국어 IPA 음성전사 핵심 규칙
| 한국어 자음 | IPA (어두) | IPA (어중 유성음 사이) |
|-----------|----------|-------------------|
| ㅂ(평음) | [p] | [b] |
| ㄷ(평음) | [t] | [d] |
| ㄱ(평음) | [k] | [g] |
| ㅈ(평음) | [tɕ] | [dʑ] |
| ㅃ(경음) | [p*] | [p*] |
| ㅍ(격음) | [pʰ] | [pʰ] |
| ㅋ(격음) | [kʰ] | [kʰ] |

핵심: 평음은 어중 유성음 사이에서 유성음화됨. 경음/격음은 위치 무관.

### 숫자세기 선별검사 해석 원리
- 숫자 1-10은 다양한 음소를 포함하여 빠른 선별 가능
- 해석: 아동의 오류 음소가 해당 연령의 습득 기준을 넘겼는지 확인
- 심화검사 기준: 습득 연령이 지난 오류가 관찰될 때
- 예: 4세에서 전방화(ㄱ→ㄷ) 잔존 → 3;6-4세 소거 기준 초과 → 심화검사 필요

### 오류패턴 빈도 분석 방법
1. 아동의 의도 발화와 실제 산출을 음소 단위로 대조
2. 각 오류를 음운변동 유형으로 분류 (파열음화, 전방화, 종성생략 등)
3. 각 변동의 출현 빈도를 세어 가장 빈번한 패턴 식별
4. 가장 빈번한 패턴 = 첫 번째 치료 목표 후보

추론 예시: "사→타, 자→다, 쳐→텨가 모두 관찰되면 → 전부 파열음화(마찰/파찰음→파열음) → 파열음화가 가장 빈번 → 파열음화 소거가 1순위 목표"

### 변별자질: 전방성(anterior)
| 자질 | 조음위치 | 해당 자음 |
|------|---------|----------|
| [+anterior] | 양순, 치조 | ㅂ,ㅃ,ㅍ,ㅁ,ㄷ,ㄸ,ㅌ,ㄴ,ㄹ,ㅅ,ㅆ |
| [-anterior] | 경구개, 연구개, 성문 | ㅈ,ㅉ,ㅊ,ㄱ,ㄲ,ㅋ,ㅇ,ㅎ |

판단 기준: 경구개치경(alveolo-palatal) 뒤쪽인가 앞쪽인가.

### 음절구조변동 vs 대치변동 감별
- 음절구조변동: 음절의 CV 구조 자체가 변함
  - 종성생략: CVC→CV (예: 가방→[다바]의 종성ㅇ탈락)
  - 음절탈락: 다음절→축소
  - 자음군 단순화: CVCC→CVC
- 대치변동: 음절 구조는 유지되고 음소만 바뀜
  - 예: 침대→[친대] (CVC+CV 구조 유지, ㅁ→ㄴ 대치만 발생)
- 핵심: 종성이 사라지면 음절구조변동, 다른 자음으로 바뀌면 대치변동

### 자극반응도(Stimulability) 평가 절차
1. 치료사가 목표 음소를 모델링 ("선생님을 잘 보면서 따라 해 봐요. [s:]")
2. 아동에게 따라하기 요청
3. 산출 성공 여부 판단
4. 수준별 확인: 음소 → 음절 → 단어 순으로 난이도 증가
- 자극반응도 있는 음소 → 치료 효과 빠름 → 우선 목표 후보
- 자극반응도 없는 음소 → 지각 훈련(ear training)부터 시작

### 조음치료 일반화 유형
| 유형 | 정의 | 예시 |
|------|------|------|
| 문맥 일반화 | 훈련하지 않은 음성 문맥으로 전이 | /사/에서 습득 → /소/로 전이 |
| 위치 일반화 | 다른 음절 위치로 전이 | 초성 /ㅅ/ 습득 → 종성 /ㅅ/ 전이 |
| 언어단위 일반화 | 작은 단위→큰 단위로 전이 | 음절(/조/) → 단어(/조개/) |
| 상황 일반화 | 치료실→일상 환경으로 전이 | 치료실 정조음 → 가정에서도 정조음 |
| 자질 일반화 | 같은 자질의 다른 음소로 전이 | /ㅅ/ 습득 → 같은 마찰음 /ㅆ/ 자동 습득 |

학습지 설계 시 일반화를 촉진하는 단어 선정이 중요:
- 문맥 일반화: 다양한 모음 환경의 단어 포함
- 자질 일반화: 목표 자질을 공유하는 여러 음소 단어 포함

### 최소대립자질쌍 판별법
1. 두 단어가 모음/종성 등 다른 요소는 모두 동일한지 확인
2. 다른 자음 사이에서 변별자질 차이가 딱 1개인지 확인
3. 자질 종류: 조음위치 / 조음방법 / 발성유형
- 예: 삼:담 → ㅅ(치조마찰평음) vs ㄷ(치조파열평음) = 조음방법만 1개 차이 ✅
- 예: 김:공 → 모음도 다르고 종성도 다름 = 최소대립 아님 ❌

### 치료 접근법 선택 추론 체인
```
[상황] 아동이 ㅅ→ㄷ, ㅈ→ㄷ, ㅊ→ㄷ 모두 보임
[분석] 세 가지 오류가 모두 "파열음화" 패턴
[추론] 개별 음소(ㅅ)가 아닌 패턴(파열음화)이 문제
       → ㅅ 하나만 치료해도 ㅈ,ㅊ에 자질 일반화 기대
[결론] 음운변동 접근법 선택
[학습지] ㅅ 목표 단어 세트 + ㅅ/ㄷ 최소대립쌍

[상황] 아동이 같은 단어를 매번 다르게 발음 (비일관적)
[분석] 음운변동 패턴이 일관적이지 않음
[추론] 음운변동 접근법은 일관적 패턴 전제 → 부적합
       핵심어휘 접근법: 고빈도 단어를 반복하여 일관성 확보
[결론] 핵심어휘 접근법 선택
[학습지] 고빈도 단어 50개 카드 (엄마, 아빠, 물, 밥 등)

[상황] 음운체계가 극도로 제한 (산출 가능 음소 3-4개)
[분석] 음운체계 확장이 시급
[추론] 최대대립자질접근법: 가장 다른 자질의 대립쌍으로
       체계 전체를 재구조화
[결론] 최대대립자질접근법 선택
[학습지] 최대 대립 단어쌍 (예: 말 vs 칼 — 양순비음 vs 연구개격음파열)
```

### 접근법 선택 가이드 (보완)
| 아동 프로필 | 추천 접근법 | 학습지 |
|------------|-----------|--------|
| 소수 오류, 왜곡 | Van Riper (전통적) | 음소 위치별 단어 카드 |
| 다수 음운변동, 일관적 | 음운변동/주기법 (Hodson) | 변동별 단어 세트 |
| 의미 변별 어려움 | 최소대립쌍 | 1개 자질 차이 단어쌍 |
| 비일관적 오류 | 핵심어휘 (Core Vocabulary) | 고빈도 단어 50개 |
| 음운체계 극도 제한 | 최대대립자질접근법 | 최대 대립 단어쌍 |
| 실생활 일반화 필요 | 의사소통중심 | 대화/역할놀이 카드 |
| 음운인식 부족 (학령기) | 상위음운접근법 | 음소 조작 퀴즈 |

## 부록 6: 2025년 기출 기반 추론 보강

### 음운 치료 접근법 선택 의사결정 트리
```
1. 오류가 비일관적인가? (같은 단어를 매번 다르게 발음)
   → YES: 핵심어휘 접근법 (Core Vocabulary)
   → NO: 다음 단계로

2. 오류가 일관적 패턴인가?
   a. PCC < 50% (심도) → 주기법 (Cycles)
   b. PCC 50-65% + 다수 변동 → 음운변동 접근법
   c. 특정 음소 대립만 문제 → 최소대립쌍
   d. 소수 음소 왜곡 → 전통적 접근법 (Van Riper)

3. 학령기 + 음운인식 약함? → 상위음운 접근법 병행

4. 중도-심도 + 효율성 극대화? → 복잡성 접근법
```

### 대립쌍 치료 유형 비교
| 유형 | 자질 차이 | 적용 | 예시 |
|------|----------|------|------|
| 최소대립쌍 | 1개 자질 | 특정 음소 변별 | 달-살 (방법만 다름) |
| 최대대립쌍 | 2개+ 자질 | 음운 체계 재구조화 | 달-할 (위치+방법) |
| 빈자리쌍(empty set) | 두 음소 모두 미산출 | 심도 음운장애 | - |
| 다중대립쌍 | 1:다수 | 동음어적 충돌 해결 | 달-살-말-발 |

### 음운변동 치료 목표 우선순위 결정 논리
```
1순위: 비발달적 변동 → 연령 무관 즉시 치료 (후방화, 초성 전체 생략)
2순위: 소거 연령이 지난 발달적 변동 (예: 5세의 파열음화, 소거 3-4세)
3순위: 같은 조건이면 출현 빈도가 높은 변동 우선 (말명료도 영향)
4순위: 자극반응도가 있는 변동 우선 (치료 효과 빠름)
참고: 소거 연령 미도달 변동 = 정상 발달 → 대기
```

### 복잡성 접근법(Complexity) vs 발달 순서 접근법
| 원칙 | 발달 순서 | 복잡성 접근법 |
|------|----------|------------|
| 목표 | 쉬운(일찍 습득) 음소부터 | 어려운(늦게 습득) 음소부터 |
| 근거 | 발달 경로를 따름 | 복잡한 목표가 단순한 것에 일반화 |
| 대상 | 경도~중등도 | 중도~심도 음운장애 |
| 장점 | 초기 성공 경험 보장 | 치료 효율성 높음 |
| 연구 | 전통적 관점 | Gierut(2001): 복잡→단순 일반화 입증 |

### 전통적 접근법(Van Riper) 절차와 근거
- 대상: 소수 음소 오류, 왜곡 오류 중심
- 절차:
  1. 변별 훈련(ear training) — 목표음 vs 오류음 구분
  2. 산출 훈련 — 감각운동적 접근으로 조음 위치/방법 교정
  3. 일반화 위계 — 음소 단독 → 음절 → 단어 → 구 → 문장 → 대화
- 근거: 운동학습이론 — 정확한 감각운동 패턴을 반복 연습하여 자동화
- 음운변동 접근법과 차이:
  전통적 = "이 음소를 어떻게 정확히 만들까" (운동적 관점)
  음운변동 = "이 패턴은 왜 나타나며 어떻게 바꿀까" (언어학적 관점)

### 환경중심 기법 간 구분
| 기법 | 절차 | 핵심 차이 |
|------|------|----------|
| 시간지연 | 상황 설정 후 침묵+기대 표정으로 대기 | 가장 자발적 유도 |
| 요구-모델 | "뭐 원해?" 질문 → 무반응 시 "사과라고 해봐" | 질문+모델 제공 |
| 우발교수 | 아동이 먼저 관심 보일 때 기회 잡아 교수 | 아동 주도 |

세 기법 모두 산출을 유도하지만, 시간지연이 가장 자연스러운 자발화 유도.

### 수직적 vs 수평적 구조화
- 수직적: 질문 연쇄로 발화를 "한 층씩" 쌓아 올림 (다수 차례교대)
  "공" → "어디?" → "위" → "뭐가 위에?" → "빨간 공이 위에 있어"
- 수평적: 아동이 여러 차례에 걸쳐 말한 요소를 한 문장으로 통합 시범
  아동: "공" ... "위" ... "빨간" → 치료사: "빨간 공이 위에 있네"
- 수직적 = MLU 확장 과정 기법, 수평적 = MLU 확장 결과 시범

### 비발달적 오류 → 감별진단 연결
| 비발달적 오류 | 의심 진단 |
|-------------|----------|
| 후방화 + 모음 오류 | 청각장애 (고주파 청력 손실) |
| 성문파열음 대치 | 구개열/연인두 기능부전 |
| 비일관적 + 비발달적 | CAS (아동기 말실행증) |
| 초성 전체 생략 | 심각한 음운장애/운동 계획 문제 |
| 비음의 구강음화 | 연인두 폐쇄부전 |

비발달적 오류 발견 시 "왜 이런 오류가 나타나는가?" 원인 추론 필수.

### CAS에 음운적 접근이 비효과적인 이유
- CAS 문제 = 운동 계획(motor planning), 음운 지식 아님
- 최소대립쌍: "소리 차이를 인식"시킴 → CAS 아동은 차이를 이미 알지만 실행 못함
- 음운변동 접근법: "패턴 변경" → CAS는 일관된 패턴 자체가 없음
- CAS에는 운동학습 원칙 기반 접근 필수: 집중 연습, 다감각 피드백, 체계적 전이

### 후방화(backing)의 임상적 중요성
- 정의: 치조음→연구개음 (ㄷ→ㄱ, "달"→"갈")
- 비발달적: 어떤 연령에서도 정상 아동의 10% 미만
- 발견 시: 즉시 치료 + CAS/청각장애 감별 필수

### 상위음운접근법(Metaphon) 2단계
Phase 1 — 개념 단계: 말소리 속성 인식 ("긴 소리 vs 짧은 소리", "앞 소리 vs 뒤 소리")
Phase 2 — 의사소통 단계: 최소대립쌍으로 의미 대조 + 메타언어적 피드백
→ "어떻게 만들까"가 아니라 "왜 이 소리가 다른 소리와 다른가"를 이해시킴

### 30개월 전후 추가 평가 체크리스트
- 표현어휘 50개 이상? → K M-B CDI 또는 SELSI
- 2어 조합 출현? → 자발화 샘플 분석
- 상징놀이 수준? → 사물 대치 출현 여부
- 공동주의(IJA) → ASD 선별
- 청력 선별 → 반드시 실시
→ 표현어휘 50 미만 + 2어 조합 없음 = late talker, 직접 중재 고려

### K-CSBS DP 3영역의 임상적 의의
- 사회적(Social): 시선, 정서, 의사소통 → ASD 조기 선별 핵심
  IJA(주도적 공동주의) 결핍 = 12개월 전후 ASD 최강 예측인자
- 발화(Speech): 자음/모음 레퍼토리, 음절구조 → 말 산출 기반
  canonical babbling 미출현 = 운동말장애 고려
- 상징(Symbolic): 사물 이해/사용 → 인지-언어 연계
  상징놀이 지연 + 언어 지연 = 전반적 발달지연 가능성

## 부록 7: Hard 벤치마크 보강 — 임상 문제해결

### 7.6 선택적 함묵증(Selective Mutism)
- 정의: 다른 상황에서는 말하지만 특정 상황(주로 학교)에서 지속적으로 말하지 않음
- 핵심: 언어장애가 아닌 불안장애(DSM-5 불안장애 분류)
- 언어재활사 역할: 제한적. 낮은 부담의 의사소통 기회 제공, 점진적 노출 보조
- 주 치료: 심리치료(CBT, 점진적 노출), 소아정신과/심리 전문가 의뢰 필수
- 감별: "말을 못하는 것"(can't)이 아닌 "말을 안 하는 것"(won't) — 1:1 안전 환경에서 정상 발화 확인
- 학습지: 단독 사용 부적절. 심리치료 병행 시 보조적 매체로만 활용 가능

### 7.7 치료 거부 및 행동 관리
아동이 울거나 치료를 거부할 때:
1. 요구 수준 낮추기: 산출 요구 제거 → 관찰/놀이만
2. 아동 선호 활동 따르기: 아동 주도 놀이로 라포 형성
3. 강화 체계: 토큰 보드, 선호 활동을 마지막에 배치
4. 점진적 노출: 치료실 환경 → 도구 → 활동 단계적 도입
5. 감각 점검: 감각과민이 원인일 수 있음 → OT 협업
6. 부모 동석: 분리불안이 원인이면 부모 함께 참여
7. 세션 구조화: 시각 스케줄로 예측가능성 제공
⚠️ 3회 이상 심한 거부 지속 → 행동 전문가/소아정신과 협업 고려

### 7.8 부모 가정연습 비협조 대응
- 원칙: 비난하지 않고 현실적 대안 제시
- 전략:
  1. 간소화: 하루 5분, 일상 루틴(식사/목욕/이동) 내 삽입
  2. 시범: 세션 중 부모에게 직접 시범 보이기
  3. 이유 설명: "가정연습이 치료 효과를 2-3배 높입니다"
  4. 영상 자료: 짧은 시범 영상 촬영하여 제공
  5. 현실적 목표: "매일 10분"이 아닌 "저녁 식사 중 2-3번"
  6. 성공 공유: 소소한 진전도 부모에게 구체적으로 보고
  7. 장벽 파악: 시간 부족? 방법 모름? 효과 의심? → 각각 다른 대응

### 7.9 복합 동반장애 시 치료 우선순위
| 우선순위 | 영역 | 이유 |
|---------|------|------|
| 1순위 | 참여 방해 요인 제거 | 감각과민, 행동문제, 불안 → OT/심리 협업 |
| 2순위 | 기능적 의사소통 확보 | 의사소통 수단(구어/AAC) 확립이 모든 학습의 기반 |
| 3순위 | 언어 이해/표현 확장 | 어휘, 문법, 화용 |
| 4순위 | 말명료도(조음) | 의사소통 기능이 확보된 후 명료도 향상 |
원칙: "의사소통 기능 > 말 형태" — 기능이 없으면 형태 교정은 무의미

### 음운변동 치료 중 새로운 변동 출현 현상
- 현상: 파열음화를 치료하면 전방화가 새로 관찰되는 등
- 원인:
  1. 가림 효과(masking): 파열음화가 전방화를 가리고 있었음
  2. 체계 재구조화: 음운 체계가 변화하며 과도기적 새 패턴 출현
  3. 이미 존재: 기저에 있었으나 파열음화로 인해 보이지 않았음
- 대응: 정상적 치료 과정. 새 변동도 치료 목표에 추가
- 주의: 비발달적 새 변동 출현 시 → CAS 재감별 필요

### 파열음화 내 발성유형 보존 분석
- ㅅ→ㄷ, ㅈ→ㄷ: 평음→평음 (조음방법만 변화)
- ㅊ→ㅌ: 격음→격음 (기식성 유지)
- ㅉ→ㄸ: 경음→경음 (긴장성 유지)
- 치료 시사: 하나의 발성유형(예: 평음 ㅅ)만 치료해도 같은 유형 내 일반화 기대

### 동일 조음위치 삼중체(평음/경음/격음) 치료 순서
같은 위치의 세 음소(예: ㅈ/ㅉ/ㅊ)가 모두 오류일 때:
1. 평음(ㅈ)을 먼저 — 한국어 기본(default) 발성유형
2. 평음 습득 후 경음(ㅉ)과 격음(ㅊ)으로 자질 일반화 기대
3. 일반화 불충분 시: 격음(ㅊ) 다음 (기식 자질 명확하여 지각 쉬움)
4. 경음(ㅉ)은 마지막 (자연 발화에서 빈도 낮음)
예외: 자극반응도가 경음이나 격음에서 더 높으면 그것부터 시작

### 반복적 질문 행동 감별
"이거 뭐야?" 등을 과도하게 반복하는 경우:
| 원인 | 특징 | 대응 |
|------|------|------|
| 정상 발달 | 2-3세, 다양한 대상에 질문, 답에 반응 | 풍부한 응답 제공 |
| 어휘 확장 시도 | 새로운 사물에 집중, 답을 기억하고 사용 | 정상, 격려 |
| ASD 반향적 | 동일 질문 반복, 답에 무관심, 기능적이지 않음 | 기능적 의사소통으로 전환 |
| 불안/루틴 | 상황 변화 시 증가, 예측가능성 추구 | 시각 스케줄, 사전 예고 |
→ 4세 이후 지속적 동일 질문 반복 + 답에 무관심 = ASD 화용 평가 필요

### 표준화 검사 결과 불일치 시 해석 원칙
1. 부모보고(SELSI, K M-B CDI) vs 직접평가(PRES, REVT) 불일치:
   - 부모보고가 높으면: 제스처/상황적 이해를 언어로 해석했을 가능성
   - 직접평가가 높으면: 낯선 환경에서 수행 저하 (불안/비협조)
   - 원칙: 직접평가에 더 비중, 단 아동 협조도 고려
2. 두 직접평가 도구 간 불일치:
   - 검사 내용/형식 차이 확인 (어휘 vs 문법 비중)
   - 낮은 점수에 주목: 약점 영역 시사
3. 하나만으로 판단하지 않고 2개+ 검사 + 자발화 분석 통합 해석

### 이야기 문법 요소 습득 순서
| 순서 | 요소 | 설명 |
|------|------|------|
| 1 | 배경(setting) | 인물/장소 — 가장 먼저 출현 |
| 2 | 계기사건(initiating event) | 무슨 일이 생겼는지 |
| 3 | 시도(attempt) | 해결 행동 |
| 4 | 결과(consequence) | 시도의 결과 |
| 5 | 내적 반응(internal response) | 감정/생각 — 마음이론 필요 |
| 6 | 반응(reaction) | 최종 감정/교훈 — 가장 늦음 |
내적 반응과 반응은 마음이론(ToM) 발달과 관련 → 7세+ 본격 출현

### 최소대립쌍 사용 전제 조건
1. 두 단어 모두 아동의 수용어휘에 포함되어야 함
2. 아동이 그림을 보고 두 단어를 변별할 수 있어야 함
3. 전제 미충족 시 → 어휘 교수 먼저 실시
4. 대안: 아동이 아는 단어 중에서 대립쌍 재선정

### 주기법에서 주기 간 퇴행 현상
- 이전 목표로 돌아왔을 때 퇴행은 정상적 현상
- 이유: 주기법은 완전 습득 전 전환 → 간섭 발생 가능
- Hodson의 원칙: 각 주기마다 점진적 향상 기대 (100% 습득 아님)
- 경고 신호: 3주기 이상 반복해도 동일 수준 → 접근법 재검토

### 복잡성 접근법 일반화 실패 시 점검
1. 대상 적합성: 중도-심도 음운장애인가? (경도/조음장애에는 부적합)
2. CAS 감별: 비일관적 오류라면 복잡성 접근법 부적합
3. 자질 관련성: 치료 목표와 일반화 기대 음소가 자질을 공유하는가?
4. 치료 강도: 충분한 반복량?
5. 기간: 최소 6-8주 이상 지속했는가?
6. 대안: 일반화 안 되면 발달 순서 접근법으로 전환 고려

### 치료 정체 시 접근 전환 의사결정
- 정체 기준: 동일 접근법 3개월+ 진행, PCC 변화 < 5%p
- 점검:
  1. 진단 재확인: CAS/청각장애/지적장애 감별 재실시
  2. 치료 충실도: 접근법 절차를 정확히 따랐는가?
  3. 강도: 주 2회 미만이면 효과 제한적
  4. 가정연습: 부모 협조 확인
  5. 동반요인: 주의력, 감각, 정서 배제
- 전환 시: 현 접근법과 다른 원리의 접근법 선택
- 6개월+ 전면 정체 → 다학제 팀 회의

### 1.6 주요 음운변동별 최소대립쌍 단어 뱅크

**전방화 (ㄱ→ㄷ) 최소대립쌍:**
