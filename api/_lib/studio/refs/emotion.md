# 감정 영역 전문가 레퍼런스 (Emotion Domain Specialist Reference)

> 이 문서는 ASD 및 발달장애 아동의 감정 인식/추론/표현 치료 자료를 생성하는 AI 시스템의 전문가 지식 베이스입니다.
> 학습지 이미지 프롬프트 생성 시 반드시 `asd-worksheet-prompt` SKILL과 함께 참조하십시오.

## 1. 감정 발달 단계 (Emotion Development Levels)

감정 능력은 계층적으로 발달합니다. 아래 단계를 건너뛸 수 없으며, 반드시 현재 수준을 정확히 파악한 후 활동을 설계해야 합니다.

| Level | 명칭 | 설명 | 핵심 능력 | 대표 활동 |
|-------|------|------|-----------|-----------|
| 0 | 얼굴 주의 (face attention) | 사람 얼굴에 주의를 기울이고 시선을 유지 | 공동주의, 얼굴 지향 | 얼굴 주의 유도 활동 (Section 8-J) |
| 1 | 표정 매칭 (같은 표정 찾기) | 같은 표정끼리 짝짓기 | 시각적 변별 | 같은것찾기 |
| 1-T | 표현 양식 전이 (modality transfer) | 실물 사진과 일러스트 간 같은 표정 연결 | 추상화 전이 | 사진↔일러스트 매칭 (Section 8-G) |
| 2 | 감정 명명 (표정→감정 단어) | 표정을 보고 감정 이름 말하기 | 표정-어휘 연결 | 짝맞추기 |
| 3 | 상황 기반 추론 (상황→감정) | 상황 그림만 보고 감정 추론 | 인과추론 | 감정추론 학습지 |
| 3-G | 일반화 (generalization) | 매체/맥락을 넘어 실생활에서 감정 인식 적용 | 탈맥락화, 일반화 | 일반화 프로토콜 (Section 8-H) |
| 4 | 복합 상황 추론 | 복잡한 상황에서 감정 추론 | 맥락 통합 | 감정추론 학습지 (고급) |
| 4+ | 혼합 감정 인식 (mixed emotions) | 한 상황에서 두 가지 감정을 동시에 느끼는 것을 이해 | 감정 병존 이해 | 혼합 감정 학습지 |
| 5 | 관점 취하기 (perspective-taking) | 같은 상황에서 서로 다른 감정 추론 | 마음이론 | 순서맞추기 + 생각풍선 |

### Level 판별 질문

```
Step 0: 선수 기술 확인 (얼굴 주의)
  Q0. "사람 얼굴을 주의깊게 바라보는가? (2초 이상 시선 유지)"
      → NO → Level 0 (얼굴 주의 유도부터 — Section 8-J)
      → YES ↓

Step 1: 감정 인식 수준 파악
  Q1. "기본 감정(행복/슬픔/화남) 표정을 시각적으로 구별할 수 있는가?"
      → NO → Level 1 (표정 매칭부터)
      → YES ↓
  Q1-T. "실물 사진 표정과 일러스트 표정을 모두 인식하는가?"
      → 사진만 OK / 일러스트 NO (또는 반대) → Level 1-T (표현 양식 전이 — Section 8-G)
      → 둘 다 OK ↓
  Q2. "표정을 보고 감정 단어를 말할 수 있는가?"
      → NO → Level 2 (감정 명명)
      → YES ↓
  Q3. "상황 그림만 보고(표정 없이) 감정을 추론할 수 있는가?"
      → NO → Level 3 (상황 기반 추론)
      → YES ↓
  Q3-G. "학습지/영상이 아닌 실제 상황에서도 감정을 인식하는가?"
      → 학습지/영상만 OK / 실생활 NO → Level 3-G (일반화 — Section 8-H)
      → 둘 다 OK ↓
  Q4. "복합 상황(기대와 현실의 불일치 등)에서 감정을 추론할 수 있는가?"
      → NO → Level 4 (복합 상황 추론)
      → YES ↓
  Q4-1. "한 상황에서 두 가지 감정을 동시에 느낄 수 있음을 이해하는가?"
      → NO → Level 4+ (혼합 감정 인식)
      → YES ↓
  Q5. "같은 상황에서 다른 사람이 다른 감정을 느낄 수 있음을 이해하는가?"
      → NO → Level 5 (관점 취하기)
      → YES → 감정 표현/조절 영역으로 확장

Step 2: 자기 감정 인식 수준 파악 (Step 1과 별도로 평가)
  Q-Self1. "자기 감정 상태를 묻는 질문에 감정 단어로 답할 수 있는가?"
      → NO → 자기 감정 인식 활동 필요 (Section 8-E)
      → YES ↓
  Q-Self2. "자기 감정의 강도를 구별할 수 있는가? (조금 화남 vs 많이 화남)"
      → NO → 감정 강도 활동 필요 (Section 2 강도 어휘 계열 + Section 8-D)
      → YES ↓
  Q-Self3. "자기 감정의 강도에 비례하는 표현을 할 수 있는가? (살짝 짜증인데 대폭발 안 함)"
      → NO → 감정 강도 조절 활동 필요 (Section 8-I)
      → YES → 자기 감정 인식 및 조절 양호

Step 3: 감정 전이/일반화 수준 파악 (Step 1과 별도로 평가)
  Q-Trans1. "사진과 일러스트 모두에서 동일 감정을 인식하는가?"
      → NO → 표현 양식 전이 필요 (Section 8-G)
      → YES ↓
  Q-Trans2. "학습 맥락(학습지/영상)과 실생활 모두에서 감정을 인식하는가?"
      → NO → 실생활 일반화 필요 (Section 8-H)
      → YES → 전이/일반화 양호
```

### 중요: Level 간 불일치 패턴

실무에서 아동의 능력이 Level 간에 균일하지 않은 경우가 흔합니다.

| 패턴 | 설명 | 대응 전략 |
|------|------|-----------|
| **표정 OK / 상황 NO** | 표정 읽기(L2)는 되지만 상황 추론(L3)이 안 됨 | 브릿징 활동: 상황+표정 함께 → 표정 점차 페이딩 |
| **인식 OK / 어휘 NO** | 감정을 구별하지만 단어를 모름 | 짝맞추기로 감정 어휘 집중 학습 (언어 영역 연계) |
| **어휘 OK / 표정(인식) NO** | 감정 단어는 알지만 표정을 보고 해당 감정과 연결 못함 | Level 1 표정 매칭부터 시작. 어휘 지식을 앵커로 활용 (단어를 먼저 보여주고 → 해당 표정 찾기). 실물 사진 스타일(Style C) 고려 |
| **사진 OK / 일러스트 NO** | 실물 사진 표정은 맞추지만 일러스트/그림 표정 인식 못함 | Level 1-T: 추상화 전이 활동 (Section 8-G). 사진→과장된 일러스트→단순 일러스트 3단계 |
| **일러스트 OK / 사진 NO** | 일러스트 표정은 맞추지만 실물 사진 표정 인식 못함 | 역방향 전이: 일러스트 옆에 대응 사진 제시 → 점진적 페이딩 |
| **매체 속 OK / 실생활 NO** | 영상/학습지에서 감정 인식은 하지만 실제 상황에서 적용 못함 | Level 3-G: 일반화 프로토콜 (Section 8-H). 매체→역할극→실생활 단계적 전이 |
| **인식 OK / 표현 NO** | 타인 감정은 읽지만 자신의 감정 표현 못함 | 표현 활동: 빈 얼굴 그리기, 생각풍선 채우기 |
| **타인 감정 OK / 자기 감정 NO** | 타인 감정은 정확히 맞추지만 자기 감정 인식·표현 못함 (alexithymia 경향) | 자기 감정 인식 활동 (Section 8-E): 신체감각-감정 연결, 감정 체크인 카드, "나라면?" 활동 |
| **추론 OK / 조절 NO** | 감정은 알지만 행동 조절이 안 됨 | 조절 전략 학습지 (시각적 스케일, 대처 카드) |
| **인식 OK / 강도 조절 NO** | 감정 인식·강도 변별은 하지만 표현 강도를 조절하지 못함 (살짝 짜증인데 대폭발) | 감정 강도 조절 활동 (Section 8-I): 감정-표현 매칭 스케일, "딱 맞는 크기" 연습 |
| **단일 감정 OK / 혼합 감정 NO** | 단일 감정 추론은 하지만 두 감정이 공존하는 상황을 이해 못함 | Level 4+ 혼합 감정 활동 (Section 8-F) |
| **감정 종류 OK / 감정 강도 NO** | 감정의 종류는 구별하지만 같은 감정의 강도 차이를 모름 | 감정 강도 어휘 계열 학습 (Section 2) + 감정 온도계 (Section 8-D) |
| **얼굴 비주의 (face inattention)** | 사람 얼굴 자체에 관심이 없어서 표정을 보지 않음 | Level 0: 얼굴 주의 유도부터 (Section 8-J). 감정 인식 이전 단계 |

## 2. 감정 어휘 발달 단계 (Emotion Vocabulary Progression)

### 단계별 목표 감정

| 단계 | 기능연령 | 감정 어휘 | 비고 |
|------|----------|-----------|------|
| 기본 | 3-4세 | 행복/기쁨, 슬픔, 화남/분노 | 최대 대비되는 3개 감정 |
| 중급 | 5-6세 | 놀람, 두려움/무서움, 부끄러움 | 기본 3개 + 3개 추가 |
| 고급 | 7-8세 | 실망, 걱정, 외로움, 질투 | 미묘한 부정적 감정 분화 |
| 복합 | 9세+ | 당혹, 후회, 감사, 자부심 | 자기성찰적 감정, 사회적 감정 |

### 감정 강도 어휘 계열 (Emotion Intensity Gradients)

같은 감정이라도 강도에 따라 다른 단어를 사용합니다. ASD 아동이 "다 같다"고 통칭하는 경우, 강도별 어휘를 시각적 크기/색상 차이와 함께 학습합니다.

| 감정 범주 | 약함 (Level 1-2) | 보통 (Level 3) | 강함 (Level 4-5) | 시각적 크기 비유 |
|-----------|-------------------|----------------|-------------------|------------------|
| **화남 계열** | 짜증남 / 기분 나쁨 | 화남 | 분노 / 폭발할 것 같음 | 작은 불꽃 → 큰 불꽃 → 화산 폭발 |
| **슬픔 계열** | 기분이 가라앉음 | 슬픔 | 매우 슬픔 / 비통함 | 작은 빗방울 → 비 → 폭우 |
| **두려움 계열** | 약간 불안함 | 무서움 | 공포 / 겁에 질림 | 작은 그림자 → 어두운 방 → 폭풍우 |
| **기쁨 계열** | 괜찮음 / 기분 좋음 | 행복함 | 매우 기쁨 / 신남 / 환희 | 작은 미소 → 큰 미소 → 점프하며 만세 |
| **걱정 계열** | 약간 신경 쓰임 | 걱정됨 | 매우 걱정됨 / 불안함 | 작은 물음표 → 여러 물음표 → 머리 위 소용돌이 |
| **부끄러움 계열** | 약간 쑥스러움 | 부끄러움 | 매우 부끄러움 / 수치심 | 볼 약간 분홍 → 볼 빨개짐 → 얼굴 전체 빨개지며 땀 |
| **놀람 계열** | 약간 의외 | 놀람 | 매우 깜짝 놀람 / 충격 | 작은 느낌표 → 큰 느낌표 → 별표+느낌표 폭발 |

#### 강도 변별 학습 프로토콜

```
1. 같은 감정의 강도 3단계를 시각적으로 제시
   → 작은/중간/큰 표정 이미지 (표정 과장도를 3단계로 조절)
   → 배경 색상 농도로 강도 표현 (연한 빨강 → 진한 빨강)
   → 캐릭터 몸짓 강도 차이 (살짝 찡그림 → 주먹 쥠 → 발 구르며 소리)

2. 시각적 크기 비유 활용 (위 표 참조)
   → "짜증은 작은 불꽃, 화남은 큰 불꽃, 분노는 화산 폭발"
   → 비유 이미지를 감정 표정 옆에 병렬 배치

3. 강도 어휘 짝맞추기 학습지
   → 왼쪽: 3단계 강도별 표정 이미지 (약/보통/강)
   → 오른쪽: 강도별 감정 단어 ("짜증나요" / "화났어요" / "너무너무 화나요")
   → 아동이 선으로 연결

4. 상황별 강도 판단 학습지
   → 상황 그림 제시 + "이 친구의 화남은 몇 단계일까?"
   → 감정 온도계(Section 8-D)와 연계
   → 예: 연필을 떨어뜨림(1단계) vs 장난감을 빼앗김(3단계) vs 아끼는 것이 부서짐(5단계)

5. 강도-표현 비례 학습 (강도 조절 연계 — Section 8-I)
   → 상황의 강도에 "맞는" 표현 크기를 고르기
   → 예: 작은 불편(연필 떨어뜨림) → 한숨 (O) vs 소리 지르기 (X)
   → "딱 맞는 크기의 반응" 개념 도입

6. 정답률 80% 이상 시 다음 감정 범주로 이동
```

### 혼동하기 쉬운 감정 쌍 (Confusable Pairs)

ASD 아동이 특히 혼동하는 감정 쌍입니다. 이 쌍들은 집중 변별 연습이 필요합니다.

| 혼동 쌍 | 핵심 시각 차이 | 핵심 상황 차이 | 핵심 신체감각 차이 |
|---------|---------------|---------------|-------------------|
| 슬픔 vs 화남 | 슬픔: 입꼬리↓, 눈물 / 화남: 눈썹 V, 주먹 | 슬픔: 잃어버림 / 화남: 빼앗김 | 슬픔: 가슴이 무겁고 축 처짐 / 화남: 얼굴 뜨겁고 주먹 꽉 |
| 슬픔 vs 실망 | 슬픔: 눈물, 고개 숙임 / 실망: 한숨, 어깨 축 처짐 | 슬픔: 상실 / 실망: 기대와 다름 | 슬픔: 눈물이 나옴 / 실망: 한숨이 나오고 힘이 빠짐 |
| 두려움 vs 놀람 | 두려움: 몸 뒤로, 떨림선 / 놀람: 눈 크게, 손 올림 | 두려움: 위협 지속 / 놀람: 갑작스러움 | 두려움: 계속 떨리고 도망가고 싶음 / 놀람: 잠깐 멈추고 심장 뜀 |
| 걱정 vs 두려움 | 걱정: 생각풍선, 손 만지작 / 두려움: 몸 떨림, 뒤로 물러남 | 걱정: 미래에 대한 불안 / 두려움: 현재 위협 | 걱정: 배가 살살 아프고 손이 만지작 / 두려움: 심장이 쿵쿵, 몸이 굳음 |
| 걱정 vs 불안 | 거의 동일 (동의어 수준) | 걱정: 구체적 대상 / 불안: 막연함 → 학습지에서는 구별하지 않음 | - |
| 부끄러움 vs 당혹 | 부끄러움: 고개 숙임, 손 뒤로 / 당혹: 얼굴 붉어짐, 당황한 눈 | 부끄러움: 자기 평가 / 당혹: 예상 못한 상황 | 부끄러움: 고개가 저절로 숙여지고 숨고 싶음 / 당혹: 얼굴이 화끈, 어찌할 바 모름 |
| 짜증 vs 화남 (강도 혼동) | 짜증: 입 삐죽, 한숨 / 화남: 눈썹 V, 주먹, 발 구르기 | 짜증: 가벼운 불편 / 화남: 명확한 원인에 의한 분노 | 짜증: 약간 불편, 한숨 / 화남: 얼굴 뜨겁, 주먹 꽉, 소리 지르고 싶음 |
| **부끄러움 vs 두려움** | 부끄러움: 볼 빨개짐, 고개 숙임, 눈 피함 / 두려움: 눈 크게 뜸, 몸 뒤로, 떨림 | 부끄러움: 타인의 시선/평가 / 두려움: 위험/위협 | **공통: 몸이 움츠러듦.** 차이: 부끄러움은 "얼굴이 뜨거워지고 숨고 싶음" / 두려움은 "심장이 쿵쿵, 도망가고 싶음" |
| **부끄러움 vs 불편함** | 부끄러움: 볼 빨개짐, 시선 회피 / 불편함: 몸 비틀기, 표정 일그러짐 | 부끄러움: 자기 노출 / 불편함: 외부 자극 | 부끄러움: 얼굴 뜨거움 / 불편함: 몸 전체가 찝찝함 |

#### 유사 신체반응을 가진 다른 감정 구분 프로토콜

아동이 "둘 다 몸이 움츠러든다"는 이유로 부끄러움과 두려움을 혼동하는 등, 유사한 신체반응을 보이는 다른 감정들을 구분하기 위한 전략입니다.

```
유사 신체반응 감정 그룹:

그룹 A: "몸이 움츠러드는" 감정들
  - 부끄러움: 움츠러듦 + 얼굴 뜨거움 + 시선 회피
  - 두려움: 움츠러듦 + 심장 빨리 뜀 + 도망가고 싶음
  - 불편함: 움츠러듦 + 몸 비틀기 + 자리 바꾸고 싶음

그룹 B: "가슴이 답답한" 감정들
  - 슬픔: 답답함 + 눈물 + 힘이 빠짐
  - 걱정: 답답함 + 배 아픔 + 계속 생각남
  - 외로움: 답답함 + 텅 빈 느낌 + 누군가 옆에 있으면 좋겠음

그룹 C: "심장이 빨리 뛰는" 감정들
  - 두려움: 심장 빠름 + 도망가고 싶음 + 위험
  - 놀람: 심장 빠름 + 멈춤 + 갑자기
  - 신남/기대: 심장 빠름 + 뛰고 싶음 + 좋은 일

변별 학습 프로토콜:

Step 1: 같은 신체반응 확인
  → "부끄러울 때도, 무서울 때도 몸이 움츠러들어요. 같은 느낌이지?"
  → 공통점을 먼저 인정 ("맞아, 둘 다 움츠러들어")

Step 2: 다른 신체 단서 찾기 (차별 단서)
  → "그런데 다른 것도 있어. 부끄러울 때는 얼굴이 뜨거워져. 무서울 때는 심장이 쿵쿵 뛰어."
  → 시각적 카드: 움츠러듦(공통) + 추가 단서(차별)를 함께 보여줌

Step 3: 맥락 단서로 최종 판단
  → "어디서 움츠러들었어? 사람들이 쳐다봐서? → 부끄러움! 무서운 게 있어서? → 두려움!"
  → 상황 그림과 함께 "이 친구는 왜 움츠러들었을까?" 질문

학습지 형식:
  → 3분할: [공통 신체반응 이미지] + [추가 단서 A vs B] + [상황 힌트]
  → 아동이 추가 단서와 상황을 보고 어떤 감정인지 판단
  → Template H 사용
```

### 과잉 일반화 대응 프로토콜

아동이 모든 부정적 감정을 하나로 통칭하는 경우 (예: 다 "슬프다"라고 함):

```
1. 혼동하는 감정 쌍을 정확히 식별
   → "화남과 슬픔을 구별하지 못함"

2. 해당 쌍만 집중 변별 학습지 설계
   → 같은것찾기: 화난 표정 3개 vs 슬픈 표정 3개 중 같은 것 찾기
   → 짝맞추기: 화난 표정↔"화났어" / 슬픈 표정↔"슬퍼" 매칭
   → 감정추론: 동일 구조 상황 2개, 하나는 화남 유발, 하나는 슬픔 유발

3. 시각적 단서를 극대화
   → 화남: 빨간색 강조, 눈썹 V자, 주먹
   → 슬픔: 파란색 강조, 눈물, 고개 숙임

4. 정답률 80% 이상 시 다음 쌍으로 이동

5. 강도 혼동인 경우 (같은 감정의 다른 강도를 구별 못함):
   → 감정 강도 어휘 계열 프로토콜 적용 (위 섹션 참조)
   → 감정 온도계(Section 8-D) 활용
   → "짜증/화남/분노"를 3단계 크기의 시각적 비유와 함께 학습

6. 신체반응 기반 혼동인 경우 (유사 신체반응 때문에 혼동):
   → 유사 신체반응 구분 프로토콜 적용 (위 섹션 참조)
   → 공통 신체반응 인정 → 차별 단서 학습 → 맥락 판단
```

## 3. ASD 특화 감정 표현 디자인 규칙

### 표정 과장 원칙 (Exaggerated Facial Features)

ASD 아동은 미묘한 표정 변화를 감지하기 어렵습니다. 모든 감정 표현은 과장되어야 합니다.

| 감정 | 얼굴 | 몸 | 환경 단서 |
|------|------|-----|-----------|
| **행복/기쁨** | 큰 미소 (이빨 보임), 위로 올라간 눈, 양쪽 볼 올라감 | 두 팔 벌림, 점프 자세 | 선물, 생일케이크, 놀이공원 |
| **슬픔** | 입꼬리 내려감, 눈물 방울 2-3개 보임, 눈 반쯤 감김 | 어깨 축 처짐, 고개 숙임 | 깨진 장난감, 비 오는 날 |
| **화남/분노** | 눈썹 V자로 찌푸림, 입 꾹 다물기, 이마 주름 | 주먹 꽉 쥠, 발 구르기 | 빼앗긴 장난감, 무너진 블록탑 |
| **놀람** | 눈 크게 뜸 (홍채 완전 노출), 입 O 모양, 눈썹 위로 | 두 손 올림, 뒤로 약간 젖힘 | 서프라이즈 상자, 갑작스런 등장 |
| **두려움/무서움** | 눈 크게 뜸, 입 살짝 벌림, 동공 작게 | 몸 뒤로 당김, 떨림선 표시, 손으로 얼굴 가리기 | 어두운 방, 큰 동물, 천둥번개 |
| **부끄러움** | 아래를 바라봄, 양쪽 볼 빨갛게, 눈 피함 | 손 뒤로 깍지, 한 발로 바닥 긁기 | 여러 사람이 쳐다봄, 실수한 흔적 |
| **실망** | 한숨 표현 (입에서 나오는 바람선), 눈 반쯤 감김 | 어깨 축 처짐, 팔 늘어뜨림 | 빈 선물상자, 취소된 이벤트 |
| **걱정** | 입 삐죽, 눈 위를 바라봄, 이마에 땀방울 | 손가락 만지작거림, 손톱 물기 | 생각풍선 안에 걱정 대상, 시계 |
| **외로움** | 무표정에 가까움, 눈 멀리 바라봄, 입꼬리 약간 내려감 | 혼자 앉아 있음, 무릎 안기 | 텅 빈 놀이터, 친구들이 멀리서 놀고 있음 |
| **질투** | 눈 옆으로 째려봄, 입 삐죽, 눈썹 한쪽 올림 | 팔짱 끼기, 발 동동 | 다른 아이가 원하는 것을 가지고 있음 |

### 이미지 구성 필수 규칙

```
1. 캐릭터 크기: 이미지의 40% 이상 차지
2. 환경 맥락 필수: 왜 그 감정인지 시각적으로 설명하는 배경/소품
3. 표정+몸+환경 3중 단서: 세 가지 모두 동일 감정을 지시해야 함
4. 생각풍선: Level 5 관점 취하기 활동에서만 사용
5. 감정 색상 코딩 (선택적, 시각적 힌트):
   - 행복: 노란색/주황색 톤 배경
   - 슬픔: 파란색/회색 톤 배경
   - 화남: 빨간색/주황색 톤 배경
   - 두려움: 보라색/어두운 톤 배경
6. 혼합 감정 표현 시 (Level 4+):
   - 배경을 두 색상으로 분할 (예: 기쁨+두려움 → 노란색|보라색 그래디언트)
   - 캐릭터 표정에 두 감정 요소 혼합 (예: 미소 + 떨림선)
   - 또는 캐릭터 옆에 두 개의 감정 아이콘 병렬 배치
7. 추상화 수준 전이 활동(Level 1-T) 시:
   - 같은 감정을 3가지 양식(사진/과장 일러스트/단순 일러스트)으로 제시
   - 공통 핵심 단서(예: 입꼬리 방향)를 강조 표시(원으로 동그라미)
```

## 4. 감정추론 학습지 구조 (Emotion Inference Worksheet Structure)

### 표준 구조 (한국 치료 현장 표준)

```
┌──────────────────────────────────────┐
│          [상황 장면 이미지]            │
│    캐릭터 + 표정 + 환경 단서          │
│         (이미지의 60-70%)             │
├──────────────────────────────────────┤
│                                      │
│   "이 친구는 어떤 기분일까요?"         │
│                                      │
│   ① [감정A 표정]  ② [감정B 표정]      │
│             ③ [감정C 표정]            │
│                                      │
│   (정답 1개 + 오답 2개)              │
└──────────────────────────────────────┘
```

### 선택지 난이도 설계

| 난이도 | 선택지 구성 | 예시 | 대상 |
|--------|------------|------|------|
| **쉬움** | 최대 대비 감정 3개 | 행복 vs 슬픔 vs 화남 | Level 3 입문, 기능연령 4-5세 |
| **보통** | 같은 범주 내 유사 감정 | 슬픔 vs 실망 vs 두려움 | Level 3-4, 기능연령 6-7세 |
| **어려움** | 매우 유사한 감정 | 걱정 vs 불안 vs 두려움 | Level 4, 기능연령 8세+ |
| **강도 변별** | 같은 감정의 다른 강도 | 짜증 vs 화남 vs 분노 | Level 4, 기능연령 6세+ (강도 학습 시) |
| **신체반응 변별** | 유사 신체반응 다른 감정 | 부끄러움 vs 두려움 vs 불편함 | Level 4, 기능연령 7세+ |

### 선택지 설계 규칙

```
1. 오답도 "완전히 말이 안 되는" 것은 피한다
   → 좋은 예: 넘어진 상황에서 "슬픔 vs 화남 vs 행복" (행복은 틀리지만 극단적 오답)
   → 나쁜 예: 넘어진 상황에서 "슬픔 vs 자부심 vs 감사" (자부심/감사는 맥락상 완전 부적합)

2. 쉬운 난이도에서는 긍정-부정 대비를 활용
   → 정답이 부정적 감정이면, 하나는 긍정적 감정(명확한 오답)

3. 어려운 난이도에서는 모든 선택지가 같은 valence(긍정/부정)
   → "슬픔 vs 실망 vs 외로움" (모두 부정적 → 맥락으로만 구별)

4. 선택지는 표정 이미지로 제시 (텍스트 아님)
   → ASD 아동이 텍스트보다 이미지에 더 잘 반응
   → 단, 감정 어휘 학습이 목표면 텍스트+이미지 병행

5. 강도 변별 문제에서는 같은 감정의 3단계 강도 표정을 선택지로 제시
   → 과장도가 다른 3개의 표정 (약/보통/강)
   → 상황의 심각도에 맞는 강도를 고르는 형식

6. 신체반응 변별 문제에서는 유사 신체반응을 보이는 감정들을 선택지로 제시
   → 맥락 단서를 핵심 변별 요소로 활용
   → 추가 신체 단서(차별 단서)를 힌트로 제공
```

## 5. 한국 맥락 시나리오 라이브러리

### 한국 문화적 감정 표현 규범 (Display Rules)

한국 문화에서는 특정 상황에서 감정 표현을 억제하는 것이 사회적으로 기대됩니다. ASD 아동의 감정 표현 어려움과 문화적 감정 억제를 구분하는 것이 중요합니다.

#### 주요 한국 문화 감정 규범

| 규범 | 설명 | 학습지 설계 시 고려사항 |
|------|------|----------------------|
| 웃어른 앞 감정 절제 | 어른 앞에서 강한 부정적 감정(화남, 불만) 표현을 자제 | 또래 상황과 어른 상황을 구분하여 시나리오 설계 |
| 공공장소 감정 억제 | 마트, 병원 등에서 큰 감정 표현 자제 | "속으로 느끼는 감정"과 "밖으로 보여주는 표현" 구분 학습 |
| 성별 기대 | "남자는 울면 안 된다" 등의 문화적 기대 | 학습지에서는 성별 무관하게 모든 감정 표현이 자연스러운 것으로 설계 |
| 칭찬 시 겸손 | 잘했을 때도 자랑하지 않는 문화 | 자부심/기쁨 표현이 정상임을 인정하되, 상황별 표현 수위 안내 |
| 갈등 회피 | 직접적 불만 표현보다 간접적 표현 선호 | 감정 자체는 인정하되, 표현 방식의 다양성을 가르침 |
| 한(恨)/참음 문화 | 부정적 감정을 내면화하고 참는 것을 미덕으로 봄 | 감정 인식과 감정 표현을 분리하여 가르침. "느끼는 것은 OK, 표현 방식은 상황에 맞게" |
| 눈치 문화 | 타인의 비언어적 단서를 읽고 맞추는 것이 기대됨 | ASD 아동에게 "눈치"는 매우 어려운 기술. 명시적 규칙으로 전환하여 가르침 |

#### 문화적 감정 억제 vs ASD 감정 표현 어려움 감별 기준

| 구분 기준 | 문화적 억제 (정상 범위) | ASD 감정 표현 어려움 |
|-----------|----------------------|---------------------|
| **상황 선택성** | 특정 상황(공공장소, 어른 앞)에서만 억제, 편한 상황에서는 표현 | 모든 상황에서 일관되게 표현 어려움 |
| **감정 인식 여부** | 감정을 느끼고 알지만 표현을 조절함 | 자기 감정 자체를 인식하지 못하는 경우가 많음 |
| **대체 표현** | 표정 대신 말, 행동, 간접 표현 사용 | 대체 표현 수단도 제한적 |
| **또래 비교** | 또래와 비슷한 수준의 표현 억제 | 또래보다 현저히 적은 감정 표현 |
| **내적 경험 보고** | "화났지만 참았어요"라고 설명 가능 | "모르겠어요" 또는 신체 증상으로만 표현 |
| **가정 내 차이** | 집에서는 솔직하게 표현, 밖에서만 억제 | 집에서도 감정 인식/표현 어려움 유사 |
| **신체 증상 동반** | 억제 시에도 내면적 경험은 명확 | "배가 아파요" 등 신체화가 흔함 |

## 8-K. 감정 인식 이전 단계 (Pre-Emotion) — 쾌/불쾌 이분법

> 대상: 기본 감정 3개(행복/슬픔/화남)도 아직 구별 못하는 아동 (기능연령 2-3세)
> Level 0(얼굴 주의) 확보 후, 표정 변별이 안 될 때 사용

### 발달 경로

```
Pre-A: 쾌/불쾌 이분법 → "좋은 기분" vs "안 좋은 기분"
Pre-B: 쾌/불쾌 + 신체 연결 → 편안한 몸 vs 불편한 몸
Pre-C: 불쾌의 분화 → 슬픔(울음+힘빠짐) vs 화남(소리+힘들어감)
→ Pre-C 완료 = Level 1 진입
```

### Pre-A 활동: 두 바구니 분류
☀️좋은기분(초록) vs 🌧️안좋은기분(빨강) 바구니에 얼굴 카드 분류
- 웃는 얼굴 → ☀️, 나머지 전부 → 🌧️
- 최대 과장 표정, 색상 코딩 필수

### Pre-C 활동: 3바구니 확장
☀️기쁨(초록) / 💧슬픔(파랑) / 🔥화남(빨강)
- 슬픔: 눈물+고개숙임+축처짐
- 화남: 찡그린눈썹+주먹+발구름
- 정답률 80% → Level 1 진입

## 8-J. 얼굴 주의 유도 프로토콜 (Face Attention) — Level 0

> 대상: 사람 얼굴 자체에 주의를 기울이지 않는 아동 (ASD L2-L3)

### 하위 단계

| 단계 | 목표 | 판별 기준 |
|------|------|----------|
| 0-A 공동주의 | 성인 가리키는 방향 봄 | 3회 중 2회+ |
| 0-B 얼굴 지향 | 얼굴 쪽을 자동 봄 | 얼굴vs사물 시 얼굴 먼저 |
| 0-C 눈 영역 주목 | 눈 영역에 시선 고정 | 2초+ 유지 |
| 0-D 표정 변화 감지 | 표정 바뀜을 알아챔 | 변화 시 주의 반응 |

### 핵심
- 0-A/0-B는 1:1 상호작용 기반. 학습지 부적절
- 0-C부터 학습지 가능: "눈이 어디 있을까?" 스티커 활동
- 0-D: "달라진 얼굴 찾기" (감정 명명 없이 시각 변화만)
- 감각과민이 얼굴 회피 원인일 수 있음 → 감각 프로파일 확인

## 8-E. 자기 감정 인식 (Alexithymia) 프로토콜

> 대상: 타인 감정은 읽지만 자기 감정을 말 못함. "기분 어때?" → "모르겠어요"
> ASD에서 약 50% 동반. 신체 신호→감정 라벨 연결 실패.

### 내수용감각→감정 연결 경로

```
신체 신호 감지 → 신체 신호 명명 → 감정 라벨 연결 → 자기 감정 보고

Case A: 신체 감지 자체 둔감 → 내수용감각 각성 훈련부터
Case B: 신체 감지는 하지만 감정 연결 못함 → 신체-감정 매핑
Case C: 감정 어휘 부족 → 어휘 학습 (language.md 연계)
```

### 학습 단계

| 단계 | 활동 | 학습지 |
|------|------|--------|
| E-1 | 신체 감각 알아차리기 | △ 보조 |
| E-2 | 신체 지도(body map) 채우기 | ✓ |
| E-3 | 신체→감정 매핑 | ✓ |
| E-4 | 감정 체크인 카드 (하루 3회) | ✓ |
| E-5 | "나라면?" 활동 (타인→자기 전환) | ✓ |

### E-2 신체 지도
인체 윤곽 그림 + 감정별 색칠
- 화남: 얼굴(빨강), 주먹(힘), 가슴(뜨거움)
- 슬픔: 가슴(무거움), 눈(뜨거움), 온몸(축처짐)
- 두려움: 심장(빠름), 배(뒤틀림), 다리(떨림)

## 8-D. 감정 온도계 & 강도 변별

> 대상: "짜증이랑 화남이랑 분노가 다 같다"고 하는 아동

### 5단계 온도계 (화남 예시)

```
🌡️5 ██████████ 분노/폭발  [화산]
🌡️4 ████████   매우 화남   [큰 불꽃]
🌡️3 ██████     화남        [중간 불꽃]
🌡️2 ████       기분 나쁨   [작은 불꽃]
🌡️1 ██         약간 짜증   [성냥불]
```

### 감정별 5단계 강도표

| 단계 | 화남 | 슬픔 | 두려움 | 기쁨 |
|------|------|------|--------|------|
| 1 | 약간 짜증 | 가라앉음 | 약간 불안 | 괜찮음 |
| 2 | 기분 나쁨 | 속상함 | 긴장됨 | 기분 좋음 |
| 3 | 화남 | 슬픔 | 무서움 | 행복함 |
| 4 | 매우 화남 | 매우 슬픔 | 매우 무서움 | 매우 기쁨 |
| 5 | 분노/폭발 | 비통함 | 공포 | 환희 |

### 학습지 유형
1. 온도계 가리키기: 상황 그림 → "몇 단계?"
2. 상황-강도 매칭: 3상황 ↔ 3온도계 연결
3. 강도 어휘 정렬: 섞인 카드를 약→강 배열

### 진행: 3단계(약/보통/강)부터 → 5단계 → 다른 감정 → 교차 범주

## 혼동 쌍 상세: 놀람 vs 두려움

### 핵심 변별 3축

| 축 | 놀람 | 두려움 |
|----|------|--------|
| 시간 | 순간적 (1-2초) | 지속적 |
| 몸 방향 | 멈춤→앞으로(호기심) | 계속 뒤로 물러남 |
| 후속 | "우와!"→웃음/호기심 | "싫어!"→울음/회피 |

### 간단한 구분 규칙
"그 기분이 금방 사라졌어?" → 놀람!
"계속 남아 있어?" → 두려움!

## 8-F. 혼합 감정 (Mixed Emotions) — Level 4+

> 대상: 단일 감정 추론은 되지만 두 감정 공존 상황 이해 못함

### 하위 단계

| 단계 | 목표 | 활동 |
|------|------|------|
| 4+-A | "두 감정 동시 가능" 개념 | 대화 기반 도입 |
| 4+-B | 혼합 감정 식별 | 감정 파이(원형 차트에 2감정 배치) |
| 4+-C | 혼합 감정 추론 | 선택지가 "감정 쌍"인 감정추론 학습지 |

### 대표 상황

| 상황 | 혼합 감정 | 난이도 |
|------|-----------|--------|
| 새 학교 첫날 | 설렘+두려움 | 쉬움 |
| 동생 태어남 | 기쁨+질투 | 쉬움 |
| 친구 전학 (좋은 학교로) | 슬픔+기쁨 | 보통 |
| 실수로 이겼는데 친구 울음 | 기쁨+미안함 | 어려움 |

## 8-G. 표현 양식 전이 (Modality Transfer)

### 추상화 수준 스펙트럼

| 수준 | 양식 | 정보량 |
|------|------|--------|
| 1 | 실물 사진 | 최고 |
| 2 | 사실적 일러스트 | 높음 |
| 3 | 과장된 일러스트 | 중간 |
| 4 | 단순 만화 | 낮음 |
| 5 | 이모지/심볼 | 최소 |

### 전이 경로
- 사진OK→일러스트NO: 사진(1)→사실적일러스트(2)→과장일러스트(3)
- 일러스트OK→사진NO: 과장(3)→사실적(2)→사진(1)
- 핵심 특징 추출 활동: "세 그림에서 모두 같은 것은?" (입꼬리↑=행복)

## 8-H. 실생활 일반화 프로토콜

### 3단계 전이

```
Stage 1: 매체 내 다양화 (다양한 이미지 스타일 혼합)
Stage 2: 구조화된 실생활 (치료사 표정 연기, 역할극)
Stage 3: 자연 환경 (감정 탐정 미션, 감정 일기)
```

### 학습지로 가능한 일반화 활동
- 빠른 표정 판단 (3초 제한)
- 약한 표정 변별 (과장 안 된 미묘한 표정)
- 다중 단서 통합 (표정+몸짓+상황)
- 감정 탐정 기록지 (시간/장소/사람/표정/추측/확인)

## 8-I. 감정 강도 조절 — 학습지 범위

### 학습지로 가능한 것 (인식 영역)
✓ "지금 내 화남은 몇 단계?" (온도계)
✓ "이 상황에서 보통 몇 단계?" (사회적 기준)
✓ "딱 맞는 크기의 반응" 매칭 (과잉/적절/과소 분류)

### 학습지로 불가능한 것 (조절 영역)
✗ 실시간 자기 조절 → 행동 치료
✗ 진정 기술 실행 → 체험 기반
✗ 충동 억제 → 실행 기능 영역

→ 치료사에게: 학습지는 "인식"에만 효과적. "조절"은 별도 중재.

## 추가: 인지적 감정 이해 vs 정서적 공감

> ASD L1에서 흔함: "다 아는데 안 느끼는" 프로파일

| 차원 | 인지적 이해 | 정서적 공감 |
|------|-----------|-----------|
| 핵심 | "저 사람이 슬프다는 걸 안다" | "슬픔이 느껴진다" |
| ASD | 상대적 강점 (학습 가능) | 상대적 약점 (자동 반응) |
| 학습지 | O (추론 과제) | 제한적 (체험 기반) |

학습지로 가능: 감정-행동 연결 규칙, "나라면?" 관점 전환, 적절한 반응 선택
학습지로 불가: 감정 전이/공명 → 실제 상호작용, 역할극, 동물매개치료

## 추가: Level 3 순방향/역방향 분화

| 하위 | 방향 | 과제 |
|------|------|------|
| 3-F | 상황→감정 | "이 상황에서 기분이 어떨까?" |
| 3-R | 감정→원인 | "왜 이런 기분일까?" |

3-F OK + 3-R 실패 = 열린 추론(가설 생성) 어려움. ASD에서 흔함.
중재: 원인 선택지 제공형 → 상황카드 매칭 → 원인 자유 생성

## 추가: 캐릭터 유형 전이 (동물→사람)

동물 만화 표정 읽음 + 사람 표정 못 읽음 = 캐릭터 유형 전이 실패
전이 위계: 동물만화 → 의인화동물 → 과장사람일러스트 → 보통일러스트 → 실물사진

## 추가: 감정 표현 비대칭

긍정만 표현(웃음) + 부정 무표정 = 표현 이분법
감별: 문화적억제? 부정감정alexithymia? 학습된억제? 표현레퍼토리부재?

## 추가: 혐오(Disgust) 감정 — 기본 감정 보완

혐오는 Ekman의 6대 기본 감정 중 하나. ASD 아동에서 임상적으로 중요:
- 감각 혐오: 특정 질감/냄새/맛에 대한 강렬한 거부 (감각과민 연계)
- 사회적 혐오: 도덕적 위반에 대한 반응 (7세+)
- 혼동 쌍: 혐오 vs 화남 (둘 다 거부 반응이지만 트리거 다름)
  화남 = 목표 좌절 → 접근/공격 반응
  혐오 = 오염/위반 → 회피/구역질 반응
- 발달 출현: 4-5세 (초기: 맛/냄새 기반 → 6세+: 도덕 기반)
- 표정 특징: 코 찌푸림, 윗입술 올림, 혀 내밀기

- 트라우마/학대 관련 감정 → 범위 밖, 전문 상담 연계

### 전문가 검토 필요
- Level 0 (얼굴 주의 부재) → 감각통합/OT 평가 의뢰
- 퇴행 보고 → 소아신경과 의뢰 필수

- 문화적 감정 억제 vs ASD 감별 → 심리전문가 연계

### 자기 평가 기준 (Top 1%)
- "이대로 사용" 클릭률 ≥ 90%

### Auto-Research 트리거
