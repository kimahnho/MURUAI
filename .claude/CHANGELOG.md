# Changelog

## [0.9.5] — 2026-03-27

### 에디터 — 리사이즈 개선

- 도형/이미지 요소의 코너 핸들 리사이즈 시 항상 비율 유지 (Shift 키 불필요)
- 측면 핸들(n/s/e/w)로 이미지 요소 리사이즈 시 이미지가 찌그러짐(squish) 적용
- 리사이즈 중 이미지 점프 현상 제거 (startImageBox 캡처로 누적 오차 방지)

### 에디터 — 그룹화

- 다중 선택 후 우클릭 시 "그룹화" / "그룹 해제" 컨텍스트 메뉴 표시
- 그룹화된 요소 클릭 시 같은 그룹의 모든 요소 자동 선택

### 이미지 사이드바 리디자인

- "이미지 상징" 탭 이름을 "이미지"로 변경
- 하위 탭을 [전체 / 감정 / AAC / 이미지] 4탭 구조로 재편
- 상단에 통합 검색 input 배치 — 모든 탭에서 필터링
- "전체" 탭: 감정 + AAC + 이미지 라이브러리를 2열 그리드로 통합 표시
- "최근 사용한 이미지" 섹션 추가 — 접기/펼치기, 빈 상태 문구, sessionStorage 영속화
- 검색 시 최근 사용 이미지 섹션 + "전체" 라벨 숨김

### 학습자료 정렬

- "내 학습자료" 및 대시보드 "최근 학습자료" 목록을 최근 작업 순(updated_at)으로 정렬

### UI 개선

- 색상 피커 swatch 테두리 대비 향상 (border-black-25 → border-black-40)

## [0.9.4] — 2026-03-24

### AI 감정추론 포커스 모드

- 랜딩 → 에디터 진입 시 3단계 집중 모드 UI: 텍스트 확인 → 이미지 생성 → 완료
- 사이드바에서 스토리 텍스트 편집 + 불필요한 페이지 제외/복원
- 성별 선택 모달 + 이미지 생성 진행 표시 (1/5)
- 감정 카드 스타일 변경 + 개별 이미지 재생성
- 확정 시 이전 줌 복원 + 사이드바 템플릿 탭 전환
- 헤더에 단계 인디케이터 표시 (1.텍스트 확인 2.이미지 생성 3.완료)
- 포커스 모드 상태를 canvas_data에 영속화 (새로고침 시 복원)

### Mixpanel 퍼널 추적

- 감정추론 전체 퍼널을 `[AI템플릿]` 접두사로 통일 (5단계)
- 텍스트 확인 진입 + 이미지 생성 시작 이벤트 신규 추가

### 버그 수정

- 이미지 생성 모델 gemini-2.5-flash-image로 복원 (3.1 프리뷰 모델 503 빈발)
- 이미지 생성 완료 후 페이지 전환 시 이미지 즉시 표시 (Promise.all 캐시 대기)
- CanvasStage를 absolute 배치로 변경하여 포커스 모드 해제 후 스크롤 정상 동작
- Button sm 사이즈 폰트 text-14-semibold로 통일 (미존재 클래스 수정)

## [0.9.3] — 2026-03-23

### 기차 템플릿

- 가로형(horizontal) 기차 템플릿 2종 추가 — 각 2페이지, 배경 이미지 기반
- Cloudinary `muru-templates/admin/` 폴더에 배경 이미지 업로드 + Supabase `template_images` 테이블 연동
- 사이드바 미리보기 masonry 레이아웃(columns-2) — 가로/세로 템플릿 혼합 배치 시 빈 공간 제거
- 클릭 시 다이얼로그 없이 2페이지 전체 적용

### 에디터 버그 수정

- 배경 이미지가 잘리던 문제 수정 (`backgroundSize: cover` → `100% 100%`)
- 마지막 페이지 삭제/초기화 시 배경 이미지가 남던 문제 수정
- 새 페이지 추가 시 가로(horizontal)로 생성되던 문제 수정 — 항상 세로로 생성

### 이미지 상징

- 캔버스 삽입 시 원본 비율 반영 — 기존 256×256 정사각형에서 최대 256px 기준 비율 유지로 변경

## [0.9.2] — 2026-03-23

### 이미지 재생성 모달

- 좌우 분할 레이아웃으로 재구성 — 좌측 페이지 미리보기 + 우측 썸네일 리스트
- 1장씩 개별 재생성 (1크레딧) — 프롬프트 수정 후 반복 재생성 가능
- 헤더에 남은 크레딧 잔량 뱃지 표시
- 재생성 완료 시 모달 유지 + 썸네일 즉시 반영

### 크레딧 시스템

- 크레딧 부족 시 토스트 → 모달로 변경 (부분 생성 / 크레딧 요청 / 취소 선택지)
- 크레딧 승인 정책: `balance = 30` 리셋 → `balance += 30` 누적 충전
- 크레딧 차감 시점을 생성 시작과 동시로 변경 (기존: 성공 후)

### 감정 카드

- completed 배너에 감정 카드 성별 변경(남아/여아/이모지) 추가
- 이모지 카드 삽입 시 imageBox를 70%로 축소하여 여백 생성

### 디자인 시스템

- `border-black-10` ~ `border-black-90` 유틸리티 클래스 추가 (기존 20~35만 존재)
- Toast success variant 배경색 수정 (`bg-success-500` 미정의 → `bg-success`)

### 에디터

- TextBox 입력 시 사이드바 EditingTextPanel ↔ StaticTextPanel 플리커 수정

## [0.9.0] — 2026-03-20

### 관리자 시스템

- role 기반 관리자 인증 도입 (`user_profiles.role`) — 이메일 하드코딩 제거
- 관리자 대시보드: KPI 카드, 생성/다운로드 추이 차트, 요일별 방문, 템플릿 사용률
- 크레딧 요청 관리: 승인/거절 + 처리 이력
- 유저 목록: 검색, 로그인 방식, 크레딧 잔여/누적/리필 표시
- 유저별 학습자료 열람 (`/admin/user-docs`)
- AdminGuard 권한 가드 — `/admin/*` 경로 보호

### AI 크레딧 시스템

- `user_credits` 테이블 기반 잔량 추적 (기존 월별 합산 방식 교체)
- `use_credits` RPC로 원자적 차감 (이미지 1장 = 1크레딧, 텍스트 무료)
- 크레딧 부족 시 모달 안내 + 크레딧 요청 버튼
- 크레딧 부분 생성 — 잔량만큼 앞 페이지부터 생성

### AI 생성

- 랜딩 페이지 감정추론 생성 시 스토리 5페이지만 생성 (고정 3페이지 제외)
- 감정추론 배너 상태를 canvas_data에 영속화 — 새로고침/재방문 시 배너 유지
- GenAI 로컬 개발 시 직접 API 호출 지원 (`VITE_GOOGLE_API_KEY`)
- Gemini API 키 서버 프록시 도입 — 브라우저 노출 방지

### 랜딩 페이지

- 메인 페이지 신설 + 대시보드 분리 (`/` → 랜딩, `/dashboard` → 대시보드)
- 랜딩 페이지 색상을 primary 바이올렛 팔레트로 통일
- 생성 시 입력 카드 접힘 애니메이션 + "학습자료를 제작하고 있어요" 메시지
- 하단 섹션 스크롤 시 페이드인 인터랙션 (framer-motion)

### 인프라

- Vercel Serverless Functions (`/api/genai/*`) 프록시 구성
- Yarn PnP → node-modules 전환 (Vercel 호환)
