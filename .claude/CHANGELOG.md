# Changelog

## [0.9.0] — 2026-03-20

### 관리자 시스템
- role 기반 관리자 인증 도입 (`user_profiles.role`) — 이메일 하드코딩 제거
- 관리자 대시보드: KPI 카드, 생성/다운로드 추이 차트, 요일별 방문, 템플릿 사용률
- 크레딧 요청 관리: 승인/거절 + 처리 이력
- 유저 목록: 검색, 로그인 방식, 크레딧 잔여/누적/리필 표시
- 유저별 학습자료 열람 (`/admin/user-docs`)
- AdminGuard 권한 가드 — `/admin/*` 경로 보호

### AI 크레딧 시스템
- `user_credits` 테이블 기반 잔량 추적 (기존 월별 합산 방식 교체)
- `use_credits` RPC로 원자적 차감 (이미지 1장 = 1크레딧, 텍스트 무료)
- 크레딧 부족 시 모달 안내 + 크레딧 요청 버튼
- 크레딧 부분 생성 — 잔량만큼 앞 페이지부터 생성

### AI 생성
- 랜딩 페이지 감정추론 생성 시 스토리 10페이지만 생성 (고정 3페이지 제외)
- 감정추론 배너 상태를 canvas_data에 영속화 — 새로고침/재방문 시 배너 유지
- GenAI 로컬 개발 시 직접 API 호출 지원 (`VITE_GOOGLE_API_KEY`)
- Gemini API 키 서버 프록시 도입 — 브라우저 노출 방지

### 랜딩 페이지
- 메인 페이지 신설 + 대시보드 분리 (`/` → 랜딩, `/dashboard` → 대시보드)
- 랜딩 페이지 색상을 primary 바이올렛 팔레트로 통일

### 인프라
- Vercel Serverless Functions (`/api/genai/*`) 프록시 구성
- Yarn PnP → node-modules 전환 (Vercel 호환)

## [0.8.0] — 2026-03-19 (이전 릴리스)
